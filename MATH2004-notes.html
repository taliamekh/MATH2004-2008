Chapter 1

Vectors in the Plane

Vectors, scalar multiplication, dot product, cross product, projections, and their geometry in ℝ² and ℝ³.

Vectors are one of the most important tools in science and engineering. They originated as a way to describe physical quantities that have both a size and a direction — things like velocity, force, and acceleration. Unlike ordinary numbers (scalars), which only tell you "how much," a vector also tells you "which way." This chapter builds the algebra and geometry of vectors from scratch, starting in two dimensions and extending to three.

1.1 — Vectors in the Plane

We begin in the Cartesian plane \(\mathbb{R}^2\). A plane is a flat, two-dimensional surface extending infinitely in both length and width. We set up the usual rectangular coordinate system with perpendicular x- and y-axes meeting at the origin \(O\). Every point \(P\) in the plane is identified by an ordered pair \((a,b)\) of real numbers.

Vector (Geometric)

A vector is a directed line segment — an arrow with a definite length (also called magnitude) and a definite direction. Every vector has an initial point (tail) where the arrow begins and a terminal point (tip) where it ends. Vectors are denoted by boldface letters \(\mathbf{v}\) or with an arrow: \(\vec{v}\), \(\vec{u}\), \(\vec{w}\).

The critical idea is that a vector is determined entirely by its length and direction — not by where it sits in the plane. If you slide an arrow to a new position without rotating or stretching it, you still have the same vector. This leads to the formal notion of equality.

Vector Equality

Two vectors are equal if and only if they have the same length and the same direction. Their position in the plane is irrelevant — you can translate a vector anywhere without changing it.

All three arrows represent the SAME vector — same length and direction, different positions.

Because position doesn't matter, we adopt a standard convention: among all the arrows representing a given vector, we pick the one whose initial point is the origin \(O=(0,0)\). Its terminal point then lands at some point \(P(a,b)\), and we identify the vector with that point. This is why we can write \(\vec{v} = (a,b)\).

Opposite and Zero Vectors

The opposite (or negative) of \(\vec{v}\) is \(-\vec{v}\): a vector with the same length but the opposite direction. The zero vector \(\vec{0} = (0,0)\) has zero length and no defined direction. It is the only vector with magnitude zero.

1.2 — Scalar Multiplication

A scalar is simply a real number — the word emphasizes that it has magnitude only, no direction. We often need to scale a vector (stretch it, shrink it, or flip it), which is done by multiplying it by a scalar.

Scalar Multiplication

For a vector \(\vec{v}\) and a scalar \(a \in \mathbb{R}\), the product \(a\vec{v}\) is a new vector whose length is \(|a|\,|\vec{v}|\) (the absolute value of \(a\) times the original length). If \(a > 0\), it points in the same direction as \(\vec{v}\). If \(a < 0\), it points in the opposite direction. If \(a = 0\), the result is the zero vector. When \(a < 0\), we say \(a\vec{v}\) and \(\vec{v}\) are antiparallel.

Magnitude of a Scalar Multiple
\[|a\vec{v}| = |a|\,|\vec{v}|\]
a — scalar multiplier|a| — absolute value of the scalar|v| — magnitude of the original vector
★ Tip

Positive scalar = stretch/shrink (same direction). Negative scalar = flip + scale (opposite direction). Zero scalar = zero vector. Scalar multiplication in components: \(a(v_1, v_2) = (av_1, av_2)\).

1.3 — Vectors and Points

The geometric picture of vectors as arrows is intuitive, but for computation we need an algebraic framework. The key idea is to identify each vector with a point in the coordinate system. For any vector, we choose the representative arrow starting at the origin. Its tip lands at a unique point, and we use that point's coordinates as the vector's components.

If \(P(x_0, y_0)\) and \(Q(x_1, y_1)\) are two points, the vector from \(P\) to \(Q\) is found by subtracting coordinates: the arrow \(\vec{PQ}\) has the same length and direction as the arrow from the origin to the point \((x_1 - x_0,\; y_1 - y_0)\).

Vector Between Two Points
\[\vec{PQ} = (x_1 - x_0,\; y_1 - y_0)\]
P = (x₀, y₀) — initial point (tail)Q = (x₁, y₁) — terminal point (tip)PQ — vector from P to Q (terminal minus initial)

The distance between two points is the length of the vector joining them. This follows directly from the Pythagorean theorem applied to the horizontal and vertical legs of the right triangle formed by the two points.

Distance Formula
\[d(P,Q) = \sqrt{(x_1 - x_0)^2 + (y_1 - y_0)^2}\]
d(P,Q) — distance between P and Q(x₁−x₀) — horizontal displacement(y₁−y₀) — vertical displacement
Magnitude (Length) of a Vector
\[|\vec{v}| = \sqrt{v_1^2 + v_2^2}\]
|v| — magnitude (always ≥ 0)v₁ — x-componentv₂ — y-component|v| = 0 only when v = 0
★ Key Fact

From now on, points and vectors are interchangeable. The vector \(\vec{v} = (a,b)\) IS the point \(P(a,b)\), and vice versa. We use whichever language is more convenient.

1.4 — Sums and Differences

Now that vectors are identified with coordinate pairs, we can define arithmetic operations. The sum and difference of vectors are defined component-by-component, which turns out to match the geometric "parallelogram law" perfectly.

Vector Addition / Subtraction
\[(a,b) \pm (c,d) = (a \pm c,\; b \pm d)\]
(a,b) — first vector(c,d) — second vectoradd/subtract component-by-component

Geometrically, vector addition follows the Parallelogram Law: place both vectors at the same starting point, complete the parallelogram, and the diagonal from the starting point is the sum. Equivalently, you can use the "tip-to-tail" method — place the tail of the second vector at the tip of the first, and the sum goes from the start of the first to the end of the second.

Parallelogram Law: the sum v + w is the diagonal of the parallelogram formed by v and w.

Scalar multiplication in component form works naturally: \(\alpha(a,b) = (\alpha a, \alpha b)\). This is consistent with the geometric definition — the resulting vector has length \(|\alpha|\) times the original and points in the same (or opposite) direction.

Unit Vectors

A unit vector is any vector with magnitude exactly 1. Unit vectors are important because they isolate direction — they tell you "which way" without any information about "how far." Any nonzero vector can be turned into a unit vector by dividing by its length.

Unit Vector (Normalization)
\[\hat{u} = \frac{\vec{v}}{|\vec{v}|}\]
û — unit vector (length = 1)v — the original nonzero vector|v| — its magnitude (must ≠ 0)

Why does this work? Because \(|\hat{u}| = \left|\frac{\vec{v}}{|\vec{v}|}\right| = \frac{|\vec{v}|}{|\vec{v}|} = 1\). The division scales the vector down (or up) until its length is exactly 1, while preserving its direction.

Standard Basis Vectors in ℝ²

The two most fundamental unit vectors in the plane are \(\mathbf{i} = (1,0)\) and \(\mathbf{j} = (0,1)\). They point along the positive x- and y-axes respectively. Every vector in \(\mathbb{R}^2\) can be written uniquely as a combination of these two: \(\vec{v} = (a,b) = a\mathbf{i} + b\mathbf{j}\). This is called the decomposition of \(\vec{v}\) into its basis components.

PropertyStatement
Commutative\(\vec{v}+\vec{w}=\vec{w}+\vec{v}\)
Associative\((\vec{u}+\vec{v})+\vec{w}=\vec{u}+(\vec{v}+\vec{w})\)
Identity\(\vec{v}+\vec{0}=\vec{v}\)
Inverse\(\vec{v}+(-\vec{v})=\vec{0}\)
Distribution (scalar)\(a(\vec{v}+\vec{w})=a\vec{v}+a\vec{w}\)
Distribution (sum of scalars)\((a+b)\vec{v}=a\vec{v}+b\vec{v}\)

1.5 — The Inclination of a Vector

Every unit vector in the plane can be written using trigonometry. If \(\vec{u}\) is a unit vector making angle \(\theta\) with the positive x-axis, then \(\vec{u} = (\cos\theta, \sin\theta)\). More generally, any nonzero vector \(\vec{v} = (a,b)\) makes some angle \(\theta\) with the x-axis, and we can write \(a = |\vec{v}|\cos\theta\) and \(b = |\vec{v}|\sin\theta\). This angle \(\theta\) is called the inclination of the vector.

Inclination Angle
\[\tan\theta = \frac{b}{a}\quad(a\neq 0),\qquad \vec{v} = |\vec{v}|(\cos\theta,\;\sin\theta)\]
θ — angle with positive x-axis (0 ≤ θ < 2π)a — horizontal componentb — vertical component|v| — magnitude

Finding \(\theta\) from \(\tan\theta = b/a\) requires care because the arctangent function only returns values in \((-\pi/2,\;\pi/2)\), which covers Quadrants I and IV. For vectors in Quadrants II and III (where \(a < 0\)), you must add \(\pi\) to the arctangent result. If \(a = 0\), the vector points straight up (\(\theta = \pi/2\)) or straight down (\(\theta = 3\pi/2\)) depending on the sign of \(b\).

★ Exam Warning — Quadrant Adjustment

The formula \(\theta = \arctan(b/a)\) only works directly in QI and QIV. For QII/QIII (where \(a < 0\)): add \(\pi\). Specifically: if \(a \geq 0, b \geq 0\) (QI): \(\theta = \arctan(b/a)\). If \(a < 0\) (QII or QIII): \(\theta = \arctan(b/a) + \pi\). If \(a > 0, b < 0\) (QIV): \(\theta = \arctan(b/a) + 2\pi\). Always sketch the vector to verify your answer is in the right quadrant.

Show Solution

2

1.6 — Vectors in ℝ³

Everything we've done in two dimensions extends naturally to three. A point in \(\mathbb{R}^3\) has three coordinates \((x,y,z)\), and vectors now have three components. The three-dimensional space is divided into eight octants (analogous to quadrants in 2D) by the three coordinate planes (\(xy\)-plane where \(z=0\), \(xz\)-plane where \(y=0\), \(yz\)-plane where \(x=0\)).

Vectors in ℝ³

A vector in \(\mathbb{R}^3\) is an ordered triple \(\vec{v} = (v_1, v_2, v_3)\). All operations — addition, scalar multiplication, magnitude — work exactly as in \(\mathbb{R}^2\) but with an extra component. The vector from \(P(x_0,y_0,z_0)\) to \(Q(x_1,y_1,z_1)\) is \(\vec{PQ} = (x_1-x_0,\;y_1-y_0,\;z_1-z_0)\).

Magnitude in ℝ³
\[|\vec{v}| = \sqrt{v_1^2 + v_2^2 + v_3^2}\]
v₁ — x-componentv₂ — y-componentv₃ — z-component|v| — 3D Pythagorean theorem
Standard Basis in ℝ³

\(\mathbf{i}=(1,0,0)\), \(\mathbf{j}=(0,1,0)\), \(\mathbf{k}=(0,0,1)\). These are mutually perpendicular unit vectors along the three axes. Any vector decomposes as \(\vec{v} = v_1\mathbf{i} + v_2\mathbf{j} + v_3\mathbf{k}\).

A 3D coordinate system with position vector OP = (x, y, z).

1.7 — The Dot Product

There are two fundamentally different ways to "multiply" vectors. The first — the dot product (also called scalar product or inner product) — takes two vectors and produces a scalar (a number). The second — the cross product (Section 1.9) — produces a vector. Remembering which is which is critical.

Dot Product — Algebraic Definition
\[\vec{v}\cdot\vec{w} = v_1w_1 + v_2w_2 + v_3w_3\]
v = (v₁, v₂, v₃) — first vectorw = (w₁, w₂, w₃) — second vectorresult is a scalar (a number, not a vector)

The algebraic definition is simple: multiply matching components and add. But the dot product has a deep geometric meaning: it measures how much two vectors "align" with each other.

Dot Product — Geometric Form
\[\vec{v}\cdot\vec{w} = |\vec{v}|\,|\vec{w}|\cos\theta\]
|v| — magnitude of v|w| — magnitude of wθ — angle between v and w (0 ≤ θ ≤ π)
Proof Sketch — Why Both Forms Are Equal

Apply the law of cosines to the triangle with sides \(\vec{v}\), \(\vec{w}\), and \(\vec{v}-\vec{w}\). The law gives \(|\vec{v}-\vec{w}|^2 = |\vec{v}|^2 + |\vec{w}|^2 - 2|\vec{v}||\vec{w}|\cos\theta\). Expanding \(|\vec{v}-\vec{w}|^2\) in components and simplifying yields \(v_1w_1 + v_2w_2 + v_3w_3 = |\vec{v}||\vec{w}|\cos\theta\), connecting the algebraic and geometric forms.

Rearranging the geometric form gives us a formula for the angle between any two nonzero vectors:

Angle Between Two Vectors
\[\cos\theta = \frac{\vec{v}\cdot\vec{w}}{|\vec{v}|\,|\vec{w}|}\]
v · w — compute the dot product|v|, |w| — compute both magnitudesθ = arccos(result) — always in [0, π]

The sign of the dot product encodes the geometric relationship between two vectors: positive means the angle is acute (vectors point roughly the same way), zero means perpendicular (90°), and negative means obtuse (vectors point roughly opposite).

Orthogonality (Perpendicularity)

Two vectors \(\vec{v}\) and \(\vec{w}\) are orthogonal (perpendicular), written \(\vec{v} \perp \vec{w}\), if and only if \(\vec{v}\cdot\vec{w} = 0\). This is because \(\cos 90° = 0\).

PropertyStatement
Commutative\(\vec{v}\cdot\vec{w}=\vec{w}\cdot\vec{v}\)
Distributive\(\vec{u}\cdot(\vec{v}+\vec{w})=\vec{u}\cdot\vec{v}+\vec{u}\cdot\vec{w}\)
Scalar factors out\(\alpha(\vec{v}\cdot\vec{w})=(\alpha\vec{v})\cdot\vec{w}=\vec{v}\cdot(\alpha\vec{w})\)
Self-dot\(\vec{v}\cdot\vec{v}=|\vec{v}|^2\)
Zero dot\(\vec{v}\cdot\vec{0}=0\)
Orthogonality\(\vec{v}\cdot\vec{w}=0 \iff \vec{v}\perp\vec{w}\)
★ Key Insight — Dot Product = Alignment

Positive → angle < 90° (same general direction). Zero → angle = 90° (perpendicular). Negative → angle > 90° (opposing directions). Equal to |v||w| → parallel (same direction). Equal to −|v||w| → antiparallel (opposite direction).

1.8 — Direction Cosines and Projections

In two dimensions, a single angle \(\theta\) determines a vector's direction. In three dimensions, we need three angles — one with each coordinate axis. The cosines of these angles are called direction cosines.

Direction Cosines
\[\cos\alpha=\frac{a}{|\vec{v}|},\quad\cos\beta=\frac{b}{|\vec{v}|},\quad\cos\gamma=\frac{c}{|\vec{v}|}\]
\[\cos^2\alpha+\cos^2\beta+\cos^2\gamma=1\]
α — angle with x-axisβ — angle with y-axisγ — angle with z-axisa, b, c — components of v|v| — magnitude

Each direction cosine is just the dot product of the unit vector \(\hat{v}\) with the corresponding basis vector. The identity \(\cos^2\alpha + \cos^2\beta + \cos^2\gamma = 1\) follows because the unit vector has magnitude 1. Any vector can be reconstructed from its direction cosines: \(\vec{v} = |\vec{v}|(\cos\alpha,\cos\beta,\cos\gamma)\).

Projections

One of the most important applications of the dot product is projection: decomposing one vector into a component parallel to another and a component perpendicular to it. Think of it as finding the "shadow" of one vector onto another.

Scalar Projection (Component)
\[\text{comp}_{\vec{v}}\,\vec{u} = \frac{\vec{u}\cdot\vec{v}}{|\vec{v}|}\]
comp_v u — signed length of shadow of u onto vu · v — dot product|v| — magnitude of v (the direction you project onto)positive if same direction, negative if opposite
Vector Projection
\[\text{proj}_{\vec{v}}\,\vec{u} = \frac{\vec{u}\cdot\vec{v}}{|\vec{v}|^2}\;\vec{v}\]
proj_v u — the vector component of u parallel to vu · v / |v|² — the scalar multiplierv — the direction being projected ontoNote: |v|² = v · v

The vector projection gives the part of \(\vec{u}\) that lies along \(\vec{v}\). The remaining part, \(\vec{u} - \text{proj}_{\vec{v}}\vec{u}\), is perpendicular to \(\vec{v}\). A key physical application is work: when a force \(\vec{F}\) moves an object through displacement \(\vec{d}\), only the component of force along the displacement does work.

Work Done by a Force
\[W = \vec{F} \cdot \vec{d} = |\vec{F}|\,|\vec{d}|\cos\theta\]
W — work (scalar)F — force vectord — displacement vectorθ — angle between F and d
★ Exam Tip

Reconstruct any vector from direction cosines: \(\vec{v} = r(\cos\alpha,\cos\beta,\cos\gamma)\) where \(r = |\vec{v}|\). For projections, remember the denominator: scalar projection uses \(|\vec{v}|\), vector projection uses \(|\vec{v}|^2\) (equivalently \(\vec{v}\cdot\vec{v}\)).

Example — Vector Projection

Find the projection of \(\vec{u}=(3,4)\) onto \(\vec{v}=(1,0)\).

Show Solution

Thought process: Projecting onto \((1,0)\) means finding the x-component of \(\vec{u}\).

1
\(\vec{u}\cdot\vec{v} = (3)(1)+(4)(0) = 3\)
2
\(|\vec{v}|^2 = 1\)
✓
Answer: \(\text{proj}_{\vec{v}}\vec{u} = \frac{3}{1}(1,0) = (3,0)\). The component of \(\vec{u}\) along the x-axis is 3, as expected.

1.9 — The Cross Product

The cross product answers a fundamental question: given two vectors in \(\mathbb{R}^3\), find a vector perpendicular to both of them. Unlike the dot product (which gives a scalar), the cross product gives a vector. It is defined only in \(\mathbb{R}^3\).

The cross product is derived by requiring \(\vec{n}\cdot\vec{v}=0\) and \(\vec{n}\cdot\vec{w}=0\) simultaneously, then solving the resulting system of equations. The result can be computed using a 3×3 determinant.

Cross Product — Component Form
\[\vec{v}\times\vec{w} = (v_2w_3 - v_3w_2,\; v_3w_1 - v_1w_3,\; v_1w_2 - v_2w_1)\]
v = (v₁, v₂, v₃) — first vectorw = (w₁, w₂, w₃) — second vectorresult is a vector ⊥ to both v and w
Cross Product — Determinant Form
\[\vec{v}\times\vec{w}=\begin{vmatrix}\mathbf{i}&\mathbf{j}&\mathbf{k}\\v_1&v_2&v_3\\w_1&w_2&w_3\end{vmatrix}\]
i, j, k — standard basis vectorsexpand along the first rowi-component: v₂w₃ − v₃w₂j-component: −(v₁w₃ − v₃w₁)k-component: v₁w₂ − v₂w₁

The direction of the cross product follows the right-hand rule: point your fingers along \(\vec{v}\), curl them toward \(\vec{w}\), and your thumb points in the direction of \(\vec{v}\times\vec{w}\). Quick reference: \(\mathbf{i}\times\mathbf{j}=\mathbf{k}\), \(\mathbf{j}\times\mathbf{k}=\mathbf{i}\), \(\mathbf{k}\times\mathbf{i}=\mathbf{j}\).

Cross Product — Magnitude
\[|\vec{v}\times\vec{w}| = |\vec{v}|\,|\vec{w}|\sin\theta\]
|v|, |w| — magnitudesθ — angle between v and w|v × w| = area of parallelogram spanned by v and w
PropertyStatement
Anticommutative\(\vec{v}\times\vec{w}=-(\vec{w}\times\vec{v})\) — order matters!
Distributive\(\vec{u}\times(\vec{v}+\vec{w})=\vec{u}\times\vec{v}+\vec{u}\times\vec{w}\)
Scalar factors out\(c(\vec{v}\times\vec{w})=(c\vec{v})\times\vec{w}=\vec{v}\times(c\vec{w})\)
Self-cross\(\vec{v}\times\vec{v}=\vec{0}\)
Zero cross\(\vec{v}\times\vec{0}=\vec{0}\)
Parallel test\(\vec{v}\times\vec{w}=\vec{0}\iff\vec{v}\parallel\vec{w}\)
NOT associative\(\vec{u}\times(\vec{v}\times\vec{w}) \neq (\vec{u}\times\vec{v})\times\vec{w}\) in general
★ Right-Hand Rule Memory Aid

Fingers along \(\vec{v}\), curl toward \(\vec{w}\) → thumb = \(\vec{v}\times\vec{w}\). Cyclic: \(\mathbf{i}\to\mathbf{j}\to\mathbf{k}\to\mathbf{i}\) (forward = positive). Reversing any pair flips the sign.

1.10 — Geometry of Cross Products

The cross product has beautiful geometric applications. Since \(|\vec{v}\times\vec{w}| = |\vec{v}||\vec{w}|\sin\theta\), and the area of a parallelogram with sides of length \(|\vec{v}|\) and \(|\vec{w}|\) separated by angle \(\theta\) is exactly \(|\vec{v}||\vec{w}|\sin\theta\), the magnitude of the cross product equals the parallelogram area. A triangle is half a parallelogram, so:

Parallelogram Area
\[\text{Area}_{\text{parallelogram}} = |\vec{u}\times\vec{v}|\]
u, v — the two edge vectors|u × v| = |u| |v| sin θ
Triangle Area
\[\text{Area}_{\triangle} = \tfrac{1}{2}|\vec{u}\times\vec{v}|\]
u, v — two edge vectors from the same vertex½ because a triangle is half a parallelogram

The Scalar Triple Product

The scalar triple product combines a dot product and a cross product to compute the volume of a parallelepiped (a "tilted box" whose faces are parallelograms). It also provides a quick test for coplanarity.

Scalar Triple Product (Volume)
\[V = |\vec{u}\cdot(\vec{v}\times\vec{w})| = \left|\det\begin{pmatrix}u_1&u_2&u_3\\v_1&v_2&v_3\\w_1&w_2&w_3\end{pmatrix}\right|\]
u, v, w — three edge vectors from the same vertexv × w — cross product (gives base area and normal)u · (v × w) — dot with third vector (gives signed volume)|...| — absolute value gives positive volume= 0 if and only if the three vectors are coplanar

The scalar triple product equals the determinant of the 3×3 matrix whose rows are the three vectors. A zero triple product means the vectors all lie in the same plane (coplanar), so the parallelepiped has zero volume — it's "flat."

Lines, Planes & Parametric Curves

From the vector toolkit of Chapter 1, we now describe geometric objects — lines, planes, and curves — in 2D and 3D space. This chapter also introduces parametric and polar representations, with applications to area and arc length.

§2.1–2.5   Lines and Planes

In single-variable calculus, a line is \(y=mx+b\). In multivariable calculus, we need a more flexible description that works in 3D. The key idea: a line is fully determined by any point on it plus a direction to move along.

Lines in Space

Line (Parametric Equation)

A line is determined by a point \(P\) and a direction vector \(\vec{v}\) parallel to the line. As the parameter \(t\) varies over all real numbers, the point \(\vec{r}(t)\) traces out every point on the line:

\[\vec{r}(t) = P + t\,\vec{v},\quad t\in\mathbb{R}\]
P — any known point on the line (the "anchor")v⃗ — direction vector (any vector parallel to the line)t — parameter (t=0 gives point P, t=1 gives P+v⃗)r⃗(t) — position vector of the point at parameter t

In 2D this gives \(\vec{r}(t)=(x,y)\) with two component equations. In 3D: \(\vec{r}(t)=(x,y,z)\), which means \(x=p_1+tv_1,\;y=p_2+tv_2,\;z=p_3+tv_3\).

If you're given two points on the line instead of a direction, compute the direction as the difference: \(\vec{v}=Q-P\). The direction vector is not unique — any scalar multiple works.

A line through point P in direction v⃗. Dashed extends mean t goes to ±∞.
★ 2D Line Shortcut

For a 2D line \(ax+by=c\), a direction vector is \((-b,\,a)\). This is perpendicular to the normal \((a,b)\).

Example 8 — Line Through Two Points

Find the equation of the line through \(P(1,2,3)\) and \(Q(3,1,1)\).

Solution

Thought process: We have two points but no direction vector. To use the parametric form \(\vec{r}=P+t\vec{v}\), we need a direction — compute it as \(Q-P\).

Setup: Point: \(P(1,2,3)\). Direction: \(\vec{v}=Q-P\).

1
Compute direction vector: \(\vec{v}=Q-P=(3-1,\;1-2,\;1-3)=(2,-1,-2)\)
2
Write parametric equation: \(\vec{r}(t)=(1,2,3)+t(2,-1,-2)\)
3
Component form: \(\boxed{x=1+2t,\quad y=2-t,\quad z=3-2t,\quad t\in\mathbb{R}}\)

★ Tips: Check your work: at \(t=0\) you should get \(P\), at \(t=1\) you should get \(Q\). Here: \(t=0\to(1,2,3)=P\) ✓, \(t=1\to(3,1,1)=Q\) ✓. Any scalar multiple of \(\vec{v}\) gives the same line — \((2,-1,-2)\) and \((-4,2,4)\) both work.

Line Segment

To trace only the portion of a line between \(P\) and \(Q\), restrict \(t\) to \([0,1]\):

\[\vec{r}(t) = (1-t)\,\vec{P}+t\,\vec{Q},\quad 0\le t\le 1\]
t=0 → at point P (start)t=1 → at point Q (end)t=½ → midpoint of PQ

This is just the parametric line formula rearranged: \(P+t(Q-P)=(1-t)P+tQ\).

Line segment from P to Q. Parameter t interpolates between endpoints.

Relations Between Two Lines in 3D

In 2D, two distinct lines either intersect or are parallel. In 3D there's a third option: skew lines, which are neither parallel nor intersecting — they pass each other in space without touching.

Classifying Two Lines

Given lines \(L_1\) with direction \(\vec{v}_1\) and \(L_2\) with direction \(\vec{v}_2\):

RelationHow to Test
Parallel\(\vec{v}_1\) and \(\vec{v}_2\) are scalar multiples (or equivalently, \(\vec{v}_1\times\vec{v}_2=\vec{0}\))
IntersectingNot parallel, AND solving the system for parameters gives consistent z-values
SkewNot parallel AND no intersection (z-values disagree after solving x,y)
Skew lines: neither parallel nor intersecting. Only possible in 3D.
Example 9 — Showing Two Lines Intersect

Show that \(L_1: x=2-t,\;y=-1-t,\;z=4-t\) and \(L_2: x=5-2s,\;y=-s,\;z=1+s\) intersect at \((1,-2,3)\).

Solution

Thought process: Different parameters (\(t\) for \(L_1\), \(s\) for \(L_2\)). Set the x-components equal and y-components equal to get two equations in two unknowns. Then CHECK if the z-components also agree — this is the critical step that distinguishes intersection from skew.

Setup: Set \(x_1=x_2\) and \(y_1=y_2\) to solve for \(t\) and \(s\). Then verify with z.

1
From \(x=x\): \(2-t=5-2s \;\Rightarrow\; 2s-t=3\) — equation (i)
2
From \(y=y\): \(-1-t=-s \;\Rightarrow\; s-t=1\) — equation (ii)
3
Subtract (ii) from (i): \(s=2\). Back-substitute: \(t=1\).
4
Verify z: \(L_1\): \(z=4-1=3\). \(L_2\): \(z=1+2=3\). Equal ✓
5
Point: \(L_1\) at \(t=1\): \((2-1,\;-1-1,\;4-1)=\boxed{(1,-2,3)}\) ✓

★ Tips: Always use different parameter names for different lines. The z-check is what makes this 3D — in 2D, two non-parallel lines always intersect.

Example 10 — Showing Lines Are Skew

Show that \(L_1: x=1+t,\;y=-2+3t,\;z=4-t\) and \(L_2: x=2s,\;y=3+s,\;z=-3+4s\) are skew.

Solution

Thought process: First check if they're parallel (compare direction vectors). If not, try to find an intersection. If that fails, they're skew.

1
Check parallel: \(\vec{v}_1=(1,3,-1)\), \(\vec{v}_2=(2,1,4)\). Not scalar multiples → not parallel.
2
From \(x=x\): \(1+t=2s\). From \(y=y\): \(-2+3t=3+s\). Solving: \(s=1.6,\;t=2.2\).
3
Verify z: \(L_1\): \(z=4-2.2=1.8\). \(L_2\): \(z=-3+4(1.6)=3.4\). Not equal → no intersection.
4
Not parallel + no intersection → \(\boxed{\text{skew lines}}\).

★ Tips: Skew lines only exist in 3D. If a problem says "in the plane" or you're working in 2D, the answer is never skew. The z-check failing is your proof of skewness.

Planes in 3D

While a line is defined by a direction along it, a plane is defined by a direction perpendicular to it. This perpendicular direction is called the normal vector.

Plane (General Equation)

A plane \(\Pi\) is determined by a point \(P(p_1,p_2,p_3)\) and a normal vector \(\vec{n}=(a,b,c)\) perpendicular to the plane:

\[ax+by+cz=d,\quad\text{where }d=ap_1+bp_2+cp_3\]
n⃗ = (a,b,c) — normal vector (perpendicular to the plane)d — constant, found by substituting any known point(a,b,c) are read directly from the normal vector

Any vector perpendicular to the plane works as a normal. If you have three points on the plane, find two in-plane vectors (e.g. \(\vec{PQ}\) and \(\vec{PR}\)), then take their cross product (recall from §1.9) to get the normal: \(\vec{n}=\vec{PQ}\times\vec{PR}\).

A plane defined by point P, with in-plane vectors PQ and PR. The normal n⃗ is perpendicular to the plane.
Example 12 — Plane Through Three Points

Find the equation of the plane through \(P(1,2,3)\), \(Q(-3,2,1)\), and \(R(2,4,5)\).

Solution

Thought process: Three points → need normal vector. Get two in-plane vectors from the three points, then cross product gives the normal. Finally, substitute any point to find \(d\).

Setup: Find \(\vec{PQ}\) and \(\vec{PR}\), cross them, then use the plane equation \(ax+by+cz=d\).

1
In-plane vectors: \(\vec{PQ}=Q-P=(-4,0,-2)\), \(\vec{PR}=R-P=(1,2,2)\)
2
Normal (cross product via determinant): \(\vec{n}=\vec{PQ}\times\vec{PR}=\begin{vmatrix}\vec{i}&\vec{j}&\vec{k}\\-4&0&-2\\1&2&2\end{vmatrix}=(0\cdot2-(-2)\cdot2,\;(-2)\cdot1-(-4)\cdot2,\;(-4)\cdot2-0\cdot1)=(4,6,-8)\)
3
Simplify: \(\vec{n}=2(2,3,-4)\), so use \((2,3,-4)\). Plane: \(2x+3y-4z=d\).
4
Sub \(P(1,2,3)\): \(d=2(1)+3(2)-4(3)=2+6-12=-4\)
5
\(\boxed{2x+3y-4z=-4}\)

★ Tips: Verify by substituting all three points. The normal is not unique — \((4,6,-8)\) and \((2,3,-4)\) give the same plane. Always simplify to avoid large coefficients.

Example 13 — Intersection of a Line and a Plane

Find the intersection between \(L: x=1+t,\;y=-2+3t,\;z=4-t\) and the plane \(3x+5y+8z=5\).

Solution

Thought process: Substitute the parametric line equations into the plane equation. This gives one equation in one unknown (\(t\)). Solve for \(t\), then plug back into the line to find the point.

1
Substitute into plane: \(3(1+t)+5(-2+3t)+8(4-t)=5\)
2
Expand: \(3+3t-10+15t+32-8t=5\;\Rightarrow\;10t+25=5\;\Rightarrow\;t=-2\)
3
Sub \(t=-2\) into line: \((1+(-2),\;-2+3(-2),\;4-(-2))=\boxed{(-1,-8,6)}\)

★ Tips: If \(t\) has no solution (contradiction like \(0=5\)), the line is parallel to the plane. If \(0=0\), the line lies entirely in the plane.

Angle and Intersection of Two Planes

Angle Between Two Planes

The angle between two planes equals the angle between their normal vectors:

\[\cos\theta=\frac{\vec{n}_1\cdot\vec{n}_2}{|\vec{n}_1|\,|\vec{n}_2|}\]
n⃗₁, n⃗₂ — normal vectors of the two planesθ — angle between the planes (same formula as angle between vectors, §1.7)

Two planes are parallel if their normal vectors are parallel (\(\vec{n}_1\times\vec{n}_2=\vec{0}\)).

Intersection of Two Planes

Two non-parallel planes intersect in a line. To find this line:

1. The direction of the line is \(\vec{n}_1\times\vec{n}_2\) (perpendicular to both normals).

2. To find one point on the line, set one variable to zero (e.g. \(z=0\)) and solve the two plane equations as a 2×2 system.

Two non-parallel planes intersect in a line. Direction from cross product of normals.
Example 16 — Intersection of Two Planes

Find the intersection of \(x-3y-2z=2\) and \(2x+y+3z=1\).

Solution

Thought process: Two non-parallel planes → intersection is a line. Get direction from cross product of normals, then find one point by setting a variable to zero.

1
Normals: \(\vec{n}_1=(1,-3,-2)\), \(\vec{n}_2=(2,1,3)\).
2
Direction: \(\vec{n}_1\times\vec{n}_2=(-7,-7,-7+0)=(-7,7,7)=7(-1,1,1)\). Use \((-1,1,1)\).
3
Find a point: set \(z=0\). System: \(x-3y=2,\;2x+y=1\). Multiply 2nd by 3: \(6x+3y=3\). Add to 1st: \(7x=5\Rightarrow x=5/7,\;y=-3/7\).
4
\(\boxed{x=\tfrac{5}{7}-t,\quad y=-\tfrac{3}{7}+t,\quad z=t}\)

★ Tips: Setting \(z=0\) is a convenience choice — any variable works. If it leads to no solution (rare), try a different variable. The direction vector can be simplified by removing common factors.

§2.6   Rotations and Translations in the Plane

Rotations and translations are coordinate transformations — they change how we describe points. An active rotation physically moves the point while keeping the axes fixed. A passive rotation keeps the point fixed but rotates the coordinate axes. Translations shift the origin to a new location.

Active rotation moves the point (uses A). Passive rotation rotates the axes (uses A⁻¹). Same θ, different matrices.
Active Rotation (Rotate the Point)

Rotate point \((x,y)\) counterclockwise by angle \(\theta\) about the origin:

\[\begin{pmatrix}x'\\y'\end{pmatrix}=\begin{pmatrix}\cos\theta&-\sin\theta\\\sin\theta&\cos\theta\end{pmatrix}\quad\begin{pmatrix}x\\y\end{pmatrix}\]
(x,y) — original point(x',y') — image after rotationθ — counterclockwise rotation angleA — rotation matrix (det A = 1, orthogonal)
Passive Rotation (Rotate the Axes)

Keep the point fixed, rotate the coordinate axes counterclockwise by \(\theta\). The point's coordinates in the new frame are:

\[\begin{pmatrix}x'\\y'\end{pmatrix}=\begin{pmatrix}\cos\theta&\sin\theta\\-\sin\theta&\cos\theta\end{pmatrix}\quad\begin{pmatrix}x\\y\end{pmatrix}\]
A⁻¹ — inverse rotation matrix (transpose of A)Signs on sin terms are flipped compared to active rotation
Translation of Origin

Move the origin to a new point \((h,k)\). The point's coordinates in the new frame:

\[\begin{pmatrix}x'\\y'\end{pmatrix}=\begin{pmatrix}x-h\\y-k\end{pmatrix}\]
(h,k) — new origin location in the old frame(x',y') — coordinates in the new frame
★ Combined Transformations

Translation + Active rotation: First translate, then rotate: \(\binom{x'}{y'}=A\binom{x-h}{y-k}\).

Translation + Passive rotation: First translate, then apply \(A^{-1}\): \(\binom{x'}{y'}=A^{-1}\binom{x-h}{y-k}\).

Order matters! Translation is always applied first (subtract the new origin), then rotation.

Example 17 — Active Rotation

Find the image of \((-6,2)\) after a counterclockwise rotation by \(\theta=\pi/3\).

Solution

Thought process: Active rotation — use \(A\) (not \(A^{-1}\)). Plug \(\theta=\pi/3\) into the rotation matrix and multiply.

1
\(\cos(\pi/3)=1/2\), \(\sin(\pi/3)=\sqrt{3}/2\). Matrix: \(A=\begin{pmatrix}1/2&-\sqrt{3}/2\\\sqrt{3}/2&1/2\end{pmatrix}\)
2
\(\begin{pmatrix}x'\\y'\end{pmatrix}=\begin{pmatrix}1/2&-\sqrt{3}/2\\\sqrt{3}/2&1/2\end{pmatrix}\begin{pmatrix}-6\\2\end{pmatrix}=\begin{pmatrix}-3-\sqrt{3}\\2\cdot\sqrt{3}/2+(-6)\cdot\sqrt{3}/2+1\end{pmatrix}\)
3
\(x'=(-6)(1/2)+(2)(-\sqrt{3}/2)=-3-\sqrt{3}\). \(y'=(-6)(\sqrt{3}/2)+(2)(1/2)=-3\sqrt{3}+1\).
4
\(\boxed{(x',y')=(-3-\sqrt{3},\;-3\sqrt{3}+1)}\)

★ Tips: Active vs passive: if the point moves, use \(A\). If the axes rotate, use \(A^{-1}\). The only difference is the sign on the \(\sin\theta\) terms.

§2.7–2.8   Parametric Curves

So far we've described curves as \(y=f(x)\). But many curves can't be written this way — a circle fails the vertical line test, and curves that cross themselves need multiple branches. Parametric equations solve this by describing both \(x\) and \(y\) as functions of a third variable (the parameter) \(t\).

A parametric curve (x,y) = (f(t), g(t)). Green arrows show the direction of increasing t.
Parametric Curve (Plane Curve)

The set of points \((x,y)=(f(t),\,g(t))\) for \(a\le t\le b\), where \(f\) and \(g\) are continuous functions. As \(t\) increases from \(a\) to \(b\), the point traces out the curve.

(f(a), g(a)) — initial point(f(b), g(b)) — terminal pointt — parameter (often represents time)

Conic Sections

The three classical conics — parabolas, ellipses, and hyperbolas — are important curves that appear throughout calculus. Each has a standard Cartesian equation and a natural parametrization.

Standard Conics — Equations and Parametrizations
ConicStandard FormParametrizationKey Relation
Parabola\((x-x_0)^2=4p(y-y_0)\)\(x=t,\; y=\tfrac{1}{4p}(t-x_0)^2+y_0\)Focus at distance \(p\) from vertex
Ellipse\(\frac{(x-x_0)^2}{a^2}+\frac{(y-y_0)^2}{b^2}=1\)\(x=x_0+a\cos t,\; y=y_0+b\sin t\)\(c^2=a^2-b^2\), \(a\ge b>0\)
Hyperbola\(\frac{(x-x_0)^2}{a^2}-\frac{(y-y_0)^2}{b^2}=1\)\(x=x_0+a\sec t,\; y=y_0+b\tan t\)\(c^2=a^2+b^2\)

The ellipse parametrization uses \(\cos^2 t+\sin^2 t=1\). The hyperbola alternatively uses \(x=x_0+a\cosh t,\;y=y_0+b\sinh t\) via the identity \(\cosh^2 t-\sinh^2 t=1\).

The three conic sections with key features: focus, directrix, foci, semi-axes, asymptotes.
General Conic Classification

For the general second-degree equation \(Ax^2+Bxy+Cy^2+Dx+Ey+F=0\), the discriminant \(\Delta=B^2-4AC\) determines the type:

\(B^2-4AC\)Type
\(=0\)Parabola (or degenerate: two parallel lines, one line, or empty)
\(>0\)Hyperbola (or two intersecting lines)
\(<0\)Ellipse (or circle, a point, or empty)

Calculus of Parametric Curves

To find slopes, tangent lines, and concavity of parametric curves, we use the chain rule. The key insight: \(dy/dx\) is the ratio of the rates \(dy/dt\) and \(dx/dt\).

First Derivative — Tangent Slope
\[\frac{dy}{dx}=\frac{dy/dt}{dx/dt},\quad\text{provided }\frac{dx}{dt}\neq 0\]
dy/dt — rate of change of y with respect to the parameter tdx/dt — rate of change of x with respect to the parameter tdy/dx — slope of the tangent line to the curve at the point
Second Derivative — Concavity
\[\frac{d^2y}{dx^2}=\frac{\frac{d}{dt}\!\left(\frac{dy}{dx}\right)}{\frac{dx}{dt}}\]
Numerator — differentiate the first derivative dy/dx with respect to tDenominator — same dx/dt as befored²y/dx² > 0 → concave up; < 0 → concave down

Do NOT compute \(d^2y/dx^2\) as \((d^2y/dt^2)/(d^2x/dt^2)\) — this is a common error. The correct formula differentiates the first derivative, not the numerator/denominator separately.

★ Horizontal and Vertical Tangents

Horizontal tangent: \(dy/dt=0\) AND \(dx/dt\neq 0\) → slope is zero.

Vertical tangent: \(dx/dt=0\) AND \(dy/dt\neq 0\) → slope is undefined (vertical).

Both zero: If \(dy/dt=0\) and \(dx/dt=0\) simultaneously → the tangent is indeterminate. Need L'Hôpital's rule or further analysis (could be a cusp, corner, or smooth point).

Horizontal tangent when numerator dy/dt vanishes. Vertical when denominator dx/dt vanishes.
Example 27 — Tangent Lines to a Parametric Curve

Find the tangent line(s) to \(x=t^5-4t^3,\;y=t^2\) at the point \((0,4)\).

Solution

Thought process: We need the slope at a specific point. First find which value(s) of \(t\) give \((0,4)\) — there may be more than one (the curve might pass through this point twice). Then compute \(dy/dx\) at each \(t\).

Setup: From \(y=t^2=4\Rightarrow t=\pm 2\). Check: \(x=(\pm 2)^5-4(\pm 2)^3=\pm 32\mp 32=0\) ✓. Both work.

1
\(\frac{dy}{dx}=\frac{2t}{5t^4-12t^2}=\frac{2}{5t^3-12t}\) (after cancelling \(t\neq 0\))
2
At \(t=-2\): slope \(=\frac{2}{5(-8)-12(-2)}=\frac{2}{-40+24}=\frac{2}{-16}=-\frac{1}{8}\)
3
At \(t=2\): slope \(=\frac{2}{5(8)-12(2)}=\frac{2}{40-24}=\frac{2}{16}=\frac{1}{8}\)
4
Tangent at \(t=-2\): \(y-4=-\frac{1}{8}(x-0)\Rightarrow\boxed{y=4-x/8}\)
5
Tangent at \(t=2\): \(\boxed{y=4+x/8}\)

★ Tips: A point on a parametric curve can correspond to multiple \(t\)-values — always check all of them. Each gives a different tangent line, creating a self-intersection.

Example 28 — Horizontal and Vertical Tangents

Find where \(x=t^3-3t,\;y=3t^2-9\) has horizontal or vertical tangents.

Solution

Thought process: Horizontal tangent: set \(dy/dt=0\) (numerator of slope). Vertical tangent: set \(dx/dt=0\) (denominator of slope). In each case verify the other derivative is nonzero.

1
\(dy/dt=6t\), \(dx/dt=3t^2-3=3(t^2-1)\)
2
Horizontal: \(6t=0\Rightarrow t=0\). Check: \(dx/dt|_{t=0}=-3\neq 0\) ✓. Point: \((0-0,\;0-9)=\boxed{(0,-9)}\).
3
Vertical: \(3(t^2-1)=0\Rightarrow t=\pm 1\). Check: \(dy/dt|_{t=1}=6\neq 0\) ✓, \(dy/dt|_{t=-1}=-6\neq 0\) ✓.
4
Points: \(t=1\to(1-3,\;3-9)=\boxed{(-2,-6)}\), \(t=-1\to(-1+3,\;3-9)=\boxed{(2,-6)}\).

★ Tips: At \(t=0\), \(dy/dt=0\) but \(dx/dt\neq 0\) → horizontal. At \(t=\pm 1\), \(dx/dt=0\) but \(dy/dt\neq 0\) → vertical. Always verify the "other" derivative isn't also zero.

§2.9   Applications to Area Problems

In Calculus I, the area under \(y=f(x)\) from \(a\) to \(b\) is \(\int_a^b f(x)\,dx\). When the curve is parametric, we substitute \(x=f(t)\) and \(dx=f'(t)\,dt\) to convert everything to the parameter \(t\).

Area Under a Parametric Curve
\[A=\int_\alpha^\beta y(t)\,x'(t)\,dt\]
y(t) — height of the curve at parameter tx'(t) dt — infinitesimal width element (replaces dx)α, β — parameter values where x(α)=a and x(β)=b

This comes directly from \(\int_a^b y\,dx\) with the substitution \(dx=x'(t)\,dt\).

Area Enclosed by a Closed Parametric Curve

For a closed curve \(C: x=f(t),\;y=g(t),\;\alpha\le t\le\beta\) with \(f(\alpha)=f(\beta),\;g(\alpha)=g(\beta)\):

\[A(R)=\left|\int_\alpha^\beta g(t)\,f'(t)\,dt\right|\]
g(t) — y-component of the curvef'(t) — derivative of x-componentAbsolute value ensures positive area regardless of orientation
Cycloid: path traced by a point on a circle rolling along a line.
Example 30 — Area Under One Arch of a Cycloid

Find the area under one arch of the cycloid: \(x=r(t-\sin t),\;y=r(1-\cos t)\).

Solution

Thought process: The cycloid traces arches, each from \(y=0\) back to \(y=0\). One arch: \(t=0\) to \(t=2\pi\). Use the parametric area formula \(\int y\,x'\,dt\).

Setup: \(y=r(1-\cos t)\), \(x'=r(1-\cos t)\). Bounds: \(0\le t\le 2\pi\).

1
\(A=\int_0^{2\pi}r(1-\cos t)\cdot r(1-\cos t)\,dt=r^2\int_0^{2\pi}(1-\cos t)^2\,dt\)
2
Expand: \((1-\cos t)^2=1-2\cos t+\cos^2 t=1-2\cos t+\frac{1+\cos 2t}{2}\) (half-angle identity from Pre-knowledge)
3
\(=\frac{3}{2}-2\cos t+\frac{\cos 2t}{2}\). Integrate over \([0,2\pi]\): \(\int_0^{2\pi}\cos t\,dt=0\), \(\int_0^{2\pi}\cos 2t\,dt=0\).
4
\(A=r^2\cdot\frac{3}{2}\cdot 2\pi=\boxed{3\pi r^2}\)

★ Tips: The cycloid arch area equals three times the area of the generating circle (\(\pi r^2\)). Memorize this result — it's a classic. The key trick is expanding \((1-\cos t)^2\) using the half-angle formula.

Example 31 — Area of an Ellipse

Find the area of the ellipse \(x=a\cos t,\;y=b\sin t,\;0\le t\le 2\pi\).

Solution

Thought process: Closed curve → use \(|{\int g(t)f'(t)\,dt}|\). Here \(f'(t)=-a\sin t\).

1
\(A=\left|\int_0^{2\pi}(b\sin t)(-a\sin t)\,dt\right|=ab\int_0^{2\pi}\sin^2 t\,dt\)
2
Half-angle: \(\sin^2 t=\frac{1-\cos 2t}{2}\). So \(\int_0^{2\pi}\sin^2 t\,dt=\pi\).
3
\(\boxed{A=\pi ab}\)

★ Tips: When \(a=b=r\), this gives \(\pi r^2\) — the familiar circle area. The half-angle identity \(\sin^2 t=(1-\cos 2t)/2\) is essential for parametric area problems.

§2.10   Arc Length

The arc length formula comes from summing infinitesimal line segments: each tiny piece has length \(\sqrt{dx^2+dy^2}=\sqrt{(dx/dt)^2+(dy/dt)^2}\,dt\).

Each infinitesimal piece ds forms a right triangle: ds² = dx² + dy² (Pythagorean theorem).
Arc Length of a Parametric Curve
\[L=\int_\alpha^\beta\sqrt{\left(\frac{dx}{dt}\right)^2+\left(\frac{dy}{dt}\right)^2}\;dt\]
dx/dt, dy/dt — component velocitiesThe integrand is the speed: √(x'²+y'²)Curve must be traced exactly once over [α,β]

For curves given as \(y=f(x)\): set \(x=t\), \(dx/dt=1\), giving \(L=\int_a^b\sqrt{1+[f'(x)]^2}\,dx\).

Example 32 — Arc Length of One Cycloid Arch

Find the length of one arch of the cycloid: \(x=r(t-\sin t),\;y=r(1-\cos t),\;0\le t\le 2\pi\).

Solution

Thought process: Compute \(x'(t)\) and \(y'(t)\), then simplify \(\sqrt{(x')^2+(y')^2}\) using trig identities before integrating.

1
\(x'=r(1-\cos t),\quad y'=r\sin t\)
2
\((x')^2+(y')^2=r^2[(1-\cos t)^2+\sin^2 t]=r^2[1-2\cos t+\cos^2 t+\sin^2 t]=r^2(2-2\cos t)\)
3
Use identity \(1-\cos t=2\sin^2(t/2)\): \(\sqrt{2-2\cos t}=2|\sin(t/2)|\). On \([0,2\pi]\), \(\sin(t/2)\ge 0\), so absolute value drops.
4
\(L=\int_0^{2\pi}2r\sin(t/2)\,dt=2r\left[-2\cos(t/2)\right]_0^{2\pi}=2r[-2(-1)-(-2)(1)]=2r\cdot 4=\boxed{8r}\)

★ Tips: Cycloid arch length is always \(8r\) — exactly 4 diameters. The key identity is \(1-\cos t=2\sin^2(t/2)\), which converts a square root of a trig expression into something integrable.

§2.11–2.14   Polar Coordinates

Cartesian coordinates locate a point by how far right and how far up it is. Polar coordinates instead specify how far from the origin and at what angle. Curves with rotational symmetry (circles, spirals, roses) have much simpler equations in polar form.

Polar coordinates (r, θ): r is the distance from the origin, θ is the angle from the positive x-axis.

When converting from Cartesian to polar, \(\tan\theta=y/x\) gives two possible angles (arctan only returns \((-\pi/2,\pi/2)\)). Always check which quadrant the point is in to pick the correct \(\theta\). For a point in Q3 like \((-1,-1)\): \(\tan\theta=1\) suggests \(\theta=\pi/4\), but the actual angle is \(\theta=\pi/4+\pi=5\pi/4\).

Example 35 — Coordinate Conversion

(a) Convert \((-4,2\pi/3)\) from polar to Cartesian. (b) Convert \((-1,-1)\) from Cartesian to polar.

Solution

Thought process: (a) Apply \(x=r\cos\theta,\;y=r\sin\theta\) directly. (b) Compute \(r\) from the Pythagorean formula, then determine \(\theta\) using the quadrant.

1
(a) \(x=-4\cos(2\pi/3)=-4(-1/2)=2\), \(y=-4\sin(2\pi/3)=-4(\sqrt{3}/2)=-2\sqrt{3}\). Answer: \(\boxed{(2,-2\sqrt{3})}\).
2
(b) \(r=\sqrt{(-1)^2+(-1)^2}=\sqrt{2}\). \(\tan\theta=(-1)/(-1)=1\). Since the point is in Q3: \(\theta=\pi/4+\pi=5\pi/4\). Answer: \(\boxed{(\sqrt{2},\;5\pi/4)}\).

★ Tips: Negative \(r\) in polar means "go in the opposite direction." The point \((-4,2\pi/3)\) is the same as \((4,2\pi/3+\pi)=(4,5\pi/3)\). Always verify by converting back.

Example 37 — Tangent to a Polar Curve

Find the tangent line to \(r=3+8\sin\theta\) at \(\theta=\pi/6\).

Solution

Thought process: Use the polar slope formula. First find \(r\) and \((x,y)\) at the given \(\theta\), then compute the slope, then write the tangent line equation.

1
At \(\theta=\pi/6\): \(r=3+8\sin(\pi/6)=3+4=7\). Point: \((x,y)=(7\cos(\pi/6),\;7\sin(\pi/6))=(7\sqrt{3}/2,\;7/2)\).
2
\(r'(\theta)=8\cos\theta\). At \(\theta=\pi/6\): \(r'=8\cdot\sqrt{3}/2=4\sqrt{3}\).
3
Slope: \(\frac{4\sqrt{3}\cdot 1/2+7\cdot\sqrt{3}/2}{4\sqrt{3}\cdot\sqrt{3}/2-7\cdot 1/2}=\frac{2\sqrt{3}+7\sqrt{3}/2}{6-7/2}=\frac{11\sqrt{3}/2}{5/2}=\frac{11\sqrt{3}}{5}\)
4
Tangent line: \(y-7/2=\frac{11\sqrt{3}}{5}(x-7\sqrt{3}/2)\;\Rightarrow\;\boxed{y=\frac{11\sqrt{3}}{5}x-\frac{98}{5}}\)

★ Tips: Don't forget to convert \((r,\theta)\) to \((x,y)\) before writing the tangent line — tangent line equations are in Cartesian form.

In polar coordinates, area elements are sectors, not rectangles. A thin sector at radius \(r\) spanning angle \(d\theta\) has area \(\frac{1}{2}r^2\,d\theta\).

Polar area: each infinitesimal sector has area dA = ½r²dθ. Sum from α to β to get total area.
Area of a Polar Region
\[A=\int_\alpha^\beta\frac{1}{2}\,r^2\,d\theta\]
r = f(θ) — the polar curve bounding the regionα, β — bounding anglesComes from sector area formula A = ½r²θ

Example 39 — Area of an Inner Loop

Find the area of the inner loop of \(r=2+4\cos\theta\).

Solution

Thought process: A limaçon \(r=a+b\cos\theta\) with \(|b|>|a|\) has an inner loop. The loop occurs where \(r\) goes negative — find the boundary by setting \(r=0\).

Setup: Find where \(r=0\): \(2+4\cos\theta=0\Rightarrow\cos\theta=-1/2\Rightarrow\theta=2\pi/3,\;4\pi/3\).

1
\(A=\int_{2\pi/3}^{4\pi/3}\frac{1}{2}(2+4\cos\theta)^2\,d\theta\)
2
Expand: \(\frac{1}{2}(4+16\cos\theta+16\cos^2\theta)=2+8\cos\theta+8\cdot\frac{1+\cos 2\theta}{2}\)
3
\(=6+8\cos\theta+4\cos 2\theta\). Integrate: \(\left[6\theta+8\sin\theta+2\sin 2\theta\right]_{2\pi/3}^{4\pi/3}\)
4
\(\boxed{A=4\pi-6\sqrt{3}}\)

★ Tips: For inner loops, always find bounds by setting \(r=0\). The integrand \(r^2\) is always positive even when \(r\) itself is negative (that's why it works). For the half-angle substitution, recall \(\cos^2\theta=(1+\cos 2\theta)/2\) from Pre-knowledge.

Example 40 — Area Between Polar Curves

Find the area outside the cardioid \(r=1+\sin\theta\) and inside the circle \(r=3\sin\theta\).

Solution

Thought process: "Outside A, inside B" → area between the curves with B as the outer curve. First find where they intersect to determine the bounds.

Setup: Intersections: \(1+\sin\theta=3\sin\theta\Rightarrow\sin\theta=1/2\Rightarrow\theta=\pi/6,\;5\pi/6\). On \([\pi/6,5\pi/6]\), verify \(3\sin\theta\ge 1+\sin\theta\) ✓ (the circle is outer).

1
\(A=\int_{\pi/6}^{5\pi/6}\frac{1}{2}\left[(3\sin\theta)^2-(1+\sin\theta)^2\right]d\theta\)
2
\(=\frac{1}{2}\int_{\pi/6}^{5\pi/6}[9\sin^2\theta-1-2\sin\theta-\sin^2\theta]\,d\theta=\frac{1}{2}\int_{\pi/6}^{5\pi/6}[8\sin^2\theta-2\sin\theta-1]\,d\theta\)
3
Use \(\sin^2\theta=(1-\cos 2\theta)/2\): integrand becomes \(3-4\cos 2\theta-2\sin\theta\). Integrate and evaluate.
4
\(\boxed{A=\pi}\)

★ Tips: Always find intersections first — they give your integration bounds. Verify which curve is outer by testing a point between the bounds. The answer \(\pi\) is deceptively clean.

Since \(x=r\cos\theta\) and \(y=r\sin\theta\) are parametric in \(\theta\), we can derive that \((x'_\theta)^2+(y'_\theta)^2=(r'_\theta)^2+r^2\). This simplifies the arc length integral.

Polar Arc Length
\[L=\int_\alpha^\beta\sqrt{\left(\frac{dr}{d\theta}\right)^2+r^2}\;d\theta\]
dr/dθ — rate of change of radius with angler — the polar radius functionSimpler than converting to Cartesian and using √(x'²+y'²)
Example 42 — Length of a Cardioid

Find the length of the cardioid \(r=1+\sin\theta,\;0\le\theta\le 2\pi\).

Solution

Thought process: Use the polar arc length formula. The challenge is simplifying the integrand \(\sqrt{(r')^2+r^2}\) using trig identities.

1
\(r'=\cos\theta\). Integrand: \(\sqrt{\cos^2\theta+(1+\sin\theta)^2}=\sqrt{2+2\sin\theta}\).
2
Use identity \(1+\sin\theta=(\sin(\theta/2)+\cos(\theta/2))^2\): \(\sqrt{2+2\sin\theta}=\sqrt{2}\,|\sin(\theta/2)+\cos(\theta/2)|\).
3
Sub \(\varphi=\theta/2\): \(L=2\sqrt{2}\int_0^\pi|\sin\varphi+\cos\varphi|\,d\varphi\). Split at \(\varphi=3\pi/4\) (where integrand changes sign).
4
\(=2\sqrt{2}\left[\int_0^{3\pi/4}(\sin\varphi+\cos\varphi)\,d\varphi-\int_{3\pi/4}^{\pi}(\sin\varphi+\cos\varphi)\,d\varphi\right]=\boxed{8}\)

★ Tips: The cardioid has length exactly \(8\) — a clean result worth memorizing. The tricky part is handling the absolute value by splitting the integral where \(\sin\varphi+\cos\varphi\) changes sign (at \(\varphi=3\pi/4\)).

Chapter 1 — Vectors

Magnitude
\[|\vec{v}|=\sqrt{\htmlData{t=v₁ — x-component of v}{\color{#D45D00}{v_1}}^2+\htmlData{t=v₂ — y-component of v}{\color{#D45D00}{v_2}}^2+\htmlData{t=v₃ — z-component of v}{\color{#D45D00}{v_3}}^2}\]

The magnitude (or length) of a vector tells you "how long is this arrow?" Think of a vector as an arrow pointing from the origin — the magnitude is the distance from start to tip. For a 2D vector ⟨3, 4⟩, imagine a right triangle with legs 3 and 4: the hypotenuse (the arrow itself) has length √(9+16) = √25 = 5. In 3D, you just add the third component squared under the square root. The magnitude is ALWAYS a non-negative number — you can't have a negative length. The only vector with magnitude zero is the zero vector ⟨0,0,0⟩.

Where it comes from: This is the Pythagorean theorem extended to 3D. In 2D: |v| = √(v₁²+v₂²) is just the distance formula. Adding the z-component gives the 3D version.

✓ Use when you see "find the length/magnitude", "normalize a vector" (step 1), "find the distance between two points" (magnitude of the difference), or "compare which vector is longer."
✗ Not for finding direction — magnitude strips away direction and gives only size. Also, |v|² = v·v (the dot product of v with itself), which is sometimes faster than square-rooting.
Length of a vector — always ≥ 0. Use for normalizing, distances.
Unit Vector
\[\hat{u}=\frac{\htmlData{t=v — the vector being normalized}{\color{#D45D00}{\vec{v}}}}{\htmlData{t=|v| — magnitude of v (must be nonzero)}{\color{#2050A0}{|\vec{v}|}}}\]

A unit vector is a vector with length exactly 1. It keeps the direction of the original vector but throws away its size. Think of it as answering "which way is this arrow pointing?" without caring how long it is. To find it, divide the vector by its own magnitude: if v = ⟨6, 0, 0⟩, then |v| = 6 and û = ⟨6/6, 0/6, 0/6⟩ = ⟨1, 0, 0⟩. The standard basis vectors î = ⟨1,0,0⟩, ĵ = ⟨0,1,0⟩, k̂ = ⟨0,0,1⟩ are unit vectors pointing along each axis.

Where it comes from: Dividing by |v| is scalar multiplication by 1/|v|. This scales the arrow down (or up) until its length is exactly 1, without changing its direction. Important: the zero vector ⟨0,0,0⟩ has NO unit vector because you'd divide by zero.

✓ Use when asked to "normalize", "find the direction of", or "find a unit vector in the direction of." Also essential for projections (proj formula uses v/|v|) and for expressing vectors as magnitude × direction: v = |v| · û.
✗ Not for the zero vector (undefined). Don't confuse normalizing (dividing by magnitude) with making components positive.
Direction only, length 1. Undefined for the zero vector.
Dot Product
\[\htmlData{t=v — first vector}{\color{#D45D00}{\vec{v}}}\cdot\htmlData{t=w — second vector}{\color{#0B7D7D}{\vec{w}}}=\htmlData{t=v₁ — x-component of v}{\color{#D45D00}{v_1}}\htmlData{t=w₁ — x-component of w}{\color{#0B7D7D}{w_1}}+\htmlData{t=v₂ — y-component of v}{\color{#D45D00}{v_2}}\htmlData{t=w₂ — y-component of w}{\color{#0B7D7D}{w_2}}+\htmlData{t=v₃ — z-component of v}{\color{#D45D00}{v_3}}\htmlData{t=w₃ — z-component of w}{\color{#0B7D7D}{w_3}}=\htmlData{t=|v| — magnitude of v}{\color{#2050A0}{|\vec{v}|}}\,\htmlData{t=|w| — magnitude of w}{\color{#2050A0}{|\vec{w}|}}\cos\htmlData{t=θ — angle between v and w}{\color{#A82870}{\theta}}\]

The dot product takes two vectors and produces a single number (a scalar, NOT a vector). It measures how much two vectors point in the same direction. There are two ways to compute it — algebraic: multiply matching components and add (v₁w₁ + v₂w₂ + v₃w₃). Geometric: multiply the magnitudes and the cosine of the angle between them (|v||w|cosθ). Both always give the same answer. The sign tells you alignment: positive means the vectors point roughly the same way (angle < 90°), zero means they're perfectly perpendicular (angle = 90°), negative means they point roughly opposite (angle > 90°).

Where it comes from: Start with the law of cosines: |v−w|² = |v|²+|w|²−2|v||w|cosθ. Expand the left side using components, and the algebraic form emerges. That's why the two forms are equivalent.

✓ Use when you need to: (1) find the angle between two vectors, (2) test if vectors are perpendicular (dot = 0?), (3) compute projections, or (4) calculate work (W = F·d). The dot product is your go-to tool for anything involving angles or alignment.
✗ Not for finding a vector perpendicular to two others (that's the cross product). Remember: dot product = scalar, cross product = vector. Never use dot product when you need a vector result.
Scalar measuring alignment. Positive = same dir, zero = ⊥, negative = opposing.
Angle Between Vectors
\[\cos\htmlData{t=θ — angle between v and w (0 to π)}{\color{#A82870}{\theta}}=\frac{\htmlData{t=v — first vector}{\color{#D45D00}{\vec{v}}}\cdot\htmlData{t=w — second vector}{\color{#0B7D7D}{\vec{w}}}}{\htmlData{t=|v| — magnitude of first vector}{\color{#2050A0}{|\vec{v}|}}\,\htmlData{t=|w| — magnitude of second vector}{\color{#2050A0}{|\vec{w}|}}}\]

This formula finds the angle between any two vectors. It's a rearrangement of the geometric dot product formula. Here's the step-by-step procedure: (1) Compute v·w using the algebraic formula (multiply components, add). (2) Compute |v| and |w| separately. (3) Divide: cosθ = (v·w)/(|v|·|w|). (4) Take arccos to get θ. The result is always between 0 and π (0° to 180°) — the "smaller" angle between the arrows. If cosθ = 0, the angle is π/2 (perpendicular). If cosθ = 1, the angle is 0 (parallel, same direction). If cosθ = −1, the angle is π (parallel, opposite direction).

Where it comes from: Starting from v·w = |v||w|cosθ, just divide both sides by |v||w| to isolate cosθ.

✓ Use when you have two known vectors (given as components) and need the angle between them. This is the standard 4-step method tested on almost every exam.
✗ Not for finding the inclination angle of a single vector with an axis (use arctan or direction cosines for that). Also, both vectors must be nonzero — you can't find an "angle" with the zero vector.
Isolate θ from the dot product. Three-step: dot → magnitudes → arccos.
Direction Cosines
\[\cos\htmlData{t=α — angle v makes with x-axis}{\color{#A82870}{\alpha}}=\frac{\htmlData{t=a — x-component of v}{\color{#5D7A20}{a}}}{\htmlData{t=|v| — magnitude of v}{\color{#2050A0}{|\vec{v}|}}},\;\cos\htmlData{t=β — angle v makes with y-axis}{\color{#A82870}{\beta}}=\frac{\htmlData{t=b — y-component of v}{\color{#5D7A20}{b}}}{\htmlData{t=|v| — magnitude of v}{\color{#2050A0}{|\vec{v}|}}},\;\cos\htmlData{t=γ — angle v makes with z-axis}{\color{#A82870}{\gamma}}=\frac{\htmlData{t=c — z-component of v}{\color{#5D7A20}{c}}}{\htmlData{t=|v| — magnitude of v}{\color{#2050A0}{|\vec{v}|}}}\]

The direction cosines are the cosines of the angles a vector makes with the three coordinate axes (x, y, z). If your vector is v = ⟨a, b, c⟩, then cosα = a/|v| (angle with x-axis), cosβ = b/|v| (angle with y-axis), cosγ = c/|v| (angle with z-axis). These are just the components of the unit vector! The identity cos²α + cos²β + cos²γ = 1 always holds — because the unit vector has magnitude 1. You can reconstruct any vector from its magnitude and direction cosines: v = |v|·(cosα, cosβ, cosγ).

Where it comes from: Each direction cosine is the dot product of the unit vector û with the corresponding basis vector: cosα = û·î = a/|v|.

✓ Use when given angles with axes and asked to find the vector (reconstruct using v = r(cosα, cosβ, cosγ)), or when asked "what angle does this vector make with the x/y/z-axis?"
✗ Not for 2D angles — in 2D, the single inclination angle θ is just arctan(v₂/v₁).
Angles with axes. Reconstruct: v = r(cosα,cosβ,cosγ).
Vector Projection
\[\text{proj}_{\htmlData{t=v — the direction being projected onto}{\color{#0B7D7D}{\vec{v}}}}\,\htmlData{t=u — the vector being projected}{\color{#D45D00}{\vec{u}}}=\frac{\htmlData{t=u — vector being projected}{\color{#D45D00}{\vec{u}}}\cdot\htmlData{t=v — projection direction}{\color{#0B7D7D}{\vec{v}}}}{\htmlData{t=|v|² — squared magnitude of v}{\color{#2050A0}{|\vec{v}|^2}}}\;\htmlData{t=v — multiply by direction vector}{\color{#0B7D7D}{\vec{v}}}\]

Think of vector projection as shining a flashlight straight down onto a ruler and seeing where the shadow falls. If you project u onto v, you're finding the piece of u that lies along the direction of v. There are two versions: Scalar projection (comp_v u) = u·v/|v| → gives a NUMBER (positive means same direction as v, negative means opposite). Vector projection (proj_v u) = (u·v/|v|²)·v → gives a VECTOR parallel to v. The leftover piece u − proj_v u is the part of u that's perpendicular to v. Step by step: (1) compute u·v, (2) compute |v|², (3) divide to get the scalar coefficient, (4) multiply by v.

Where it comes from: We want the point on the line through v that's closest to the tip of u. Setting up the perpendicularity condition (u − tv) ⊥ v gives t = u·v/|v|², so proj = tv.

✓ Use when "project u onto v", "find the component of u along v", decomposing a force into parallel and perpendicular parts, or computing work. Very common in physics problems.
✗ Not for finding the perpendicular component directly — but you can get it by subtracting: u_perp = u − proj_v u.
Shadow of u onto v. Scalar version: u·v/|v|. Vector version: (u·v/|v|²)v.
Work
\[\htmlData{t=W — work done (scalar quantity)}{\color{#5D7A20}{W}}=\htmlData{t=F — force vector}{\color{#D45D00}{\vec{F}}}\cdot\htmlData{t=d — displacement vector}{\color{#0B7D7D}{\vec{d}}}=\htmlData{t=|F| — magnitude of force}{\color{#2050A0}{|\vec{F}|}}\,\htmlData{t=|d| — magnitude of displacement}{\color{#2050A0}{|\vec{d}|}}\cos\htmlData{t=θ — angle between force and displacement}{\color{#A82870}{\theta}}\]

Work measures how much energy a force transfers to an object as it moves. If you push a box with force F across a displacement d, only the part of your force that's in the direction of motion counts. Pushing perfectly forward → maximum work. Pushing at an angle → less work. Pushing perpendicular (like pushing down on something sliding sideways) → zero work. The formula W = F·d = |F||d|cosθ captures this: it's the dot product of force and displacement. Work is a SCALAR — it's measured in joules (or newton-meters). It can be negative if you push against the motion (like friction).

Where it comes from: Work = (force component along displacement) × (distance traveled). The "force component along displacement" is exactly the scalar projection of F onto d, which equals |F|cosθ. Multiply by |d| and you get |F||d|cosθ = F·d.

✓ Use when "find the work done by force F over displacement d", or any physics problem involving energy transfer through forces.
✗ Not for torque (rotational force) — that uses the CROSS product: τ = r × F.
Work = F·d. Only the component of force along displacement matters.
Cross Product
\[\htmlData{t=v — first vector}{\color{#D45D00}{\vec{v}}}\times\htmlData{t=w — second vector}{\color{#0B7D7D}{\vec{w}}}=\begin{vmatrix}\mathbf{i}&\mathbf{j}&\mathbf{k}\\\htmlData{t=v₁ — x-component of v}{\color{#D45D00}{v_1}}&\htmlData{t=v₂ — y-component of v}{\color{#D45D00}{v_2}}&\htmlData{t=v₃ — z-component of v}{\color{#D45D00}{v_3}}\\\htmlData{t=w₁ — x-component of w}{\color{#0B7D7D}{w_1}}&\htmlData{t=w₂ — y-component of w}{\color{#0B7D7D}{w_2}}&\htmlData{t=w₃ — z-component of w}{\color{#0B7D7D}{w_3}}\end{vmatrix}\]

The cross product takes two vectors and produces a NEW VECTOR that's perpendicular to both of them. While the dot product gives a scalar measuring alignment, the cross product gives a vector pointing "out of the plane" formed by the two inputs. To compute it, set up a 3×3 determinant with î, ĵ, k̂ in the first row, v's components in the second row, and w's components in the third row. Expand along the first row: v×w = î(v₂w₃−v₃w₂) − ĵ(v₁w₃−v₃w₁) + k̂(v₁w₂−v₂w₁). The result's direction follows the right-hand rule: curl your fingers from v toward w, and your thumb points in the direction of v×w. Critical: order matters! v×w = −(w×v). Swapping the order flips the sign.

Where it comes from: Expanding the determinant along the first row using cofactor expansion. The formula is constructed specifically so the result is perpendicular to both input vectors.

✓ Use when you need: (1) a normal vector to a plane (cross two in-plane vectors), (2) parallelogram area, (3) to test if vectors are parallel (cross product = 0?), or (4) torque in physics.
✗ Not for testing perpendicularity (use dot product = 0 for that). Cross product is NOT commutative (order matters) and NOT associative. It only works in 3D.
Vector ⊥ to both. |v×w| = parallelogram area. Right-hand rule for direction.
Cross Product — Magnitude
\[|\htmlData{t=v — first vector}{\color{#D45D00}{\vec{v}}}\times\htmlData{t=w — second vector}{\color{#0B7D7D}{\vec{w}}}|=\htmlData{t=|v| — magnitude of v}{\color{#2050A0}{|\vec{v}|}}\,\htmlData{t=|w| — magnitude of w}{\color{#2050A0}{|\vec{w}|}}\sin\htmlData{t=θ — angle between v and w}{\color{#A82870}{\theta}}\]

The magnitude of the cross product equals the area of the parallelogram formed by the two vectors. Picture laying v and w tail-to-tail — they form a parallelogram. The area of that parallelogram is |v||w|sinθ. When the vectors are parallel (θ = 0 or π), sinθ = 0, so the "parallelogram" collapses to a line with zero area — this is why cross product = 0 means parallel. When they're perpendicular (θ = 90°), sinθ = 1, and the area is maximized at |v|·|w|.

Where it comes from: The base of the parallelogram is |v|, and the height is |w|sinθ (the perpendicular distance). Area = base × height = |v||w|sinθ.

✓ Use when you need the parallelogram area or want to know the LENGTH of the cross product vector without computing the full determinant.
✗ Not for finding the direction of the cross product — this formula gives only the magnitude. For the full vector, use the determinant method.
|v×w| = parallelogram area = |v||w|sinθ.
Triangle Area
\[\text{Area}=\tfrac{1}{2}\,|\htmlData{t=u — first edge from shared vertex}{\color{#5D7A20}{\vec{u}}}\times\htmlData{t=v — second edge from shared vertex}{\color{#D45D00}{\vec{v}}}|\]

A triangle formed by two vectors from a shared vertex is exactly half of the parallelogram they span. So: take the two edge vectors, cross them, take the magnitude, and divide by 2. Given three points A, B, C: form vectors AB = B−A and AC = C−A from the same vertex, then Area = ½|AB × AC|. This works in both 2D and 3D — no need to find the base and height separately.

Where it comes from: A parallelogram has area |u×v|. A diagonal cuts it into two congruent triangles, so each triangle has half that area.

✓ Use when you have 3 points in space (or two edge vectors from a vertex) and need the triangle area. Works in any dimension — no need to find base and height.
✗ Not for 2D problems where ½|base × height| is simpler. Also don't forget both vectors must share a vertex — pick any one vertex and form two edges from it.
Half the parallelogram area: ½|u×v|.
Scalar Triple Product (Volume)
\[V=|\htmlData{t=u — height direction vector}{\color{#5D7A20}{\vec{u}}}\cdot(\htmlData{t=v — first base edge}{\color{#D45D00}{\vec{v}}}\times\htmlData{t=w — second base edge}{\color{#0B7D7D}{\vec{w}}})|\]

The scalar triple product u·(v×w) computes the volume of the 3D box (parallelepiped) formed by three vectors. First, v×w gives a vector whose magnitude is the base area. Then dotting with u projects u onto that direction, giving the height. Volume = base area × height = |u·(v×w)|. The absolute value is needed because the sign depends on the orientation (right-handed vs left-handed). Key insight: if the volume is zero, the three vectors are coplanar — they all lie in the same plane (the "box" is flat). This is also equal to the determinant of the 3×3 matrix with u, v, w as rows.

Where it comes from: Base parallelogram area = |v×w|. Height = scalar projection of u onto the direction of v×w. Volume = base × height = |u·(v×w)|.

✓ Use when (1) finding the volume of a parallelepiped from three edges, or (2) testing coplanarity (if triple product = 0, the vectors lie in one plane). Also: tetrahedron volume = ⅙|u·(v×w)|.
✗ Not for tetrahedra directly — their volume is 1/6 of the parallelepiped, so divide by 6.
|u·(v×w)| = parallelepiped volume. = 0 ⟺ coplanar.
Quick Geometry Tests
\[\text{Perp: }\htmlData{t=v — first vector (perp test)}{\color{#D45D00}{\vec{v}}}\cdot\htmlData{t=w — second vector}{\color{#0B7D7D}{\vec{w}}}=0\quad\text{Parallel: }\htmlData{t=v — first vector (parallel test)}{\color{#D45D00}{\vec{v}}}\times\htmlData{t=w — second vector}{\color{#0B7D7D}{\vec{w}}}=\mathbf{0}\quad\text{Coplanar: }\htmlData{t=u — third vector (coplanar test)}{\color{#5D7A20}{\vec{u}}}\cdot(\htmlData{t=v — first vector}{\color{#D45D00}{\vec{v}}}\times\htmlData{t=w — second vector}{\color{#0B7D7D}{\vec{w}}})=0\]

Three quick yes/no tests that come up on almost every exam: (1) Perpendicular? Compute the DOT product. If v·w = 0, they're perpendicular. This works because cos90° = 0. (2) Parallel? Compute the CROSS product. If v×w = 0⃗ (zero vector), they're parallel. This works because sin0° = 0. Equivalently, one vector is a scalar multiple of the other. (3) Coplanar? Compute the TRIPLE product. If u·(v×w) = 0, all three vectors lie in the same plane. This works because the parallelepiped volume is zero when the "box" is flat. COMMON MISTAKE: Students mix up which test uses which product. Memory trick: "⊥ uses DOT, ∥ uses CROSS" — it's counterintuitive but correct.

✓ Use when asked "are these vectors perpendicular / parallel / coplanar?" Pick the right test based on what's being asked.
✗ Not for confusing the tests! ⊥ → dot = 0, ∥ → cross = 0, coplanar → triple = 0. Write these on your formula sheet.
Three yes/no tests: dot=0 (⊥), cross=0 (∥), triple=0 (coplanar).

Chapter 2 — Lines, Planes & Parametric Curves

Line (Parametric)
\[\vec{r}(t)=\htmlData{t=P — a known point on the line}{\color{#D45D00}{P}}+\htmlData{t=t — parameter (real number)}{\color{#5D7A20}{t}}\,\htmlData{t=v⃗ — direction vector parallel to the line}{\color{#0B7D7D}{\vec{v}}}\]

A parametric line is defined by giving a starting point P and a direction vector v. As the parameter t varies over all real numbers, the point r(t) = P + tv traces out the entire line. At t=0, you're at P. At t=1, you're at P+v (one "step" in the direction of v). At t=−1, you're at P−v (one step in the opposite direction). In component form: x = p₁+tv₁, y = p₂+tv₂, z = p₃+tv₃. To get a line through two points P and Q: direction = Q−P, then r(t) = P + t(Q−P). This also means r(t) = (1−t)P + tQ, which traces from P (at t=0) to Q (at t=1).

Where it comes from: "Start at P, go in direction v" is the geometric definition of a line. The parameter t scales how far you go.

✓ Use when given a point and direction, or two points. Restrict t ∈ [0,1] for a line segment, t ∈ [0,∞) for a ray. Two lines → set r₁(t) = r₂(s) to find intersections.
✗ Not for planes (planes use a normal vector, not a direction vector — completely different concept).
Point + direction vector → parametric line. t ∈ ℝ for full line, [0,1] for segment.
Plane Equation
\[\htmlData{t=a — x-component of normal}{\color{#D45D00}{a}}x+\htmlData{t=b — y-component of normal}{\color{#D45D00}{b}}y+\htmlData{t=c — z-component of normal}{\color{#D45D00}{c}}z=\htmlData{t=d — constant found by substituting a known point}{\color{#5D7A20}{d}}\]

A plane is a flat, infinite surface in 3D. It's defined by a point on the plane AND a normal vector (an arrow perpendicular to the surface). The equation ax+by+cz=d says: the normal vector is n = (a,b,c), and d is found by plugging in any known point. To find a plane through 3 points P, Q, R: (1) form two in-plane vectors PQ = Q−P and PR = R−P, (2) cross them: n = PQ × PR (this gives the normal — perpendicular to both), (3) write ax+by+cz = d using n's components, (4) substitute any of the three points to find d.

Where it comes from: The condition "point X is on the plane" means the vector from P to X is perpendicular to the normal: n·(X−P) = 0. Expanding gives ax+by+cz = n·P = d.

✓ Use when given 3 non-collinear points, or a point + normal. For two-plane intersection: cross normals → direction of intersection line, then find a point.
✗ Not for lines — lines use direction vectors (parallel to the line), not normal vectors. Don't confuse line direction with plane normal.
Normal vector determines the plane. Cross product of two in-plane vectors gives it.
Parametric Derivatives
\[\frac{dy}{dx}=\frac{\htmlData{t=dy/dt — derivative of y(t) w.r.t. parameter}{\color{#0B7D7D}{dy/dt}}}{\htmlData{t=dx/dt — derivative of x(t) w.r.t. parameter}{\color{#D45D00}{dx/dt}}},\quad \frac{d^2y}{dx^2}=\frac{\frac{d}{dt}\!\left(\frac{dy}{dx}\right)}{\htmlData{t=dx/dt — must be nonzero}{\color{#D45D00}{dx/dt}}}\]

Parametric derivatives let you find slopes and concavity of curves written as x(t), y(t) instead of y = f(x). The slope dy/dx is NOT just dy/dt divided by dx/dt in the usual sense — it IS their ratio, thanks to the chain rule: dy/dx = (dy/dt)/(dx/dt). Three key cases: (1) Horizontal tangent when dy/dt = 0 AND dx/dt ≠ 0 → slope = 0/nonzero = 0. (2) Vertical tangent when dx/dt = 0 AND dy/dt ≠ 0 → slope = nonzero/0 = ∞. (3) Both zero → indeterminate (cusp, corner, or smooth point — need further analysis). For the second derivative: d²y/dx² = [d/dt(dy/dx)] / (dx/dt). Do NOT compute this as (d²y/dt²)/(d²x/dt²) — that's the #1 mistake on parametric curve exams.

Where it comes from: Chain rule: dy/dx = (dy/dt)·(dt/dx) = (dy/dt)/(dx/dt). The second derivative requires differentiating the first derivative with respect to t, then dividing by dx/dt again.

✓ Use when finding tangent lines, slopes, horizontal/vertical tangents, or concavity for any parametric curve.
✗ Not for curves given as y = f(x) — just use regular dy/dx directly. The parametric formula is needed only when x and y are both functions of a parameter t.
Slope = (dy/dt)/(dx/dt). Second derivative: differentiate slope w.r.t. t, then divide by dx/dt.
Arc Length (Parametric)
\[L=\int_{\htmlData{t=α — start of parameter range}{\color{#A82870}{\alpha}}}^{\htmlData{t=β — end of parameter range}{\color{#A82870}{\beta}}}\sqrt{\left[\htmlData{t=x'(t)}{\color{#D45D00}{x'(t)}}\right]^2+\left[\htmlData{t=y'(t)}{\color{#0B7D7D}{y'(t)}}\right]^2}\,dt\]

Arc length measures the actual distance along a curve — like laying a string along it and measuring the string. The idea: chop the curve into tiny segments, each of length ds = √(dx²+dy²) (the hypotenuse of a tiny right triangle with legs dx and dy). Since dx = x'(t)dt and dy = y'(t)dt, each piece has length √((x')²+(y')²)·dt. Sum them all up (integrate) from t=α to t=β. The integrand √((x')²+(y')²) is the speed — how fast the point moves along the curve. Special case: for y = f(x), set x = t, so x'=1 and get L = ∫√(1+(dy/dx)²) dx. Key results: cycloid arch = 8r, circle of radius R = 2πR.

Where it comes from: Pythagorean theorem on each infinitesimal piece: ds² = dx² + dy². Factor out dt²: ds = √((x')²+(y')²) dt. Integrate.

✓ Use when "find the length of the curve", "arc length from t=a to t=b". Make sure the curve is traced exactly ONCE (no backtracking or overlapping).
✗ Not for polar curves — use the polar arc length formula √((r')²+r²) dθ instead. Using this formula on a polar curve would require converting to x(θ), y(θ) first — the polar formula is a shortcut.
Length = ∫ speed dt. Cycloid arch = 8r.
Polar Area
\[A=\int_{\htmlData{t=α — starting angle}{\color{#A82870}{\alpha}}}^{\htmlData{t=β — ending angle}{\color{#A82870}{\beta}}}\frac{1}{2}\htmlData{t=r — polar radius function}{\color{#D45D00}{r}}^2\,d\theta\]

In polar coordinates, area is computed using sectors (pie slices), not rectangles. A thin sector at radius r spanning a tiny angle dθ looks like a sliver of pie with area dA = ½r²dθ (think of the area of a circular sector: ½r²θ). To find the total area enclosed by a polar curve r = f(θ) from angle α to β, sum up all these slices: A = ½∫r² dθ. Area between two polar curves: ½∫(r_outer² − r_inner²) dθ. Critical step: always find where the curves intersect FIRST — these give your integration bounds. Some intersections only show up when r=0 at different θ values (they both pass through the origin but at different angles).

Where it comes from: A circular sector of radius r and angle Δθ has area ½r²Δθ. As Δθ→0, sum these infinitesimal sectors → integral.

✓ Use when "find the area enclosed by r = f(θ)", "area between two polar curves", or "area of one petal of a rose curve." Always sketch the curve first and find bounds carefully.
✗ Not for Cartesian regions — use ∫y dx or ∫∫dA there. Don't forget the ½ in front — students lose easy marks by dropping it.
½∫r² dθ. Between curves: ½∫(r_out² − r_in²) dθ.
Polar Arc Length
\[L=\int_{\htmlData{t=α — starting angle}{\color{#A82870}{\alpha}}}^{\htmlData{t=β — ending angle}{\color{#A82870}{\beta}}}\sqrt{\htmlData{t=r' — derivative dr/dθ}{\color{#0B7D7D}{(r'_\theta)}}^2+\htmlData{t=r — polar radius}{\color{#D45D00}{r}}^2}\;d\theta\]

Polar arc length measures the distance along a polar curve r = f(θ). Since x = rcosθ and y = rsinθ, you could convert to Cartesian and use the parametric arc length formula, but there's a shortcut: when you compute (dx/dθ)² + (dy/dθ)² and simplify (using product rule and trig identities), everything collapses to (r')² + r². So the arc length integral is just ∫√((r')²+r²) dθ, which is much simpler. Key result: the cardioid r = 1+sinθ has total perimeter 8 (one full loop from 0 to 2π).

Where it comes from: Start with parametric arc length: (dx/dθ)² + (dy/dθ)². Use product rule on x = rcosθ and y = rsinθ, then expand and simplify using sin²+cos² = 1. The cross terms cancel, leaving (r')² + r².

✓ Use when "find the length of the polar curve r = f(θ) from θ=α to θ=β."
✗ Not for parametric curves x(t), y(t) — use √((x')²+(y')²) dt there. The polar formula only works when the curve is given as r = f(θ).
√(r'²+r²) dθ. Simpler than converting to Cartesian.
Line Segment
\[\vec{r}(t)=(1-\htmlData{t=t — parameter from 0 to 1}{\color{#5D7A20}{t}})\,\htmlData{t=P — start point}{\color{#D45D00}{\vec{P}}}+\htmlData{t=t — parameter}{\color{#5D7A20}{t}}\,\htmlData{t=Q — end point}{\color{#0B7D7D}{\vec{Q}}},\quad 0\le t\le 1\]

A line segment from P to Q is a weighted average of the two endpoints. The parameter t acts like a slider: at t=0 you're 100% at P, at t=1 you're 100% at Q, and at t=½ you're exactly at the midpoint. The formula r(t) = (1−t)P + tQ means "take (1−t) parts of P and t parts of Q." As t increases from 0 to 1, you smoothly slide from P to Q. If you allow t outside [0,1], you get the full infinite line through P and Q. This is also called linear interpolation (or "lerp" in computer graphics).

Where it comes from: Starting from the line formula r(t) = P + t(Q−P), distribute: = P + tQ − tP = (1−t)P + tQ.

✓ Use when you need a finite segment between two endpoints, need to find the midpoint (set t=½), or want a point that's a specific fraction of the way between P and Q.
✗ Not for full lines extending to infinity — for those, use r(t) = P + tv with t ∈ ℝ.
Weighted average of two endpoints. t=½ gives midpoint.
Angle Between Planes
\[\cos\htmlData{t=θ — angle between the planes}{\color{#A82870}{\theta}}=\frac{\htmlData{t=n⃗₁ — normal to first plane}{\color{#D45D00}{\vec{n}_1}}\cdot\htmlData{t=n⃗₂ — normal to second plane}{\color{#0B7D7D}{\vec{n}_2}}}{|\vec{n}_1|\,|\vec{n}_2|}\]

The angle between two planes is the same as the angle between their normal vectors. Think of two sheets of paper intersecting — the angle of the "dihedral" (the fold between them) is determined by how their perpendicular directions relate. The formula is identical to the angle-between-vectors formula from Ch 1, but applied to the normals n₁ and n₂ instead of arbitrary vectors. Steps: (1) identify the normals (coefficients of x, y, z in each plane equation), (2) compute n₁·n₂, (3) compute |n₁| and |n₂|, (4) cosθ = (n₁·n₂)/(|n₁||n₂|), (5) θ = arccos(result). If the normals are parallel, the planes are parallel (θ = 0). If the normals are perpendicular, the planes are perpendicular.

Where it comes from: The angle between planes is defined as the angle between their normals. Same formula as §1.7 angle between vectors.

✓ Use when "find the angle between planes ax+by+cz=d and ex+fy+gz=h" — just apply the angle formula to their normals.
✗ Not for finding the LINE of intersection (for that: cross the normals n₁×n₂ → direction vector of the intersection line, then find a point on it).
Angle between planes is the angle between their normal vectors.
Rotation Matrices
\[\text{Active: }A=\begin{pmatrix}\cos\htmlData{t=θ — CCW rotation angle}{\color{#A82870}{\theta}}&-\sin\theta\\\sin\theta&\cos\theta\end{pmatrix}\quad\text{Passive: }A^{-1}=\begin{pmatrix}\cos\theta&\sin\theta\\-\sin\theta&\cos\theta\end{pmatrix}\]

Rotation matrices rotate either points or axes by angle θ. There are two kinds: Active rotation physically moves the point while keeping axes fixed — use matrix A. Passive rotation rotates the axes while keeping the point fixed — use A⁻¹ (the transpose, which flips the sign on the sinθ terms). To apply: multiply the matrix by the coordinate vector. If you also have a translation (shifting the origin to (h,k)), apply the translation FIRST: replace (x,y) with (x−h, y−k), THEN multiply by the rotation matrix. This is used to eliminate the Bxy cross-term in conic equations — rotate the axes so the conic aligns with the new axes.

Where it comes from: Active rotation maps (1,0) → (cosθ, sinθ) and (0,1) → (−sinθ, cosθ). These become the columns of the matrix. Passive rotation is the inverse (transpose for orthogonal matrices).

✓ Use when rotating coordinates to simplify equations, eliminating cross-terms in conics (Bxy terms), or physics problems involving rotated reference frames.
✗ Not for mixing up active vs passive — they differ by the sign on sinθ terms. Applying the wrong one rotates in the opposite direction.
Active: rotate point. Passive: rotate axes (flip sin signs). Translation first, then rotate.
Parametric Area
\[A=\int_{\htmlData{t=α — start parameter}{\color{#A82870}{\alpha}}}^{\htmlData{t=β — end parameter}{\color{#A82870}{\beta}}}\htmlData{t=y(t) — height function}{\color{#0B7D7D}{y(t)}}\,\htmlData{t=x'(t) — derivative of x-component}{\color{#D45D00}{x'(t)}}\,dt\]

Parametric area finds the area under (or enclosed by) a parametric curve. It comes from the standard area formula A = ∫y dx, but since x depends on t, we substitute dx = x'(t)dt. The result: A = ∫y(t)·x'(t) dt. For a closed curve (one that loops back to its starting point), the enclosed area is |∫g(t)·f'(t) dt| — take the absolute value since orientation might make it negative. Key results to memorize: cycloid arch area = 3πr² (three times the rolling circle's area — a beautiful result), ellipse area = πab (reduces to πr² when a=b).

Where it comes from: Substitution in ∫y dx: replace dx with x'(t)dt, replace y with y(t), change bounds from x-values to t-values.

✓ Use when "find the area under/enclosed by the parametric curve x=f(t), y=g(t)." Watch the bounds carefully — they're in terms of t, not x.
✗ Not for polar curves — those use ½∫r² dθ, a completely different formula. Don't mix them up.
Replace dx → x'(t)dt. Cycloid arch gives 3πr². Ellipse gives πab.
Polar ↔ Cartesian
\[\htmlData{t=x — Cartesian horizontal}{\color{#D45D00}{x}}=\htmlData{t=r — distance from origin}{\color{#5D7A20}{r}}\cos\htmlData{t=θ — angle from positive x-axis}{\color{#A82870}{\theta}},\quad \htmlData{t=y — Cartesian vertical}{\color{#0B7D7D}{y}}=r\sin\theta;\quad r=\sqrt{x^2+y^2}\]

Polar coordinates describe a point by its distance from the origin (r) and the angle from the positive x-axis (θ), instead of the usual (x, y). To convert: Polar → Cartesian: x = rcosθ, y = rsinθ (just trigonometry on the right triangle). Cartesian → Polar: r = √(x²+y²) (distance formula), θ = arctan(y/x) — BUT you must check which quadrant the point is in, because arctan only returns values in (−π/2, π/2). A point in Q3 like (−1,−1) has tanθ = 1, which gives θ = π/4 — but the actual angle is π/4 + π = 5π/4. Convention: negative r means go in the opposite direction: (−r, θ) is the same point as (r, θ+π).

Where it comes from: Drop a perpendicular from point P to the x-axis. The horizontal leg is x = rcosθ, the vertical leg is y = rsinθ. Pythagoras gives r = √(x²+y²).

✓ Use when converting between coordinate systems, simplifying equations with circular symmetry (e.g., x²+y² = r²), or evaluating polar integrals that need Cartesian bounds.
✗ Not for 3D — use cylindrical (r,θ,z) or spherical (ρ,θ,φ) coordinates in 3D.
Polar→Cart: x=rcosθ, y=rsinθ. Cart→Polar: r=√(x²+y²), check quadrant for θ.
Polar Slope
\[\frac{dy}{dx}=\frac{\htmlData{t=r' — derivative dr/dθ}{\color{#5D7A20}{r'}}\sin\htmlData{t=θ — polar angle}{\color{#A82870}{\theta}}+\htmlData{t=r — polar radius}{\color{#D45D00}{r}}\cos\theta}{\htmlData{t=r' — derivative dr/dθ}{\color{#5D7A20}{r'}}\cos\theta-\htmlData{t=r — polar radius}{\color{#D45D00}{r}}\sin\theta}\]

The slope of a polar curve r = f(θ) at a point isn't simply dr/dθ — that would be the rate of change of the radius, not the slope of the tangent line. Since x = rcosθ and y = rsinθ, we use the parametric derivative formula with θ as the parameter: dy/dx = (dy/dθ)/(dx/dθ). Applying the product rule: dy/dθ = r'sinθ + rcosθ and dx/dθ = r'cosθ − rsinθ. Their ratio gives the slope formula. After finding the slope, convert the point from polar (r,θ) to Cartesian (x,y) before writing the tangent line equation — tangent lines are always written in Cartesian form: y−y₀ = m(x−x₀).

Where it comes from: Product rule on x = rcosθ: dx/dθ = r'cosθ + r(−sinθ) = r'cosθ − rsinθ. Similarly for y = rsinθ: dy/dθ = r'sinθ + rcosθ. Then dy/dx = (dy/dθ)/(dx/dθ).

✓ Use when "find the tangent line to the polar curve r = f(θ) at θ = θ₀." Always remember to convert to Cartesian for the final tangent line equation.
✗ Not for parametric curves x(t), y(t) — use (dy/dt)/(dx/dt) directly for those.
Slope of polar curve. Don't forget to convert (r,θ) → (x,y) for the line equation.
Conic Discriminant
\[\htmlData{t=B² − 4AC — the conic discriminant}{\color{#D45D00}{B^2-4AC}}:\quad {\lt}0\;\text{ellipse},\quad {\gt}0\;\text{hyperbola},\quad {=}0\;\text{parabola}\]

The conic discriminant tells you what type of curve the general second-degree equation Ax²+Bxy+Cy²+Dx+Ey+F = 0 represents. Compute B²−4AC and check the sign: Negative (B²−4AC < 0): ellipse (or circle if A=C and B=0). Zero (B²−4AC = 0): parabola. Positive (B²−4AC > 0): hyperbola. Degenerate cases are possible: two lines, a single point, or the empty set (e.g., x²+y² = −1 has no solution). When B ≠ 0 (there's a cross-term Bxy), the conic is rotated relative to the axes. To classify it, you can use the discriminant without rotating first. To find the standard form, rotate axes by θ where cot2θ = (A−C)/B to eliminate the cross-term.

Where it comes from: The discriminant B²−4AC is invariant under rotation — it doesn't change when you rotate the axes. This is a deep result from linear algebra.

✓ Use when asked "what type of conic is Ax²+Bxy+Cy²+Dx+Ey+F = 0?" Just compute B²−4AC and check the sign.
✗ Not for finding the actual equation in standard form — for that, complete the square and/or rotate axes. The discriminant only tells you the TYPE.
B²−4AC classifies: negative→ellipse, positive→hyperbola, zero→parabola.

Ch 1 — Vectors ✓

Builds the vector toolkit in ℝ² and ℝ³ — definitions, operations, and geometric applications that underpin all of multivariable calculus.

Vector Basics
What:
Vectors defined by magnitude and direction, represented as ordered components in ℝ² or ℝ³.
Why:
The fundamental objects everything else builds on — displacements, forces, velocities.
How:
\(\vec{v}=\langle v_1,v_2,v_3\rangle\), \(|\vec{v}|=\sqrt{v_1^2+v_2^2+v_3^2}\)
Trap:
Equality means same magnitude AND direction — position in space doesn't matter.
Vector Arithmetic
What:
Addition, subtraction, and scalar multiplication — all component-wise.
Why:
Combines forces, displacements, and velocities algebraically; parallelogram law gives geometric intuition.
How:
\(\vec{v}+\vec{w}=\langle v_1+w_1,\,v_2+w_2,\,v_3+w_3\rangle\), \(c\vec{v}=\langle cv_1,cv_2,cv_3\rangle\)
Unit Vectors & Direction
What:
Normalization to length 1; standard basis î, ĵ, k̂; direction cosines and inclination angles.
Why:
Separates direction from magnitude — essential for projections and coordinate decomposition.
How:
\(\hat{v}=\vec{v}/|\vec{v}|\), direction cosines satisfy \(\cos^2\alpha+\cos^2\beta+\cos^2\gamma=1\)
Trap:
The zero vector has no unit vector — can't normalize it.
Dot Product
What:
Scalar product measuring how aligned two vectors are.
Why:
Unlocks angles between vectors, orthogonality tests, and projections.
How:
\(\vec{v}\cdot\vec{w}=v_1w_1+v_2w_2+v_3w_3=|\vec{v}||\vec{w}|\cos\theta\), angle: \(\theta=\arccos\!\bigl(\frac{\vec{v}\cdot\vec{w}}{|\vec{v}||\vec{w}|}\bigr)\)
Trap:
Result is a SCALAR, not a vector. Dot = 0 means perpendicular, not that a vector is zero.
Projections & Work
What:
Scalar and vector projection of one vector onto another; work as force dotted with displacement.
Why:
Decomposes vectors into parallel/perpendicular parts — central to physics and later integration.
How:
Scalar: \(\text{comp}_{\vec{v}}\vec{u}=\frac{\vec{u}\cdot\vec{v}}{|\vec{v}|}\), Vector: \(\text{proj}_{\vec{v}}\vec{u}=\frac{\vec{u}\cdot\vec{v}}{|\vec{v}|^2}\vec{v}\), Work: \(W=\vec{F}\cdot\vec{d}\)
Cross Product
What:
A vector perpendicular to both inputs whose magnitude equals the parallelogram area.
Why:
Produces normal vectors to planes, computes torque, and measures area.
How:
\(\vec{v}\times\vec{w}\) via 3×3 determinant with î, ĵ, k̂ in the first row. Magnitude: \(|\vec{v}\times\vec{w}|=|\vec{v}||\vec{w}|\sin\theta\)
Trap:
NOT commutative — \(\vec{v}\times\vec{w}=-(\vec{w}\times\vec{v})\). Use the right-hand rule for direction.
Geometric Applications
What:
Parallelogram area, triangle area, and parallelepiped volume from cross and triple products.
Why:
Turns abstract vector algebra into measurable geometry — areas, volumes, coplanarity.
How:
Area△ = \(\tfrac{1}{2}|\vec{u}\times\vec{v}|\), Volume = \(|\vec{u}\cdot(\vec{v}\times\vec{w})|\)
Trap:
Triple product = 0 means the vectors are coplanar, not that there is "no volume."
Connections → Dot product → scalar projections → work integrals (Ch 6) · Cross product → normal vectors → planes (Ch 2) → surface integrals (Ch 7) · Magnitude & distance → arc length & curvature (Ch 3)

Ch 2 — Lines, Planes & Parametric Curves ✓

Extends vectors to describe lines, planes, and curves in space. Introduces parametric and polar representations with their calculus.

Lines & Planes
What:
Lines from point + direction vector. Planes from point + normal vector. Relations: parallel, intersecting, skew.
Why:
Describes geometry in 3D — positions, orientations, intersections of flat surfaces.
How:
Line: \(\vec{r}(t)=P+t\vec{v}\). Plane: \(ax+by+cz=d\) with \(\vec{n}=(a,b,c)\). Three points → normal = cross product.
Trap:
Skew lines: not parallel AND not intersecting — must check BOTH z-values match after solving x,y.
Parametric Curves & Calculus
What:
Curves as (x(t), y(t)). First/second derivatives via chain rule. Conics as special cases.
Why:
Many curves can't be expressed as y=f(x). Parametric form handles self-intersections, loops, and 3D paths.
How:
\(dy/dx=(dy/dt)/(dx/dt)\). Horiz tangent: dy/dt=0. Vert tangent: dx/dt=0. Area: \(\int y\,x'\,dt\). Arc length: \(\int\sqrt{(x')^2+(y')^2}\,dt\).
Trap:
Both derivatives zero at same point → tangent undefined (cusp or corner).
Polar Coordinates
What:
Points as (r, θ). Conversions: x=r cosθ, y=r sinθ. Special curves: cardioids, limaçons, rose curves.
Why:
Curves with rotational symmetry (circles, spirals) have much simpler polar equations.
How:
Slope: \(\frac{r'\sin\theta+r\cos\theta}{r'\cos\theta-r\sin\theta}\). Area: \(\frac{1}{2}\int r^2\,d\theta\). Arc length: \(\int\sqrt{(r')^2+r^2}\,d\theta\).
Trap:
For area between curves, find ALL intersections (some may only appear as simultaneous r=0 at different θ).
Connections → Lines/planes → tangent planes & normal lines (Ch 3) · Parametric curves → line integrals (Ch 4) · Polar coordinates → double integrals in polar (Ch 5) · Arc length ds → line integral ds (Ch 4)
