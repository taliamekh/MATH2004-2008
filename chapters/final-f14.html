A1 — Slope of Tangent to Polar Curve
🧠 What is this asking?

Find the slope \(dy/dx\) of the tangent line to \(r = 4\cos\theta\) at \(\theta = \pi/3\).

📋 Formulas Needed
  • Polar slope: \(\frac{dy}{dx} = \frac{r'\sin\theta + r\cos\theta}{r'\cos\theta - r\sin\theta}\)
📝 Full Solution

\(r = 4\cos\theta\), \(r' = -4\sin\theta\).

At \(\theta = \pi/3\): \(r = 4\cos(\pi/3) = 4 \cdot \frac{1}{2} = 2\), \(r' = -4\sin(\pi/3) = -4\cdot\frac{\sqrt{3}}{2} = -2\sqrt{3}\).

\[\frac{dy}{dx} = \frac{(-2\sqrt{3})(\frac{\sqrt{3}}{2}) + (2)(\frac{1}{2})}{(-2\sqrt{3})(\frac{1}{2}) - (2)(\frac{\sqrt{3}}{2})} = \frac{-3 + 1}{-\sqrt{3} - \sqrt{3}} = \frac{-2}{-2\sqrt{3}} = \frac{1}{\sqrt{3}} = \frac{\sqrt{3}}{3}\]

✅ Answer: (b) \(\dfrac{\sqrt{3}}{3}\)
A2 — Angle Between Planes
📝 Full Solution

Planes: \(3x-y+2z=2\) and \(2x+3y+z=4\). Normals: \(\mathbf{n}_1=(3,-1,2)\), \(\mathbf{n}_2=(2,3,1)\).

\[\cos\alpha = \frac{|\mathbf{n}_1\cdot\mathbf{n}_2|}{|\mathbf{n}_1||\mathbf{n}_2|} = \frac{|6-3+2|}{\sqrt{9+1+4}\sqrt{4+9+1}} = \frac{5}{\sqrt{14}\sqrt{14}} = \frac{5}{14}\]

✅ Answer: (c) \(\dfrac{5}{14}\)
A3 — Volume of Parallelepiped
📝 Full Solution

Vectors: \(\mathbf{u}=(2,0,0)\), \(\mathbf{v}=(0,\sqrt{2},0)\), \(\mathbf{w}=(0,0,\sqrt{2})\).

\[V = |\mathbf{u}\cdot(\mathbf{v}\times\mathbf{w})| = \left|\det\begin{pmatrix}2&0&0\\0&\sqrt{2}&0\\0&0&\sqrt{2}\end{pmatrix}\right| = |2\cdot\sqrt{2}\cdot\sqrt{2}| = |2\cdot 2| = 4\]

✅ Answer: (a) 4

Check (0,0,0): \(-2(0)+3(0)+3(0)=0\) ✓

A5 — Chain Rule: \(\partial w/\partial t\) at t=0
📝 Full Solution

\(w = x^2+y+3xy^4\), \(x=\sin 2t\), \(y=\cos t\). At \(t=0\): \(x=0\), \(y=1\).

\(w_x = 2x+3y^4\), \(w_y = 1+12xy^3\). At (0,1): \(w_x=3\), \(w_y=1\).

\(x'(t) = 2\cos 2t\), \(y'(t) = -\sin t\). At \(t=0\): \(x'=2\), \(y'=0\).

\[\frac{dw}{dt} = w_x\cdot x' + w_y\cdot y' = 3(2) + 1(0) = 6\]

✅ Answer: (d) 6
A6 — Gradient / Steepest Ascent Direction
📝 Full Solution

\(f = \frac{x}{y}+\frac{y}{z}\). At \((1,-1,2)\):

\(f_x = \frac{1}{y} = -1\), \(f_y = -\frac{x}{y^2}+\frac{1}{z} = -1+\frac{1}{2} = -\frac{1}{2}\), \(f_z = -\frac{y}{z^2} = \frac{1}{4}\).

\(\nabla f = (-1, -\frac{1}{2}, \frac{1}{4})\).

✅ Answer: (b) \(-\mathbf{i} - \frac{1}{2}\mathbf{j} + \frac{1}{4}\mathbf{k}\)
A7 — Relative Extrema (Second Derivative Test)
📝 Full Solution

\(f = -x^2 - 3y^2 + 4x - 6y + 8\). \(f_x = -2x+4=0 \Rightarrow x=2\). \(f_y = -6y-6=0 \Rightarrow y=-1\). CP: (2,−1).

\(f_{xx}=-2\), \(f_{yy}=-6\), \(f_{xy}=0\). \(D = (-2)(-6)-0 = 12 > 0\), \(f_{xx}<0\) → maximum.

A8 — Double Integral with Reversed Order
📝 Full Solution

\(\int_0^2\int_0^{x/2} e^{-x^2}\,dy\,dx\). Inner: \(\int_0^{x/2}dy = x/2\). So \(\int_0^2 \frac{x}{2}e^{-x^2}\,dx\).

Let \(u = -x^2\), \(du = -2x\,dx\):

\[= \frac{1}{2}\int_0^2 xe^{-x^2}\,dx = \frac{1}{2}\cdot\left[-\frac{1}{2}e^{-x^2}\right]_0^2 = \frac{1}{2}\cdot\left(-\frac{1}{2}e^{-4}+\frac{1}{2}\right) = \frac{1-e^{-4}}{4} = \frac{e^4-1}{4e^4}\]

✅ Answer: (a) \(\dfrac{e^4-1}{4e^4}\)
A9 — Triple Integral over Non-Standard Region
📝 Full Solution

\(\iiint_T xyz\,dV\) where \(T: 0\le x\le 1, 0\le y\le 1, \sqrt{x^2+y^2}\le z\le 2\).

\[I = \int_0^1\int_0^1 xy\left[\frac{z^2}{2}\right]_{\sqrt{x^2+y^2}}^2 dy\,dx = \int_0^1\int_0^1 xy\cdot\frac{4-(x^2+y^2)}{2}\,dy\,dx\]

\[= \frac{1}{2}\int_0^1\int_0^1(4xy - x^3y - xy^3)\,dy\,dx\]

\[= \frac{1}{2}\int_0^1\left[2xy^2-\frac{x^3y^2}{2}-\frac{xy^4}{4}\right]_0^1 dx = \frac{1}{2}\int_0^1\left(2x-\frac{x^3}{2}-\frac{x}{4}\right)dx\]

\[= \frac{1}{2}\int_0^1\left(\frac{7x}{4}-\frac{x^3}{2}\right)dx = \frac{1}{2}\left[\frac{7x^2}{8}-\frac{x^4}{8}\right]_0^1 = \frac{1}{2}\cdot\frac{6}{8} = \frac{3}{8}\]

✅ Answer: (e) \(\dfrac{3}{8}\)
A10 — Change of Variables for Elliptic Region
📝 Full Solution

\(x=5u, y=2v\). Jacobian: \(J = \frac{\partial(x,y)}{\partial(u,v)} = 5\cdot 2 = 10\).

\(xy = (5u)(2v) = 10uv\). The integral becomes \(\iint_S 10uv\cdot 10\,dA = \iint_S 100uv\,dA\), where S is the unit disk \(u^2+v^2\le 1\).

✅ Answer: (d) \(\iint_S 100uv\,dA\)
A11 — Line Integral (Conservative Field Shortcut)
📝 Full Solution

\(\int_C (6xy^2-y^3)\,dx + (6x^2y-3xy^2)\,dy\) from (1,2) to (3,4).

Check if conservative: \(P = 6xy^2-y^3\), \(Q = 6x^2y-3xy^2\).

\(P_y = 12xy-3y^2\), \(Q_x = 12xy-3y^2\) ✓ → conservative!

Find potential: \(f_x = 6xy^2-y^3 \Rightarrow f = 3x^2y^2 - xy^3 + g(y)\).

\(f_y = 6x^2y-3xy^2+g'(y) = 6x^2y-3xy^2 \Rightarrow g'(y) = 0\). So \(f = 3x^2y^2-xy^3\).

By FTCLI: \(f(3,4)-f(1,2) = [3(9)(16)-3(64)] - [3(1)(4)-1(8)] = [432-192]-[12-8] = 240-4 = 236\).

✅ Answer: (b) 236
A12 — Surface Integral via Divergence Theorem (Sphere)
📝 Full Solution

\(\mathbf{F} = (x+\sin z, 2y+\cos x, 3z+\tan z)\). On unit sphere \(x^2+y^2+z^2=1\).

\(\text{div}\,\mathbf{F} = 1+2+3 = 6\).

\[\iint_S \mathbf{F}\cdot\mathbf{n}\,dS = \iiint 6\,dV = 6\cdot\frac{4}{3}\pi(1)^3 = 8\pi\]

✅ Answer: (c) \(8\pi\)

Part B — Long Answer (F14)

B1 [8 marks] — Lagrange Multipliers: \(f=x^2y^2\) on \(x+y=1\)
📝 Full Solution

\(\nabla f = (2xy^2, 2x^2y)\), \(\nabla g = (1,1)\) where \(g = x+y-1\).

Lagrange: \(2xy^2 = \lambda\) and \(2x^2y = \lambda\). Setting equal: \(2xy^2 = 2x^2y\).

Case 1: \(x=0 \Rightarrow y=1\). \(f(0,1)=0\).

Case 2: \(y=0 \Rightarrow x=1\). \(f(1,0)=0\).

Case 3: \(xy\ne 0 \Rightarrow y=x\). Constraint: \(2x=1 \Rightarrow x=y=\frac{1}{2}\). \(f = \frac{1}{16}\).

✅ Answer: Local minima at (0,1) and (1,0) with value 0. Local maximum at \((\frac{1}{2},\frac{1}{2})\) with value \(\frac{1}{16}\).
B2 [7 marks] — Polar Double Integral: \(\iint xy\sqrt{x^2+y^2}\,dA\)
📝 Full Solution

Region: \(1 \le x^2+y^2 \le 4\), \(x\ge 0\), \(y\ge 0\) → polar: \(1\le r\le 2\), \(0\le\theta\le\pi/2\).

\(xy\sqrt{x^2+y^2} = (r\cos\theta)(r\sin\theta)(r) = r^3\sin\theta\cos\theta\). With \(dA = r\,dr\,d\theta\):

\[I = \int_0^{\pi/2}\int_1^2 r^4\sin\theta\cos\theta\,dr\,d\theta = \int_0^{\pi/2}\sin\theta\cos\theta\,d\theta\cdot\int_1^2 r^4\,dr\]

\[= \left[\frac{\sin^2\theta}{2}\right]_0^{\pi/2}\cdot\left[\frac{r^5}{5}\right]_1^2 = \frac{1}{2}\cdot\frac{31}{5} = \frac{31}{10}\]

✅ Answer: \(\dfrac{31}{10}\)
B3 [7 marks] — Line Integral on Helix
📝 Full Solution

\(\int_C z\,dx + x\,dy + y\,dz\) on \(x=\cos t, y=\sin t, z=t\) from \((1,0,0)\) to \((1,0,2\pi)\).

\(dx=-\sin t\,dt\), \(dy=\cos t\,dt\), \(dz=dt\). Parameter: \(t\in[0,2\pi]\).

\[I = \int_0^{2\pi}\left[t(-\sin t) + \cos t(\cos t) + \sin t(1)\right]dt = \int_0^{2\pi}(-t\sin t + \cos^2 t + \sin t)\,dt\]

Three separate integrals:

\(\int_0^{2\pi}\cos^2 t\,dt = \pi\) (half-angle formula).

\(\int_0^{2\pi}\sin t\,dt = 0\) (full period).

\(\int_0^{2\pi}-t\sin t\,dt\): use integration by parts with \(u=t\), \(dv=-\sin t\,dt\):

\(= [t\cos t]_0^{2\pi} - \int_0^{2\pi}\cos t\,dt = 2\pi - 0 = 2\pi\).

\[I = 2\pi + \pi + 0 = 3\pi\]

✅ Answer: \(I = 3\pi\)
B4 [7 marks] — No Vector Field with Given Curl (div(curl F) ≠ 0)
🧠 What is this asking?

Does there exist F with \(\nabla\times\mathbf{F} = (2xy, -2yz, 2xy)\)?

📝 Full Solution

Key identity: \(\text{div}(\text{curl}\,\mathbf{F}) = 0\) for ANY vector field F. Always.

Check: \(\text{div}(2xy, -2yz, 2xy) = 2y - 2z + 0 = 2y-2z \ne 0\).

Since the divergence of the proposed curl is NOT identically zero, no such F can exist.

✅ Answer: No such F exists. The identity \(\text{div}(\text{curl}\,\mathbf{F})=0\) is violated since \(\text{div}(2xy,-2yz,2xy) = 2y-2z \ne 0\).
⚡ Quick Guide — "Does a field with this curl exist?"
  1. Compute the divergence of the given curl expression
  2. If div ≠ 0 → no such field exists (by the identity div(curl F) = 0 always)
  3. If div = 0 → existence is possible (but you'd need to construct it)
B5 [7 marks] — Divergence Theorem on Hemisphere
📝 Full Solution

\(\mathbf{F} = (xz^2, x^2y-z^3, 2xy+y^2z)\). Surface S: hemisphere \(z=\sqrt{1-x^2-y^2}\) plus base \(z=0\).

\(\text{div}\,\mathbf{F} = z^2 + x^2 + y^2 = \rho^2\) (in spherical, this is \(\rho^2\)).

Solid T in spherical: \(0\le\rho\le 1\), \(0\le\theta\le 2\pi\), \(0\le\phi\le\pi/2\).

\[\iiint_T\rho^2\cdot\rho^2\sin\phi\,d\rho\,d\phi\,d\theta = \int_0^{2\pi}d\theta\int_0^{\pi/2}\sin\phi\,d\phi\int_0^1\rho^4\,d\rho\]

\[= 2\pi\cdot[-\cos\phi]_0^{\pi/2}\cdot\frac{1}{5} = 2\pi\cdot 1\cdot\frac{1}{5} = \frac{2\pi}{5}\]

✅ Answer: \(\dfrac{2\pi}{5}\)
B6 [8 marks] — Stokes' Theorem on Elliptic Surface
🧠 What is this asking?

Evaluate \(\iint_S \text{curl}\,\mathbf{F}\cdot\mathbf{n}\,dS\) where \(\mathbf{F}=(x^2,2x,z^2)\) and S is the elliptic region \(4x^2+y^2=4\) on \(z=0\).

📝 Full Solution

Method 1 (Direct): \(\text{curl}\,\mathbf{F} = (0,0,2)\). On plane \(z=0\), \(\mathbf{n}=\mathbf{k}\). So \(\text{curl}\,\mathbf{F}\cdot\mathbf{n}=2\).

\[\iint_S 2\,dS = 2\cdot\text{Area of ellipse} = 2\cdot\pi(1)(2) = 4\pi\]

(Ellipse: \(x^2+\frac{y^2}{4}=1\), so \(a=1\), \(b=2\).)

Method 2 (Stokes'): \(\oint_C\mathbf{F}\cdot d\mathbf{r}\) where C is the ellipse. Parametrize: \(x=\cos t, y=2\sin t, z=0\).

\[\int_0^{2\pi}(\cos^2 t,2\cos t,0)\cdot(-\sin t,2\cos t,0)\,dt = \int_0^{2\pi}(-\cos^2 t\sin t+4\cos^2 t)\,dt = 0+4\pi = 4\pi\]

✅ Answer: \(4\pi\)
