Q3. Angle between two planes
Ch2 — Planes · §2.4

Find the angle between \(x+2y-z=4\) and \(2x-y+3z=1\).

📝 Full solution

\(\mathbf{n}_1=(1,2,-1),\; \mathbf{n}_2=(2,-1,3)\).
\(\cos\theta = \frac{|2-2-3|}{\sqrt{6}\sqrt{14}} = \frac{3}{\sqrt{84}} = \frac{3}{2\sqrt{21}}\).
\(\theta = \arccos\frac{3}{2\sqrt{21}} \approx \arccos(0.327) \approx 70.9°\).

✅ \(\theta = \arccos\frac{3}{2\sqrt{21}}\)
Q4. Arc length of a space curve
Ch2 — Arc Length · §2.10

Length of \(\mathbf{r}(t) = \langle t, t^2, t^3\rangle\) from \(t=0\) to \(t=1\).

📝 Full solution

\(\mathbf{r}'(t) = (1, 2t, 3t^2)\). \(|\mathbf{r}'| = \sqrt{1+4t^2+9t^4}\).

\[L = \int_0^1\sqrt{1+4t^2+9t^4}\,dt\]

This integral has no elementary closed form. Numerical evaluation gives \(L \approx 1.548\).

✅ \(L = \int_0^1\sqrt{1+4t^2+9t^4}\,dt \approx 1.548\)

Q7. Area inside a cardioid
Ch2 — Polar Area · §2.13

Area inside \(r=1+\cos\theta\).

📝 Full solution

\[A = \frac{1}{2}\int_0^{2\pi}(1+\cos\theta)^2\,d\theta = \frac{1}{2}\int_0^{2\pi}(1+2\cos\theta+\cos^2\theta)\,d\theta\]

\(= \frac{1}{2}\left[2\pi+0+\pi\right] = \frac{3\pi}{2}\).

✅ \(\frac{3\pi}{2}\)
Q8. Arc length of a polar curve
Ch2 — Polar Arc Length · §2.14

Arc length of \(r=2\theta\) for \(0\le\theta\le\pi\).

📝 Full solution

\(r'=2\). \(L = \int_0^\pi\sqrt{(2\theta)^2+4}\,d\theta = \int_0^\pi\sqrt{4\theta^2+4}\,d\theta = 2\int_0^\pi\sqrt{\theta^2+1}\,d\theta\).

Using the formula \(\int\sqrt{u^2+1}\,du = \frac{1}{2}\left[u\sqrt{u^2+1}+\ln(u+\sqrt{u^2+1})\right]\):

\[L = 2\cdot\frac{1}{2}\left[\theta\sqrt{\theta^2+1}+\ln(\theta+\sqrt{\theta^2+1})\right]_0^\pi = \pi\sqrt{\pi^2+1}+\ln(\pi+\sqrt{\pi^2+1})\]

✅ \(L = \pi\sqrt{\pi^2+1}+\ln(\pi+\sqrt{\pi^2+1})\)

Q10. Directional derivative
Ch3 — Directional Derivatives · §3.5

\(f=x^2+y^2\) at \((1,1)\) in direction \(\langle 3,4\rangle\).

📝 Full solution

\(\nabla f = (2x,2y) = (2,2)\). Unit vector: \(\hat{\mathbf{u}} = \frac{(3,4)}{5} = (\frac{3}{5},\frac{4}{5})\).

\(D_{\hat{\mathbf{u}}}f = (2,2)\cdot(\frac{3}{5},\frac{4}{5}) = \frac{6}{5}+\frac{8}{5} = \frac{14}{5}\).

✅ \(\frac{14}{5}\)
Q11. Chain rule dz/dt
Ch3 — Chain Rule · §3.6

\(z=xy\) where \(x=\sin t, y=e^t\). Find \(dz/dt\).

📝 Full solution

\(\frac{dz}{dt} = z_x\cdot x' + z_y\cdot y' = y\cos t + x e^t = e^t\cos t + \sin t\cdot e^t = e^t(\cos t+\sin t)\).

✅ \(\frac{dz}{dt} = e^t(\cos t+\sin t)\)
Q12. Double integral
Ch5 — Double Integrals · §5.2

Evaluate \(\int_0^2\int_0^x(x^2+y^2)\,dy\,dx\).

📝 Full solution

\[\int_0^2\left[x^2y+\frac{y^3}{3}\right]_0^x dx = \int_0^2\left(x^3+\frac{x^3}{3}\right)dx = \int_0^2\frac{4x^3}{3}\,dx = \frac{4}{3}\cdot\frac{x^4}{4}\Big|_0^2 = \frac{4}{3}\cdot 4 = \frac{16}{3}\]

✅ \(\frac{16}{3}\)

\(\iiint_E z\,dV\) over \(0\le x\le 1, 0\le y\le 2, 0\le z\le 3\).

Q14. Polar integral to Cartesian
Ch5 — Polar · §5.4

Convert \(\int_0^{2\pi}\int_0^1 r^2\,dr\,d\theta\) to Cartesian and evaluate.

📝 Full solution

In polar: \(\int_0^{2\pi}\int_0^1 r^2\,dr\,d\theta = 2\pi\cdot\frac{1}{3} = \frac{2\pi}{3}\).

In Cartesian this is \(\iint_{x^2+y^2\le 1}\sqrt{x^2+y^2}\,dA\) (since \(r^2\,dr\,d\theta = r\cdot r\,dr\,d\theta = r\,dA\)).

✅ \(\frac{2\pi}{3}\)
Q15. Jacobian of a transformation
Ch5 — Change of Variables · §5.4

\(x=u+v, y=u-v\). Find the Jacobian.

📝 Full solution

\[J = \begin{vmatrix}\frac{\partial x}{\partial u}&\frac{\partial x}{\partial v}\\\frac{\partial y}{\partial u}&\frac{\partial y}{\partial v}\end{vmatrix} = \begin{vmatrix}1&1\\1&-1\end{vmatrix} = -1-1 = -2\]

\(|J| = 2\).

✅ \(J = -2\), so \(|J| = 2\)
Q16. Change of variables in double integral
Ch5 — Change of Variables · §5.4

Use \(u=x-y, v=y\) to evaluate \(\int_0^2\int_0^{2y}(x-y)^2\,dx\,dy\).

📝 Full solution

Direct evaluation is simpler here. Inner integral: \(\int_0^{2y}(x-y)^2\,dx = \left[\frac{(x-y)^3}{3}\right]_0^{2y} = \frac{y^3}{3}-\frac{(-y)^3}{3} = \frac{y^3+y^3}{3} = \frac{2y^3}{3}\).

\[\int_0^2\frac{2y^3}{3}\,dy = \frac{2}{3}\cdot\frac{y^4}{4}\Big|_0^2 = \frac{2}{3}\cdot 4 = \frac{8}{3}\]

✅ \(\frac{8}{3}\)
Q17. Line integral of F = ⟨y, −x⟩ around unit circle
Ch4/Ch6 — Green's Theorem · §4.2, §6.3

Compute \(\oint_C \mathbf{F}\cdot d\mathbf{r}\) for \(\mathbf{F}=(y,-x)\) along unit circle CCW.

📝 Full solution

By Green's: \(\oint P\,dx+Q\,dy = \iint_R(Q_x-P_y)\,dA\).
\(Q_x = -1,\; P_y = 1\). So \(Q_x-P_y = -2\).
\(\oint = -2\cdot\text{Area}(\text{unit disk}) = -2\pi\).

✅ \(-2\pi\)
Q18. Is F = ⟨2xy, x²⟩ conservative?
Ch3 — Conservative Test · §3.7

Check if \(\mathbf{F}=(2xy, x^2)\) is conservative.

📝 Full solution

\(P_y = 2x,\; Q_x = 2x\). Equal! ✓ So yes, F is conservative.

Potential: \(f_x=2xy \Rightarrow f=x^2y+g(y)\). \(f_y=x^2+g'(y)=x^2 \Rightarrow g'=0\). So \(f=x^2y\).

✅ Yes. Potential: \(f(x,y) = x^2y + C\)
Q19. Potential function for conservative field
Ch3/Ch4 — Potential · §3.7

Find potential for \(\mathbf{F}=(3x^2+2y,\; 2x+4y^3)\).

📝 Full solution

Check: \(P_y=2, Q_x=2\). ✓ Conservative.

\(f_x=3x^2+2y \Rightarrow f=x^3+2xy+g(y)\).
\(f_y=2x+g'(y)=2x+4y^3 \Rightarrow g'(y)=4y^3 \Rightarrow g(y)=y^4\).

✅ \(f(x,y) = x^3+2xy+y^4+C\)

Q23. Flux of F = ⟨0, 0, z⟩ through a paraboloid
Ch6 — Flux · §6.2

Flux through \(z=4-x^2-y^2\) over disk \(x^2+y^2\le 4\).

📝 Full solution

For \(z=f(x,y)=4-x^2-y^2\): \(f_x=-2x, f_y=-2y\). \(\mathbf{F}=(0,0,z)=(0,0,4-x^2-y^2)\).

Flux = \(\iint_R(-P f_x - Q f_y + R)\,dA = \iint_R(0+0+(4-x^2-y^2))\,dA\).

In polar: \(\int_0^{2\pi}\int_0^2(4-r^2)r\,dr\,d\theta = 2\pi\left[2r^2-\frac{r^4}{4}\right]_0^2 = 2\pi(8-4) = 8\pi\).

✅ \(8\pi\)
Q24. Green's Theorem on unit square
Ch6 — Green's Theorem · §6.3

\(\oint_C(x^2-y)\,dx+(xy+x)\,dy\) where C bounds \([0,1]\times[0,1]\).

📝 Full solution

\(Q_x-P_y = (y+1)-(-1) = y+2\).

\[\iint_R(y+2)\,dA = \int_0^1\int_0^1(y+2)\,dy\,dx = 1\cdot\left[\frac{y^2}{2}+2y\right]_0^1 = \frac{1}{2}+2 = \frac{5}{2}\]

Q25. Stokes' Theorem — F = ⟨y, −x, z⟩ on hemisphere
Ch6 — Stokes' Theorem · §6.4

Evaluate \(\oint_C\mathbf{F}\cdot d\mathbf{r}\) where C is the boundary of the upper hemisphere \(x^2+y^2+z^2=1\).

📝 Full solution

The boundary C is the unit circle in the z = 0 plane. By Stokes': \(\oint_C\mathbf{F}\cdot d\mathbf{r} = \iint_S(\nabla\times\mathbf{F})\cdot\mathbf{n}\,dS\).

\(\nabla\times\mathbf{F} = (0,0,-1-1) = (0,0,-2)\).

Use the flat disk in z = 0 (same boundary, simpler surface). \(\mathbf{n}=\mathbf{k}\).
\(\iint (0,0,-2)\cdot(0,0,1)\,dA = -2\cdot\pi(1)^2 = -2\pi\).

✅ \(-2\pi\)

Q27. Critical points of f = x³ − 3xy²
Ch7 — Critical Points · §7.2

📝 Full solution

\(f_x = 3x^2-3y^2=0 \Rightarrow x^2=y^2\).
\(f_y = -6xy=0 \Rightarrow xy=0\).

From \(xy=0\): \(x=0\) or \(y=0\). Combined with \(x^2=y^2\):
If \(x=0\): \(y=0\). If \(y=0\): \(x=0\). Only CP: \((0,0)\).

Classify: \(f_{xx}=6x=0,\; f_{yy}=-6x=0,\; f_{xy}=-6y=0\). \(D=0-0=0\). Inconclusive.

Examine nearby: \(f(ε,0)=ε^3>0\) for ε > 0, \(f(-ε,0)=-ε^3<0\). So (0,0) is a saddle point (neither max nor min).

✅ One critical point at \((0,0)\): saddle point (SDT inconclusive, verified by sign change).
Q28. Lagrange multipliers — f = xy on the unit circle
Ch7 — Lagrange Multipliers · §7.3

Max/min of \(f=xy\) subject to \(x^2+y^2=1\).

📝 Full solution

\(\nabla f=\lambda\nabla g\): \((y,x)=\lambda(2x,2y)\).

\(y=2\lambda x\) … (1) and \(x=2\lambda y\) … (2).

From (1): \(\lambda=\frac{y}{2x}\). Sub into (2): \(x=2\cdot\frac{y}{2x}\cdot y=\frac{y^2}{x} \Rightarrow x^2=y^2\).

So \(y=\pm x\). With \(x^2+y^2=1\): \(2x^2=1 \Rightarrow x=\pm\frac{1}{\sqrt{2}}\).

Four CPs: \((\frac{1}{\sqrt{2}},\frac{1}{\sqrt{2}}), (-\frac{1}{\sqrt{2}},-\frac{1}{\sqrt{2}}), (\frac{1}{\sqrt{2}},-\frac{1}{\sqrt{2}}), (-\frac{1}{\sqrt{2}},\frac{1}{\sqrt{2}})\).

\(f = xy = \frac{1}{2}\) at the first two (maxima), \(f = -\frac{1}{2}\) at the last two (minima).

✅ Max \(f = \frac{1}{2}\) at \((\pm\frac{1}{\sqrt{2}},\pm\frac{1}{\sqrt{2}})\). Min \(f = -\frac{1}{2}\) at \((\pm\frac{1}{\sqrt{2}},\mp\frac{1}{\sqrt{2}})\).
