Q1 — Parametric Line Through a Point
🧠 What is this asking?

Find parametric equations of the line through (1,2,3) parallel to \(\langle 4,-1,2\rangle\).

📝 Full Solution

\(\mathbf{r}(t) = (1,2,3) + t(4,-1,2)\). So: \(x = 1+4t\), \(y = 2-t\), \(z = 3+2t\).

✅ Answer: \(x=1+4t,\; y=2-t,\; z=3+2t\)
Q2 — Equation of a Plane
📝 Full Solution

Through (0,1,−2) with normal \(\mathbf{n}=(2,-3,1)\):

\(2(x-0)-3(y-1)+1(z+2)=0 \Rightarrow 2x-3y+z = -5\).

✅ Answer: \(2x-3y+z=-5\)
Q3 — Angle Between Planes
📝 Full Solution

\(\mathbf{n}_1=(1,2,-1)\), \(\mathbf{n}_2=(2,-1,3)\).

\[\cos\theta = \frac{|2-2-3|}{\sqrt{1+4+1}\sqrt{4+1+9}} = \frac{3}{\sqrt{6}\sqrt{14}} = \frac{3}{\sqrt{84}} = \frac{3}{2\sqrt{21}} = \frac{\sqrt{21}}{14}\]

\(\theta = \arccos\left(\frac{3}{2\sqrt{21}}\right) \approx 70.9°\).

✅ Answer: \(\theta = \arccos\!\left(\dfrac{3}{2\sqrt{21}}\right)\)
Q4 — Arc Length of Space Curve
📝 Full Solution

\(\mathbf{r}(t) = (t, t^2, t^3)\), \(0\le t\le 1\). \(\mathbf{r}'(t) = (1, 2t, 3t^2)\).

\[L = \int_0^1\sqrt{1+4t^2+9t^4}\,dt\]

This has no closed-form antiderivative. Numerically: \(L \approx 1.548\).

If the exam expects a setup-only answer: \(\boxed{L = \int_0^1\sqrt{1+4t^2+9t^4}\,dt}\).

✅ Answer: \(L = \int_0^1\sqrt{1+4t^2+9t^4}\,dt \approx 1.548\)
Q5 — Tangent Vector to Space Curve
📝 Full Solution

\(\mathbf{r}(t) = (\cos t,\sin t, t)\). \(\mathbf{r}'(t) = (-\sin t,\cos t, 1)\).

At \(t=\pi\): \(\mathbf{r}'(\pi) = (-\sin\pi,\cos\pi,1) = (0,-1,1)\).

✅ Answer: \(\langle 0,-1,1\rangle\)
Q6 — Cartesian to Polar Conversion
📝 Full Solution

\(x^2+y^2=4\). Since \(x^2+y^2=r^2\): \(r^2=4 \Rightarrow r=2\).

✅ Answer: \(r=2\)
Q7 — Area Inside Cardioid
📝 Full Solution

\(r = 1+\cos\theta\). Area \(= \frac{1}{2}\int_0^{2\pi}(1+\cos\theta)^2\,d\theta\).

Expand: \((1+\cos\theta)^2 = 1+2\cos\theta+\cos^2\theta = 1+2\cos\theta+\frac{1+\cos 2\theta}{2} = \frac{3}{2}+2\cos\theta+\frac{\cos 2\theta}{2}\).

\[A = \frac{1}{2}\left[\frac{3}{2}\theta+2\sin\theta+\frac{\sin 2\theta}{4}\right]_0^{2\pi} = \frac{1}{2}\cdot 3\pi = \frac{3\pi}{2}\]

✅ Answer: \(\dfrac{3\pi}{2}\)
Q8 — Polar Arc Length: \(r=2\theta\)
📝 Full Solution

\(r=2\theta\), \(r'=2\). \(L = \int_0^{\pi}\sqrt{4\theta^2+4}\,d\theta = 2\int_0^{\pi}\sqrt{\theta^2+1}\,d\theta\).

Using the formula \(\int\sqrt{u^2+1}\,du = \frac{1}{2}(u\sqrt{u^2+1}+\sinh^{-1}u)\):

\[L = 2\cdot\frac{1}{2}\left[\theta\sqrt{\theta^2+1}+\ln(\theta+\sqrt{\theta^2+1})\right]_0^{\pi}\]

\[= \pi\sqrt{\pi^2+1}+\ln(\pi+\sqrt{\pi^2+1})\]

✅ Answer: \(\pi\sqrt{\pi^2+1}+\ln(\pi+\sqrt{\pi^2+1})\)
Q9 — Gradient of \(f=x^2y+yz^3\)
📝 Full Solution

\(\nabla f = (2xy,\; x^2+z^3,\; 3yz^2)\). At (1,1,1): \(\nabla f = (2, 2, 3)\).

✅ Answer: \(\nabla f(1,1,1) = (2,2,3)\)
Q10 — Directional Derivative
📝 Full Solution

\(f=x^2+y^2\). \(\nabla f = (2x,2y)\). At (1,1): \(\nabla f = (2,2)\).

Direction \(\mathbf{v}=(3,4)\), unit vector \(\mathbf{u}=\frac{1}{5}(3,4)\).

\[D_{\mathbf{u}}f = (2,2)\cdot\frac{1}{5}(3,4) = \frac{6+8}{5} = \frac{14}{5}\]

✅ Answer: \(\dfrac{14}{5}\)
Q11 — Chain Rule: \(dz/dt\) for \(z=xy\)
📝 Full Solution

\(z=xy\), \(x=\sin t\), \(y=e^t\).

\[\frac{dz}{dt} = z_x\cdot x' + z_y\cdot y' = y\cos t + xe^t = e^t\cos t + \sin t\cdot e^t = e^t(\cos t+\sin t)\]

✅ Answer: \(\dfrac{dz}{dt} = e^t(\cos t+\sin t)\)
Q12 — Double Integral: \(\int_0^2\int_0^x(x^2+y^2)\,dy\,dx\)
📝 Full Solution

Inner: \(\int_0^x(x^2+y^2)\,dy = x^2y+\frac{y^3}{3}\Big|_0^x = x^3+\frac{x^3}{3} = \frac{4x^3}{3}\).

Outer: \(\int_0^2\frac{4x^3}{3}\,dx = \frac{4}{3}\cdot\frac{x^4}{4}\Big|_0^2 = \frac{4}{3}\cdot 4 = \frac{16}{3}\).

✅ Answer: \(\dfrac{16}{3}\)
Q13 — Triple Integral: \(\iiint_E z\,dV\) over box
📝 Full Solution

Box: \(0\le x\le 1, 0\le y\le 2, 0\le z\le 3\). Separable:

\[\int_0^1 dx\cdot\int_0^2 dy\cdot\int_0^3 z\,dz = 1\cdot 2\cdot\frac{9}{2} = 9\]

✅ Answer: 9
Q14 — Polar Integral to Cartesian
📝 Full Solution

\(\int_0^{2\pi}\int_0^1 r^2\,dr\,d\theta\). In Cartesian, \(r^2=x^2+y^2\) and \(r\,dr\,d\theta = dA\). But the polar integrand is \(r^2\,dr\,d\theta = r\cdot r\,dr\,d\theta = r\,dA\). So \(\int_0^{2\pi}\int_0^1 r^2\,dr\,d\theta = \iint_{x^2+y^2\le 1}\sqrt{x^2+y^2}\,dA\).

Evaluation in polar is easier: \(\int_0^{2\pi}d\theta\cdot\int_0^1 r^2\,dr = 2\pi\cdot\frac{1}{3} = \frac{2\pi}{3}\).

✅ Answer: \(\dfrac{2\pi}{3}\)
Q15 — Jacobian Determinant
📝 Full Solution

\(x=u+v\), \(y=u-v\). \[J = \frac{\partial(x,y)}{\partial(u,v)} = \det\begin{pmatrix}1&1\\1&-1\end{pmatrix} = -1-1 = -2\]

\(|J| = 2\).

✅ Answer: \(J = -2\), \(|J|=2\)
Q16 — Change of Variables Integral
📝 Full Solution

\(\int_0^2\int_0^{2y}(x-y)^2\,dx\,dy\). Let \(u=x-y\), \(v=y\). Then \(x=u+v\), \(y=v\), \(|J|=1\).

When \(x=0\): \(u=-v\). When \(x=2y=2v\): \(u=v\). When \(y=0\): \(v=0\). When \(y=2\): \(v=2\).

\[\int_0^2\int_{-v}^{v} u^2\,du\,dv = \int_0^2\frac{2v^3}{3}\,dv = \frac{2}{3}\cdot\frac{v^4}{4}\Big|_0^2 = \frac{2}{3}\cdot 4 = \frac{8}{3}\]

✅ Answer: \(\dfrac{8}{3}\)
Q17 — Line Integral of \(\mathbf{F}=(y,-x)\) on Unit Circle
📝 Full Solution

Parametrize: \(x=\cos t, y=\sin t\), \(0\le t\le 2\pi\).

\[\oint\mathbf{F}\cdot d\mathbf{r} = \int_0^{2\pi}(\sin t,\;-\cos t)\cdot(-\sin t,\;\cos t)\,dt = \int_0^{2\pi}(-\sin^2 t-\cos^2 t)\,dt = -2\pi\]

Or via Green's: \(Q_x-P_y = 0-1 = -1\). \(\iint -1\,dA = -\pi(1)^2 = -\pi\). Wait — let me recheck.

\(P=y\), \(Q=-x\). \(Q_x=-1\), \(P_y=1\). Curl \(= Q_x-P_y = -1-1 = -2\).

\(\oint\mathbf{F}\cdot d\mathbf{r} = \iint(-2)\,dA = -2\pi\). ✓

✅ Answer: \(-2\pi\)
Q18 — Conservative Test: \(\mathbf{F}=(2xy,x^2)\)
📝 Full Solution

\(P=2xy\), \(Q=x^2\). \(P_y=2x\), \(Q_x=2x\). \(P_y=Q_x\) ✓ → conservative.

✅ Answer: Yes, conservative (since \(\partial P/\partial y = \partial Q/\partial x = 2x\))
Q19 — Potential Function: \(\mathbf{F}=(3x^2+2y,\;2x+4y^3)\)
📝 Full Solution

\(f_x = 3x^2+2y \Rightarrow f = x^3+2xy+g(y)\).

\(f_y = 2x+g'(y) = 2x+4y^3 \Rightarrow g'(y)=4y^3 \Rightarrow g(y)=y^4+C\).

✅ Answer: \(f(x,y) = x^3+2xy+y^4+C\)
Q20 — Divergence of \(\mathbf{F}=(x,y,z)\)
📝 Full Solution

\(\text{div}\,\mathbf{F} = \frac{\partial x}{\partial x}+\frac{\partial y}{\partial y}+\frac{\partial z}{\partial z} = 1+1+1 = 3\).

✅ Answer: 3
Q21 — Curl of \(\mathbf{F}=(yz,xz,xy)\)
📝 Full Solution

\[\nabla\times\mathbf{F} = \left(\frac{\partial(xy)}{\partial y}-\frac{\partial(xz)}{\partial z},\;\frac{\partial(yz)}{\partial z}-\frac{\partial(xy)}{\partial x},\;\frac{\partial(xz)}{\partial x}-\frac{\partial(yz)}{\partial y}\right) = (x-x,\;y-y,\;z-z) = (0,0,0)\]

This field is conservative! (It's \(\nabla(xyz)\).)

✅ Answer: \(\nabla\times\mathbf{F} = \mathbf{0}\)
Q22 — Surface Area of Plane \(z=x+y\)
📝 Full Solution

\(z=x+y\), \(z_x=1\), \(z_y=1\). \(dS = \sqrt{1+1+1}\,dA = \sqrt{3}\,dA\).

\[A = \iint_{[0,1]\times[0,1]}\sqrt{3}\,dA = \sqrt{3}\cdot 1 = \sqrt{3}\]

✅ Answer: \(\sqrt{3}\)
Q23 — Flux of \(\mathbf{F}=(0,0,z)\) Through Paraboloid
📝 Full Solution

Surface: \(z=4-x^2-y^2\) over \(x^2+y^2\le 4\). \(\mathbf{F}=(0,0,z)\).

Flux formula (graph, upward normal): \(\iint(-P z_x-Q z_y+R)\,dA = \iint z\,dA\) where \(z=4-x^2-y^2\).

\[\iint(4-x^2-y^2)\,dA = \int_0^{2\pi}\int_0^2(4-r^2)r\,dr\,d\theta = 2\pi\left[2r^2-\frac{r^4}{4}\right]_0^2 = 2\pi(8-4) = 8\pi\]

✅ Answer: \(8\pi\)
Q24 — Green's Theorem on Unit Square
📝 Full Solution

\(\oint_C(x^2-y)\,dx+(xy+x)\,dy\). \(P=x^2-y\), \(Q=xy+x\).

\(Q_x-P_y = (y+1)-(-1) = y+2\).

\[\iint_{[0,1]^2}(y+2)\,dA = \int_0^1\int_0^1(y+2)\,dy\,dx = \int_0^1\left[\frac{y^2}{2}+2y\right]_0^1 dx = \int_0^1\frac{5}{2}\,dx = \frac{5}{2}\]

✅ Answer: \(\dfrac{5}{2}\)
Q25 — Stokes' Theorem: \(\mathbf{F}=(y,-x,z)\) on Upper Hemisphere
📝 Full Solution

\(\nabla\times\mathbf{F} = (0,0,-1-1) = (0,0,-2)\).

By Stokes': \(\oint_C\mathbf{F}\cdot d\mathbf{r} = \iint_S(0,0,-2)\cdot\mathbf{n}\,dS\).

The boundary C is the unit circle in \(z=0\). Any surface with this boundary gives the same value. Choose the flat disk \(z=0\), \(\mathbf{n}=\mathbf{k}\) (upward).

\[\iint(0,0,-2)\cdot(0,0,1)\,dA = \iint(-2)\,dA = -2\pi\]

✅ Answer: \(-2\pi\)
Q26 — Divergence Theorem: \(\mathbf{F}=(x,y,z)\) on Unit Cube
📝 Full Solution

\(\text{div}\,\mathbf{F} = 1+1+1 = 3\). Volume of \([0,1]^3 = 1\).

\[\iint_S\mathbf{F}\cdot\mathbf{n}\,dS = \iiint 3\,dV = 3\cdot 1 = 3\]

✅ Answer: 3
Q27 — Critical Points of \(f=x^3-3xy^2\)
📝 Full Solution

\(f_x = 3x^2-3y^2 = 0 \Rightarrow x^2=y^2\). \(f_y = -6xy = 0 \Rightarrow x=0\) or \(y=0\).

If \(x=0\): \(y^2=0 \Rightarrow y=0\). If \(y=0\): \(x^2=0 \Rightarrow x=0\).

Only critical point: (0,0).

SDT: \(f_{xx}=6x=0\), \(f_{yy}=-6x=0\), \(f_{xy}=-6y=0\). \(D=0\) → inconclusive.

Check behavior: along \(y=0\): \(f=x^3\) (changes sign → not extremum). This is actually a monkey saddle — a higher-order saddle point.

✅ Answer: Only CP is (0,0); SDT inconclusive (D=0). It is a saddle point (monkey saddle).
Q28 — Lagrange: \(f=xy\) on \(x^2+y^2=1\)
📝 Full Solution

\(\nabla f = (y,x)\), \(\nabla g = (2x,2y)\). Lagrange: \(y=2\lambda x\), \(x=2\lambda y\).

Multiply: \(xy = 4\lambda^2 xy\). If \(xy\ne 0\): \(\lambda^2=\frac{1}{4} \Rightarrow \lambda=\pm\frac{1}{2}\).

\(\lambda=\frac{1}{2}\): \(y=x\). Constraint: \(2x^2=1 \Rightarrow x=\pm\frac{1}{\sqrt{2}}\). \(f=\frac{1}{2}\) → max.

\(\lambda=-\frac{1}{2}\): \(y=-x\). Constraint: \(2x^2=1 \Rightarrow x=\pm\frac{1}{\sqrt{2}}\). \(f=-\frac{1}{2}\) → min.

✅ Answer: Max = \(\frac{1}{2}\) at \((\frac{1}{\sqrt{2}},\frac{1}{\sqrt{2}})\) and \((-\frac{1}{\sqrt{2}},-\frac{1}{\sqrt{2}})\). Min = \(-\frac{1}{2}\) at \((\frac{1}{\sqrt{2}},-\frac{1}{\sqrt{2}})\) and \((-\frac{1}{\sqrt{2}},\frac{1}{\sqrt{2}})\).
