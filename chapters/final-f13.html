A1 — Polar Tangent Slope: \(r=3\sin\theta\) at \(\theta=\pi/2\)
📝 Full Solution

\(r=3\sin\theta\), \(r'=3\cos\theta\). At \(\theta=\pi/2\): \(r=3\), \(r'=0\).

\[\frac{dy}{dx} = \frac{r'\sin\theta+r\cos\theta}{r'\cos\theta-r\sin\theta} = \frac{0\cdot 1+3\cdot 0}{0\cdot 0-3\cdot 1} = \frac{0}{-3} = 0\]

✅ Answer: (a) 0

A3 — Triangle Area via Cross Product
📝 Full Solution

P(1,2,3), Q(−3,0,1), R(2,4,5). \(\vec{PQ}=(-4,-2,-2)\), \(\vec{PR}=(1,2,2)\).

\[\vec{PQ}\times\vec{PR} = \begin{vmatrix}\mathbf{i}&\mathbf{j}&\mathbf{k}\\-4&-2&-2\\1&2&2\end{vmatrix} = ((-2)(2)-(-2)(2),\;(-2)(1)-(-4)(2),\;(-4)(2)-(-2)(1))\]

\[= (-4+4,\;-2+8,\;-8+2) = (0, 6, -6)\]

\[|\vec{PQ}\times\vec{PR}| = \sqrt{0+36+36} = 6\sqrt{2}\]

Area \(= \frac{1}{2}\cdot 6\sqrt{2} = 3\sqrt{2}\).

✅ Answer: (d) \(3\sqrt{2}\)

A7 — Critical Points of \(h(x,y) = y^2-2y-y\sqrt{x}+\frac{x}{2}\)
📝 Full Solution

\(h_x = -\frac{y}{2\sqrt{x}}+\frac{1}{2} = 0 \Rightarrow y = \sqrt{x}\).

\(h_y = 2y-2-\sqrt{x} = 0\). Substitute \(y=\sqrt{x}\): \(2\sqrt{x}-2-\sqrt{x}=0 \Rightarrow \sqrt{x}=2 \Rightarrow x=4, y=2\).

Also check \(x=0\): \(h_y(0,y)=2y-2=0\Rightarrow y=1\). But \(h_x\) involves \(1/\sqrt{x}\) → undefined at \(x=0\). The point (0,0) is a boundary point where the domain starts (\(x\ge 0\)).

✅ Answer: (a) (4,2) and (0,0)
A8 — Double Integral over Rectangle
📝 Full Solution

\(\iint_R e^{-x}\sin y\,dx\,dy\) where \(R: 0\le x\le\ln 2, 0\le y\le\pi/2\).

\[= \int_0^{\ln 2}e^{-x}\,dx\cdot\int_0^{\pi/2}\sin y\,dy = [-e^{-x}]_0^{\ln 2}\cdot[-\cos y]_0^{\pi/2}\]

\[= (-\tfrac{1}{2}+1)(0+1) = \frac{1}{2}\]

✅ Answer: (d) \(\dfrac{1}{2}\)

B1 [8 marks] — Lagrange: \(f=x+2y-2z\) on \(x^2+2y^2+4z^2=1\)
📝 Full Solution

\(\nabla f = (1,2,-2)\), \(\nabla g = (2x,4y,8z)\). Lagrange: \(1=2\lambda x\), \(2=4\lambda y\), \(-2=8\lambda z\).

From each: \(x=\frac{1}{2\lambda}\), \(y=\frac{1}{2\lambda}\), \(z=\frac{-1}{4\lambda}\).

Constraint: \(\frac{1}{4\lambda^2}+\frac{2}{4\lambda^2}+\frac{4}{16\lambda^2} = 1 \Rightarrow \frac{1}{4\lambda^2}+\frac{1}{2\lambda^2}+\frac{1}{4\lambda^2} = 1 \Rightarrow \frac{1}{\lambda^2} = 1 \Rightarrow \lambda = \pm 1\).

\(\lambda=1\): \(x=\frac{1}{2}, y=\frac{1}{2}, z=-\frac{1}{4}\). \(f = \frac{1}{2}+1+\frac{1}{2} = 2\) → maximum.

\(\lambda=-1\): \(x=-\frac{1}{2}, y=-\frac{1}{2}, z=\frac{1}{4}\). \(f = -\frac{1}{2}-1-\frac{1}{2} = -2\) → minimum.

✅ Answer: Max = 2 at \((\frac{1}{2},\frac{1}{2},-\frac{1}{4})\). Min = −2 at \((-\frac{1}{2},-\frac{1}{2},\frac{1}{4})\).
B2 [7 marks] — Polar Change of Variables: \(\int\int\cos(x^2+y^2)\)
📝 Full Solution

Region: \(0\le y\le 1\), \(0\le x\le\sqrt{1-y^2}\) → quarter disk in first quadrant, radius 1.

Polar: \(0\le r\le 1\), \(0\le\theta\le\pi/2\).

\[\int_0^{\pi/2}\int_0^1\cos(r^2)\cdot r\,dr\,d\theta = \frac{\pi}{2}\int_0^1 r\cos(r^2)\,dr\]

Let \(u=r^2\): \(\frac{\pi}{2}\cdot\frac{1}{2}[\sin u]_0^1 = \frac{\pi}{4}\sin 1\).

✅ Answer: \(\dfrac{\pi}{4}\sin 1\)
B3 [7 marks] — Parametric Arc Length
📝 Full Solution

\(x=2t^2, y=1+t^3\), \(0\le t\le 1\). \(x'=4t, y'=3t^2\).

\[L = \int_0^1\sqrt{16t^2+9t^4}\,dt = \int_0^1 t\sqrt{16+9t^2}\,dt\]

Let \(u=16+9t^2\), \(du=18t\,dt\):

\[= \frac{1}{18}\int_{16}^{25}\sqrt{u}\,du = \frac{1}{18}\cdot\frac{2}{3}[u^{3/2}]_{16}^{25} = \frac{1}{27}(125-64) = \frac{61}{27}\]

✅ Answer: \(\dfrac{61}{27}\)
B4 [7 marks] — Area Between Cardioid and Circle
📝 Full Solution

Find area between \(r=3\sin\theta\) (outer) and \(r=1+\sin\theta\) (inner).

Intersection: \(3\sin\theta=1+\sin\theta \Rightarrow \sin\theta=\frac{1}{2} \Rightarrow \theta=\frac{\pi}{6},\frac{5\pi}{6}\).

\[A = \frac{1}{2}\int_{\pi/6}^{5\pi/6}(9\sin^2\theta-(1+\sin\theta)^2)\,d\theta = \frac{1}{2}\int_{\pi/6}^{5\pi/6}(8\sin^2\theta-2\sin\theta-1)\,d\theta\]

Using \(\sin^2\theta = \frac{1-\cos 2\theta}{2}\):

\[= \frac{1}{2}\int_{\pi/6}^{5\pi/6}(3-4\cos 2\theta-2\sin\theta)\,d\theta = \frac{1}{2}\left[3\theta-2\sin 2\theta+2\cos\theta\right]_{\pi/6}^{5\pi/6}\]

\[= \frac{1}{2}\left[(5\pi/2+\sqrt{3}-\sqrt{3})-(\pi/2-\sqrt{3}+\sqrt{3})\right] = \frac{1}{2}\cdot 2\pi = \pi\]

✅ Answer: \(A = \pi\)
B5 [7 marks] — Curl = 0 → Find Potential: \(\mathbf{F}=(\frac{1}{yz},-\frac{x}{y^2z},-\frac{x}{yz^2})\)
📝 Full Solution

(a) Curl F = 0 (verify all three cross-partials match — straightforward computation).

(b) Find f: \(f_x = \frac{1}{yz} \Rightarrow f = \frac{x}{yz}+g(y,z)\).

\(f_y = -\frac{x}{y^2z}+g_y = -\frac{x}{y^2z} \Rightarrow g_y = 0\).

\(f_z = -\frac{x}{yz^2}+g_z = -\frac{x}{yz^2} \Rightarrow g_z = 0\).

✅ Answer: \(f(x,y,z) = \dfrac{x}{yz}\)
B6 [8 marks] — Stokes'/Direct: curl F on disk \(x^2+y^2=4, z=1\)
📝 Full Solution

\(\mathbf{F}=(-5y,4x,z)\). \(\text{curl}\,\mathbf{F} = (0,0,4-(-5)) = (0,0,9)\).

Surface: \(z=1\), \(\mathbf{n}=\mathbf{k}\). \(\text{curl}\,\mathbf{F}\cdot\mathbf{n}=9\).

\[\iint_S 9\,dS = 9\cdot\pi(2^2) = 36\pi\]

✅ Answer: \(36\pi\)
