Chapter 7

Maxima, Minima & Lagrange Multipliers

Finding and classifying critical points of multivariable functions. The Second Derivative Test for surfaces. Constrained optimization via Lagrange multipliers — the elegant technique for optimizing one function subject to another equation.

§7.2 — Maxima and Minima

In single-variable calculus you found max/min by setting f'(x) = 0, then used the second derivative to classify. We now extend this idea to functions of two (or more) variables: z = f(x, y). The core question is the same — where are the peaks and valleys of a surface? — but we now have two partial derivatives to set to zero simultaneously, and the classification test is more involved.

Critical Points

Recall from §3.5 that the gradient ∇f = (f_x, f_y) points in the direction of steepest increase. At a maximum or minimum, there is no direction of increase or decrease — the surface is momentarily "flat." This means the gradient must be the zero vector.

Critical Point

A point P(a, b) in the domain of f(x, y) is a critical point (or stationary point) if both partial derivatives vanish at P:

\[\frac{\partial f}{\partial x}(a, b) = 0 \quad \text{and} \quad \frac{\partial f}{\partial y}(a, b) = 0\]
f_x = partial w.r.t. xf_y = partial w.r.t. y(a,b) = coordinates of the critical point

Equivalently: ∇f(a, b) = 0. Every local maximum and every local minimum of a differentiable function occurs at a critical point. However, not every critical point is a max or min — it could be a saddle point.

The reasoning is clean: if P is a local max, then along any line through P the function has a single-variable max. In particular, moving along the x-direction, f(x, b) has a max at x = a, forcing f_x(a, b) = 0. The same argument along the y-direction gives f_y(a, b) = 0.

★ Step-by-Step Strategy for Finding Critical Points

Step 1: Compute both partial derivatives f_x and f_y.

Step 2: Set f_x = 0 and f_y = 0 simultaneously — you need BOTH to be zero at the same point.

Step 3: Factor. Look for common factors. For example, f_x = y(2x + y − 1) = 0 gives two branches: y = 0 OR 2x + y − 1 = 0.

Step 4: Case analysis. For each branch from Step 3, substitute into the OTHER equation and solve. Track every solution pair (x, y). This is where most students lose points — they check one case and forget the others.

Step 5: List ALL critical points found from ALL cases. Verify each satisfies BOTH f_x = 0 and f_y = 0.

Step 6: Classify each one using the Second Derivative Test (compute D at each point).

The Second Derivative Test

Once you have the critical points, you need to classify each one: is it a local max, local min, or saddle point? In single-variable calculus, you check the sign of f''(x). For two variables, we need to consider not just f_xx and f_yy, but also the mixed partial f_xy, because the surface can curve differently in different directions.

The key quantity is the discriminant D, built from the second-order partial derivatives evaluated at the critical point:

Discriminant (Hessian Determinant)
\[D = f_{xx} \cdot f_{yy} - (f_{xy})^2 \quad \text{at } P(a,b)\]
f_xx = ∂²f/∂x²f_yy = ∂²f/∂y²f_xy = ∂²f/∂x∂y (mixed partial)

This is the determinant of the Hessian matrix \(H = \begin{pmatrix} f_{xx} & f_{xy} \\ f_{xy} & f_{yy}\end{pmatrix}\). The discriminant tells you whether the surface curves the same way in all directions (D > 0) or curves up in one direction and down in another (D < 0).

Second Derivative Test (for two variables)

Let P(a, b) be a critical point of f (so f_x = f_y = 0 at P). Compute D = f_xx · f_yy − (f_xy)² at P. Then:

\[\begin{cases} D > 0 \text{ and } f_{xx} > 0 &\Rightarrow \text{local minimum at } P \\ D > 0 \text{ and } f_{xx} < 0 &\Rightarrow \text{local maximum at } P \\ D < 0 &\Rightarrow \text{saddle point at } P \\ D = 0 &\Rightarrow \text{inconclusive (need other methods)}\end{cases}\]
D = f_xx f_yy − f_xy² (discriminant)f_xx determines concavity when D > 0
★ Why f_xx and not f_yy?

When D > 0, the signs of f_xx and f_yy must agree (since D = f_xx · f_yy − f_xy² > 0 forces f_xx and f_yy to have the same sign). So checking f_xx alone suffices — you could equally check f_yy and get the same answer.

Saddle Point

A critical point where D < 0 is called a saddle point. At a saddle point, the surface curves upward in one direction and downward in another — like a mountain pass or horse saddle. The function is neither a local max nor a local min there, regardless of the signs of f_xx and f_yy individually.

Fig 7.1 — The three outcomes of the Second Derivative Test. Left: local minimum (valley, D > 0, f_xx > 0). Center: local maximum (hilltop, D > 0, f_xx < 0). Right: saddle point (up one way and down the other, D < 0). Solid curve = x-direction cross-section, dashed = y-direction.

Worked Examples — Critical Points & Second Derivative Test

Example 7.1 — Relative Extrema of a Quadratic Surface

Problem: Find the relative maxima and minima of \(z = 2x + 4y - x^2 - y^2 - 3\).

Show full solution

Thought process: This is a degree-2 function with negative leading coefficients on x² and y². We expect a single hilltop (maximum). Set both partials to zero.

Step 1 — Find critical points:

\[f_x = 2 - 2x = 0 \;\Rightarrow\; x = 1\]
\[f_y = 4 - 2y = 0 \;\Rightarrow\; y = 2\]

The only critical point is P(1, 2).

Step 2 — Compute the discriminant:

\[f_{xx} = -2, \quad f_{yy} = -2, \quad f_{xy} = 0\]
\[D = (-2)(-2) - 0^2 = 4 > 0\]

Step 3 — Classify: D > 0 and f_xx = −2 < 0, so P(1, 2) is a local maximum.

The maximum value is f(1, 2) = 2 + 8 − 1 − 4 − 3 = 2.

Note: since the domain is all of ℝ² and f → −∞ as ||(x,y)|| → ∞, this local max is also the global maximum. There is no global minimum.

Problem: Classify the critical points of \(f(x, y) = x^2 y + x y^2 - xy\) on ℝ².

\[f_y = x^2 + 2xy - x = x(x + 2y - 1) = 0\]

Extrema on Closed Regions

When f is continuous on a closed, bounded region 𝒟 (including its boundary), the Extreme Value Theorem guarantees a global max and min. Finding them requires checking both interior and boundary.

Local vs. Global Extrema

A local maximum at P means f(P) ≥ f(Q) for all Q near P. A global (absolute) maximum means f(P) ≥ f(Q) for all Q in the domain. The global max is the largest of all local maxima. Similarly for minima. On a closed bounded domain, both global max and global min are guaranteed to exist.

Fig 7.2 — Local vs. global extrema on a closed domain [a, b]. The global min is the lowest of all local minima and boundary values. The global max is the highest. On a closed bounded region, both are guaranteed to exist (Extreme Value Theorem). Double-ring markers ◎ denote the global extrema.
Closed-Region Extrema Algorithm

1. Find and classify all critical points inside 𝒟 (not on the boundary) using the SDT.

2. Find critical points of f on the boundary — parametrize each boundary curve and reduce to single-variable optimization.

3. Evaluate f at all critical points (interior + boundary) and at all corners/endpoints.

4. The largest value = absolute maximum; smallest = absolute minimum.

★ Always Check Corners/Endpoints

Derivatives don't exist at corners of a polygonal boundary. These points must be checked by direct evaluation. On a rectangle: 4 corners. On a triangle: 3. On a disk: the boundary parametrization handles everything (no corners).

Fig 7.3 — Absolute extrema on a closed rectangle: check interior CPs (from f_x = f_y = 0), boundary CPs (single-variable calculus on each edge), and all 4 corners (direct evaluation).

§7.3 — Lagrange Multipliers

What if we need to optimize f(x, y) but the variables must satisfy a constraint g(x, y) = 0? The method of Lagrange multipliers handles this without substitution.

The Geometric Idea — Why Lagrange Works

Imagine you're hiking on a mountain (the surface z = f(x, y)) but you're constrained to walk along a specific trail (the curve g(x, y) = 0). Where on this trail is the highest point? It's NOT necessarily where the mountain's own peak is — it's the highest point reachable from the trail.

Here's the key insight: as you walk along the constraint curve, the function f changes. At the constrained maximum, f has stopped increasing along the trail. This means the gradient ∇f (which points in the direction of steepest increase of f) must be perpendicular to the trail at that point — if ∇f had any component along the trail, you could still increase f by walking further.

But wait — ∇g is ALSO perpendicular to the constraint curve g = 0 (recall from §3.5 that the gradient is always perpendicular to level curves). Since both ∇f and ∇g point perpendicular to the same curve at the same point, they must be parallel:

\[\nabla f(P) = \lambda \, \nabla g(P)\]

where λ is a scalar called the Lagrange multiplier. We don't care about the value of λ itself — it's just a ratio telling us how much bigger (or smaller) ∇f is compared to ∇g.

★ The "Kissing Curves" Picture

Think of the level curves of f (f = 1, f = 2, f = 3, ...) as concentric rings. The constraint g = 0 is another curve cutting through them. As you "inflate" the level curves (increasing c), at some point the level curve f = c will just touch ("kiss") the constraint curve — tangent to it, not crossing it. At that tangent point, the two curves share the same normal direction, meaning ∇f ∥ ∇g. That's your constrained optimum!

Fig 7.4 — At a constrained extremum P, ∇f is parallel to ∇g (both perpendicular to the constraint curve g = 0). The dashed ellipses are level curves of f. The constrained optimum occurs where a level curve "kisses" the constraint.
Lagrange Multiplier Method — Step by Step

Given: Optimize f(x, y) subject to the constraint g(x, y) = 0.

Step 1: Write the Lagrangian: \(\ell(x, y, \lambda) = f(x, y) - \lambda \, g(x, y)\)

Step 2: Take partial derivatives and set each to zero:

\[\frac{\partial \ell}{\partial x} = 0 \;\Rightarrow\; f_x = \lambda \, g_x \qquad \frac{\partial \ell}{\partial y} = 0 \;\Rightarrow\; f_y = \lambda \, g_y \qquad \frac{\partial \ell}{\partial \lambda} = 0 \;\Rightarrow\; g(x,y) = 0\]

Step 3: Solve this system of 3 equations in 3 unknowns (x, y, λ). Use case analysis: try λ = 0 vs λ ≠ 0, or eliminate λ by dividing equations.

Step 4: Evaluate f at every solution (x, y). The largest value = constrained max; smallest = constrained min.

∂ℓ/∂λ = 0 recovers the constraintλ is a helper variable — its value usually doesn't matter

For 3 variables: ℓ(x,y,z,λ) = f − λg. Set ∂ℓ/∂x = ∂ℓ/∂y = ∂ℓ/∂z = ∂ℓ/∂λ = 0 (4 equations, 4 unknowns).

★ Lagrange Does NOT Classify

Lagrange finds candidates. To classify, evaluate f at all candidates and compare. The largest value is the constrained max and the smallest is the constrained min. On a closed constraint (like a circle), both must exist.

★ Case-by-Case Analysis

Lagrange systems involve products. When a factor can be zero, consider all possibilities: if y = 0, what does the constraint give? If y ≠ 0, divide and continue. Missing a case = missing critical points.

Worked Examples — Lagrange Multipliers

Example 7.5 — Largest Rectangle in a Semicircle

Problem: Find the area of the largest rectangle inscribed in a semicircle of radius a.

Show full solution

Setup: Place semicircle centered at origin. Rectangle has corners (±x, 0) and (±x, y) with (x,y) on x²+y² = a². Maximize A = 2xy with constraint g = x²+y²−a² = 0.

Lagrangian: ℓ = 2xy − λ(x²+y²−a²).

ℓ_x = 2y − 2λx = 0 → y = λx [1]

ℓ_y = 2x − 2λy = 0 → x = λy [2]

ℓ_λ = −(x²+y²−a²) = 0 → x²+y² = a² [3]

From [1] and [2]: y = λx and x = λy = λ²x. Since y ≠ 0: λ² = 1 → λ = ±1. With x,y > 0: y = x.

From [3]: 2x² = a² → x = a/√2 = y.

Max area = 2·(a/√2)·(a/√2) = a².

The largest inscribed rectangle is a square of the circle's radius!

Example 7.6 — Distance from Origin to a Rotated Ellipse

Problem: Find the shortest and longest distance from the origin to \(x^2 + xy + y^2 = 16\).

Show full solution

Thought process: Minimize/maximize f = x²+y² (distance²) subject to g = x²+xy+y²−16 = 0. The curve is a rotated ellipse (B²−4AC = 1−4 = −3 < 0; recall conic discriminant from §2.8).

Lagrangian: ℓ = x²+y² − λ(x²+xy+y²−16).

ℓ_x = 2x − λ(2x+y) = 0 [1]

ℓ_y = 2y − λ(x+2y) = 0 [2]

ℓ_λ = −(x²+xy+y²−16) = 0 [3]

Case λ = 0: → x = y = 0, violates [3]. No solution.

Case λ ≠ 0, 2x+y ≠ 0: λ = 2x/(2x+y). Sub into [2] and simplify → y²−x² = 0 → y = ±x.

y = x: 3x² = 16 → x = ±4/√3. Distance² = 32/3. Distance = √(32/3).

y = −x: x² = 16 → x = ±4. Distance² = 32. Distance = √32.

Shortest = 4√(2/3). Longest = 4√2.

Example 7.7 — Lagrange in Three Variables

Problem: Maximize \(f = xy^2z^3\) subject to \(x+y+z = 12\) with x,y,z > 0.

Show full solution

Setup: g = x+y+z−12 = 0. Lagrangian: ℓ = xy²z³ − λ(x+y+z−12).

ℓ_x = y²z³ − λ = 0, ℓ_y = 2xyz³ − λ = 0, ℓ_z = 3xy²z² − λ = 0.

Setting all three equal to λ: y²z³ = 2xyz³ = 3xy²z².

From first pair: y = 2x. From first and third: z = 3x.

Constraint: x + 2x + 3x = 12 → x = 2, y = 4, z = 6.

Max = 2·16·216 = 6912 at (2, 4, 6).

Pattern: Exponents (1, 2, 3) split the sum proportionally: x gets 1/6, y gets 2/6, z gets 3/6 of 12.

Example 7.8 — Interior + Lagrange Boundary

Problem: Find the absolute max/min of \(f = 3x^2+2y^2-2x-1\) on the disk \(x^2+y^2 \le 9\).

Show full solution

Interior: f_x = 6x−2 = 0 → x = 1/3. f_y = 4y = 0 → y = 0. Check: (1/3)²+0² = 1/9 < 9 ✓. Value: f(1/3,0) = 1/3−2/3−1 = −4/3.

Boundary (Lagrange): g = x²+y²−9 = 0. ℓ = f − λg.

ℓ_x = 6x−2−2λx = 0 → x(3−λ) = 1 [1]

ℓ_y = 4y−2λy = 0 → y(2−λ) = 0 [2]

Case y = 0: x² = 9 → x = ±3. f(3,0) = 20, f(−3,0) = 32.

Case λ = 2: From [1]: x = 1. From [3]: y² = 8 → y = ±√8. f(1,±√8) = 3+16−2−1 = 16.

Example 7.9 — Substitution Method (Compare with Lagrange)

Problem: Maximize xy²z³ with x+y+z = 12 and x,y,z > 0 using substitution instead of Lagrange.

Show full solution

Substitute x = 12−y−z → g(y,z) = (12−y−z)y²z³.

g_y = yz³(24−3y−2z) = 0. Since y,z > 0: 24−3y−2z = 0.

g_z = y²z²(36−3y−4z) = 0. Since y,z > 0: 36−3y−4z = 0.

Subtract first from second: 12−2z = 0 → z = 6, y = 4, x = 2.

Second derivative test at (4,6): D = f_yy·f_zz − f_yz² = 2985984 > 0 and g_yy = −2592 < 0 → local max.

Max value = 6912. Same answer as Lagrange (Ex 7.7)!

★ Substitution vs. Lagrange — When to Use Which

Substitution is simplest when the constraint is linear (easy to isolate one variable). Lagrange is symmetric and cleaner for nonlinear constraints (circles, ellipses, spheres). Both give the same answer — choose based on which creates less algebraic mess.
