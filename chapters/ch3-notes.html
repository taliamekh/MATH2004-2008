Chapter 3

Partial Derivatives & Vector Fields

We extend calculus to functions of several variables: limits, continuity, partial derivatives, directional derivatives, the gradient, the multivariable chain rule, tangent planes, and the trio of vector field operators — gradient, curl, and divergence.

§3.1 — Limits and Continuity

In single-variable calculus, a function \(y=f(x)\) takes one input and produces one output. Now we work with functions of two (or more) variables: \(z = f(x,y)\). The graph of such a function is a surface in \(\mathbb{R}^3\). Just as we studied limits of \(f(x)\) as \(x \to x_0\), we now study limits of \(f(x,y)\) as the point \((x,y)\) approaches a target point \((x_0,y_0)\) — but here the approach can come from infinitely many directions, not just left and right.

Limit of a Function of Two Variables

We write

\[\lim_{(x,y)\to(x_0,y_0)} f(x,y) = L\]

if for every \(\varepsilon > 0\) there exists a \(\delta > 0\) such that whenever \(\sqrt{(x-x_0)^2+(y-y_0)^2} < \delta\), we have \(|f(x,y)-L| < \varepsilon\).

In words: no matter how tight a tolerance we demand on the output (\(\varepsilon\)), we can find a small enough disk around \((x_0,y_0)\) (\(\delta\)) so that every point inside that disk maps to within \(\varepsilon\) of \(L\).

★ Key Difference from Single-Variable

In one dimension, \(x \to x_0\) means approaching from the left or right (two directions). In two dimensions, \((x,y) \to (x_0,y_0)\) means approaching along any path whatsoever — straight lines, parabolas, spirals, or any curve. If even one path gives a different limit, the limit does not exist.

Proving a limit exists typically requires an \(\varepsilon\)-\(\delta\) argument or algebraic bounding. Proving a limit does NOT exist is often easier: find two paths to the same point that give different limiting values.

Example 3.1 — Proving a Limit Exists (ε-δ)

Problem: Prove that \(\displaystyle\lim_{(x,y)\to(0,0)} \frac{x^2 y^2}{x^2+y^2} = 0.\)

Show Solution ▾

Thought process: Plugging in gives 0/0 — indeterminate. We guess \(L = 0\) since the numerator has degree 4 while the denominator has degree 2, so the fraction should shrink to 0. We need an ε-δ proof.

Setup: We must show: for every \(\varepsilon > 0\), there exists \(\delta > 0\) such that if \(\sqrt{x^2+y^2} < \delta\), then \(\left|\frac{x^2y^2}{x^2+y^2}\right| < \varepsilon\).

Solution:

Since \(|x| \le \sqrt{x^2+y^2} < \delta\) and \(|y| \le \sqrt{x^2+y^2} < \delta\), multiplying gives \(|x||y| \le \delta^2\).

Also, \((|x|-|y|)^2 \ge 0\) implies \(x^2+y^2 \ge 2|x||y|\), so \(\frac{1}{x^2+y^2} \le \frac{1}{2|x||y|}\).

Therefore: \[\frac{x^2y^2}{x^2+y^2} \le \frac{|x|^2|y|^2}{2|x||y|} = \frac{|x||y|}{2} < \frac{\delta^2}{2}.\]

Choose \(\delta = \sqrt{2\varepsilon}\). Then \(\delta^2/2 = \varepsilon\), so the expression is less than \(\varepsilon\). ∎

\[\boxed{\lim_{(x,y)\to(0,0)} \frac{x^2 y^2}{x^2+y^2} = 0}\]
★ Tip

When bounding, use \(|x| \le \sqrt{x^2+y^2}\) and \(|y| \le \sqrt{x^2+y^2}\) freely. The key trick is expressing everything in terms of \(\delta = \sqrt{x^2+y^2}\).

Example 3.2 — Proving a Limit Does Not Exist (Two-Path Test)

Problem: Prove that \(\displaystyle\lim_{(x,y)\to(0,0)} \frac{xy}{x^2+y^2}\) does not exist.

Show Solution ▾

Thought process: Try two different paths to \((0,0)\). If they give different limits, the overall limit cannot exist.

Path 1 — along \(y = 0\): \(\frac{x \cdot 0}{x^2+0} = 0\). Limit = 0.

Path 2 — along \(y = x\): \(\frac{x \cdot x}{x^2+x^2} = \frac{x^2}{2x^2} = \frac{1}{2}\). Limit = 1/2.

Since \(0 \neq 1/2\), the two paths give different limits.

\[\boxed{\text{The limit does not exist (DNE).}}\]
★ Tip

Popular test paths: \(y=0\), \(x=0\), \(y=x\), \(y=mx\), \(y=x^2\). If \(y=mx\) gives a limit that depends on \(m\), the limit doesn't exist. You can also try polar: \(x=r\cos\theta, y=r\sin\theta\) — if the limit as \(r\to 0\) depends on \(\theta\), it DNE.

Figure 3.1 — The two-path test: if different paths to (0,0) give different limits, the overall limit does not exist.
Continuity of f(x,y) at a Point

A function \(f\) is continuous at \((x_0,y_0)\) if:

1. \(f(x_0,y_0)\) is defined (a finite real number), and

2. \(\displaystyle\lim_{(x,y)\to(x_0,y_0)} f(x,y) = f(x_0,y_0)\).

Geometrically: the surface \(z=f(x,y)\) has no break, hole, or jump at that point.

★ Key Fact — Continuous Functions

All polynomials in \(x\) and \(y\) (like \(x^2y + y^2 - 3xy + 6\)) are continuous everywhere. Rational functions (ratio of polynomials) are continuous wherever the denominator is nonzero. Compositions with \(\sin, \cos, e^x, \ln\) remain continuous on their domains.

Example 3.3 — Proving Continuity

Problem: Show that \(f(x,y) = \sin(xy)\) is continuous at \((0,0)\).

Show Solution ▾

Step 1: \(f(0,0) = \sin(0) = 0\). ✓ Defined.

Step 2: Show \(\lim \sin(xy) = 0\). Use \(|\sin r| \le |r|\): \(|f(x,y)| = |\sin(xy)| \le |xy| \le |x||y| < \delta^2\).

Choose \(\delta = \sqrt{\varepsilon}\). Then \(|f(x,y)| < \varepsilon\). ∎

\[\boxed{f(x,y) = \sin(xy) \text{ is continuous at } (0,0).}\]
Example 3.4 — Function That Cannot Be Made Continuous

Problem: Let \(f(x,y) = \begin{cases} \frac{y^2}{x^2+y^2} & (x,y)\neq(0,0) \\ c & (x,y)=(0,0) \end{cases}\). Show \(f\) is not continuous at \((0,0)\) for any \(c\).

Show Solution ▾

Path 1 — \(y=0\): \(f(x,0) = 0\). Limit = 0, forces \(c = 0\).

Path 2 — \(x=0\): \(f(0,y) = y^2/y^2 = 1\). Limit = 1, forces \(c = 1\).

Contradiction: \(c\) can't be both 0 and 1.

\[\boxed{f \text{ cannot be made continuous at } (0,0) \text{ for any } c.}\]

§3.2 — Partial Derivatives

When a function depends on several variables, we can ask: how does the output change when we wiggle just one input, holding all others fixed? This is a partial derivative.

Partial Derivative Definitions
\[\frac{\partial f}{\partial x}(x_0,y_0) = \lim_{h\to 0} \frac{f(x_0+h,\,y_0) - f(x_0,y_0)}{h}, \qquad \frac{\partial f}{\partial y}(x_0,y_0) = \lim_{h\to 0} \frac{f(x_0,\,y_0+h) - f(x_0,y_0)}{h}\]
\(\partial f/\partial x\) — hold \(y\) constant, differentiate w.r.t. \(x\) \(\partial f/\partial y\) — hold \(x\) constant, differentiate w.r.t. \(y\)

Geometric meaning: \(f_x(x_0,y_0)\) is the slope of the curve formed by slicing the surface \(z = f(x,y)\) with the plane \(y = y_0\). Similarly, \(f_y\) is the slope of the slice by the plane \(x = x_0\). Each partial derivative gives the rate of change along one coordinate axis.

Notation: All equivalent: \(\frac{\partial f}{\partial x},\ f_x,\ f_{(1)},\ D_xf\).

★ NOT Implicit Differentiation

Partial differentiation treats \(x\) and \(y\) as independent variables. Implicit differentiation (§3.6.2) assumes one variable depends on the others. Don't confuse them.

Example 3.5 — Computing Partial Derivatives

Problem: Let \(z = x^2 + 3xy + y^2\). Find \(\partial z/\partial x\) and \(\partial z/\partial y\).

Show Solution ▾

For \(\partial z/\partial x\) — treat \(y\) as a constant:

\[\frac{\partial z}{\partial x} = \frac{\partial}{\partial x}(x^2) + \frac{\partial}{\partial x}(3xy) + \frac{\partial}{\partial x}(y^2) = 2x + 3y + 0 = 2x + 3y\]

(\(y^2\) is a constant → derivative is 0. In \(3xy\), the factor \(3y\) is a constant coefficient.)

For \(\partial z/\partial y\) — treat \(x\) as a constant:

\[\frac{\partial z}{\partial y} = 0 + 3x + 2y = 3x + 2y\]

\[\boxed{\frac{\partial z}{\partial x} = 2x+3y, \qquad \frac{\partial z}{\partial y} = 3x+2y}\]
Example 3.6 — Three-Variable Partial Derivatives

Problem: Let \(f(x,y,z) = \ln(x+y^2+z^3)\). Find \(f_x, f_y, f_z\).

Show Solution ▾

Chain rule with outer \(\ln u\), inner \(u = x+y^2+z^3\):

\[f_x = \frac{1}{x+y^2+z^3}, \quad f_y = \frac{2y}{x+y^2+z^3}, \quad f_z = \frac{3z^2}{x+y^2+z^3}\]

\[\boxed{f_x = \frac{1}{x+y^2+z^3},\quad f_y = \frac{2y}{x+y^2+z^3},\quad f_z = \frac{3z^2}{x+y^2+z^3}}\]

§3.3 — Higher-Order Partial Derivatives

Just as in single-variable calculus, we can differentiate partial derivatives again. The four second-order partials of \(f(x,y)\) are: \(f_{xx}\), \(f_{xy}\), \(f_{yx}\), and \(f_{yy}\).

Clairaut's Theorem (Young's Theorem)
\[f_{xy} = f_{yx}\]
\(f_{xy}\) — differentiate w.r.t. \(x\) first, then \(y\) \(f_{yx}\) — differentiate w.r.t. \(y\) first, then \(x\)

If \(f\) and all its partial derivatives up to order 2 are continuous, then mixed partials are equal — the order of differentiation doesn't matter. This extends to higher orders.

Example 3.7 — Second-Order Partials and Clairaut's Theorem

Problem: Let \(z = 2x^2 - 5xy + y^2\). Find all four second-order partials. Verify \(z_{xy}=z_{yx}\).

Show Solution ▾

First partials: \(z_x = 4x - 5y\), \(\;z_y = -5x + 2y\).

Second partials:

\(z_{xx} = \frac{\partial}{\partial x}(4x-5y) = 4\)

\(z_{xy} = \frac{\partial}{\partial y}(4x-5y) = -5\)

\(z_{yx} = \frac{\partial}{\partial x}(-5x+2y) = -5\)

\(z_{yy} = \frac{\partial}{\partial y}(-5x+2y) = 2\)

✓ \(z_{xy} = z_{yx} = -5\) — Clairaut's theorem confirmed.

\[\boxed{z_{xx}=4,\quad z_{xy}=z_{yx}=-5,\quad z_{yy}=2}\]

§3.5 — Directional Derivatives and Gradients

Partial derivatives measure rates of change along the coordinate axes. But what about any other direction? The directional derivative answers this, and the gradient ties it all together.

The Gradient

The gradient of \(f\) at a point \(P\), written \(\nabla f(P)\), is the vector of all partial derivatives:

\[\nabla f = \left(\frac{\partial f}{\partial x},\;\frac{\partial f}{\partial y}\right) \quad\text{(2D)}, \qquad \nabla f = \left(\frac{\partial f}{\partial x},\;\frac{\partial f}{\partial y},\;\frac{\partial f}{\partial z}\right) \quad\text{(3D)}\]

Directional Derivative via the Gradient
\[D_{\mathbf{u}}f(P) = \nabla f(P) \cdot \mathbf{u} = |\nabla f(P)|\,|\mathbf{u}|\cos\theta\]
\(D_{\mathbf{u}}f\) — rate of change in direction \(\mathbf{u}\) \(\nabla f\) — gradient vector \(\theta\) — angle between \(\nabla f\) and \(\mathbf{u}\)

When \(\mathbf{u}\) is a unit vector, this is the directional derivative: the instantaneous rate of change per unit distance in that direction. Setting \(\mathbf{u} = \mathbf{i}\) recovers \(f_x\); setting \(\mathbf{u} = \mathbf{j}\) recovers \(f_y\).

Direction of Steepest Ascent & Descent
\[\max D_{\mathbf{u}}f = |\nabla f| \;\;\text{when } \mathbf{u} = \frac{\nabla f}{|\nabla f|}, \qquad \min D_{\mathbf{u}}f = -|\nabla f| \;\;\text{when } \mathbf{u} = -\frac{\nabla f}{|\nabla f|}\]
\(|\nabla f|\) — maximum rate of increase \(\nabla f/|\nabla f|\) — direction of steepest ascent

\(f\) increases fastest in the gradient direction. It decreases fastest opposite to the gradient. Perpendicular to the gradient (\(\cos\theta = 0\)), \(D_{\mathbf{u}}f = 0\) — you're on a level curve.

★ Gradient Perpendicular to Level Curves

The gradient \(\nabla f\) at any point is perpendicular to the level curve \(f(x,y) = c\) through that point, and points toward increasing \(f\). This is why hikers going straight uphill are walking in the gradient direction, and contour lines on a map are perpendicular to the steepest slope.

Example 3.8 — Maximum Directional Derivative

Problem: Let \(f(x,y) = e^x\cos y\), \(P(2, -\pi/4)\). Find the unit vector giving the maximum \(D_{\mathbf{u}}f\) and its value.

Show Solution ▾

Gradient: \(\nabla f = (e^x\cos y,\;-e^x\sin y)\). At \(P\): \(\nabla f = \left(\frac{e^2}{\sqrt{2}},\;\frac{e^2}{\sqrt{2}}\right)\).

\(|\nabla f| = \sqrt{\frac{e^4}{2}+\frac{e^4}{2}} = e^2\). Direction: \(\mathbf{u} = \frac{\nabla f}{|\nabla f|} = \left(\frac{1}{\sqrt{2}},\frac{1}{\sqrt{2}}\right)\).

\[\boxed{\mathbf{u} = \left(\tfrac{1}{\sqrt{2}},\tfrac{1}{\sqrt{2}}\right), \quad \max D_{\mathbf{u}}f = e^2}\]
Example 3.9 — Directional Derivative in a Given Direction

Problem: Let \(f(x,y) = x^2-y^2\), \(P(2,-1)\), \(\mathbf{u} = \mathbf{i}-\mathbf{j}\). Find \(D_{\mathbf{u}}f(P)\) and the directional derivative at \(P\) in the same direction.

Show Solution ▾

Gradient: \(\nabla f = (2x, -2y)\). At \(P(2,-1)\): \(\nabla f = (4, 2)\).

Derivative w.r.t. \(\mathbf{u}\): \(D_{\mathbf{u}}f = \nabla f \cdot \mathbf{u} = (4,2)\cdot(1,-1) = 4 - 2 = 2\).

Directional derivative (requires unit vector): \(\mathbf{v} = \frac{\mathbf{u}}{|\mathbf{u}|} = \frac{(1,-1)}{\sqrt{2}}\).

\(D_{\mathbf{v}}f = \nabla f \cdot \mathbf{v} = (4,2)\cdot\frac{(1,-1)}{\sqrt{2}} = \frac{2}{\sqrt{2}} = \sqrt{2}\).

Check: \(D_{\mathbf{u}}f = |\mathbf{u}|\cdot D_{\mathbf{v}}f = \sqrt{2}\cdot\sqrt{2} = 2\). ✓

\[\boxed{D_{\mathbf{u}}f(P) = 2, \qquad D_{\mathbf{v}}f(P) = \sqrt{2}}\]

§3.6 — The Chain Rule in ℝ² and ℝ³

When the input variables of \(f\) are themselves functions of other parameters, we chain the derivatives. The multivariable chain rule generalizes \(\frac{d}{dt}f(r(t)) = f'(r(t))\,r'(t)\) to higher dimensions.

Chain Rule — One Parameter
\[\frac{dz}{dt} = \frac{\partial z}{\partial x}\frac{dx}{dt} + \frac{\partial z}{\partial y}\frac{dy}{dt}\]
\(z = f(x,y)\) where \(x = x(t),\;y = y(t)\) \(dz/dt\) — total derivative w.r.t. \(t\)

For three variables: add \(\frac{\partial z}{\partial w}\frac{dw}{dt}\). The pattern: sum over all intermediate variables.

Chain Rule — Two Parameters
\[\frac{\partial z}{\partial s} = \frac{\partial z}{\partial x}\frac{\partial x}{\partial s} + \frac{\partial z}{\partial y}\frac{\partial y}{\partial s}, \qquad \frac{\partial z}{\partial t} = \frac{\partial z}{\partial x}\frac{\partial x}{\partial t} + \frac{\partial z}{\partial y}\frac{\partial y}{\partial t}\]
\(x = x(s,t),\;y = y(s,t)\) — depend on two parameters
★ Tree Diagram Mnemonic

Draw a tree: \(z\) at the top branches to \(x\) and \(y\); each branches to \(s,t\). Each path from \(z\) to a bottom variable gives one term (multiply partials along the path). Add all paths to the same bottom variable.

Figure 3.3 — Chain rule tree: multiply partials along each path from z to t, then add. Each root-to-leaf path gives one term.
Example 3.10 — Related Rates via Chain Rule

Problem: A cylinder has radius 6″ (increasing at 0.2 in/s) and height 8″ (decreasing at 0.4 in/s). Find \(dV/dt\).

Show Solution ▾

\(V = \pi r^2 h\). \(\frac{dV}{dt} = 2\pi rh\frac{dr}{dt} + \pi r^2\frac{dh}{dt} = 2\pi(6)(8)(0.2) + \pi(36)(-0.4) = 19.2\pi - 14.4\pi\).

\[\boxed{\frac{dV}{dt} = 4.8\pi \approx 15.08\;\text{in}^3/\text{s}}\]

§3.6.2 — Implicit Differentiation

When a surface is defined implicitly by \(f(x,y,z) = 0\), we can find \(\partial z/\partial x\) and \(\partial z/\partial y\) without solving for \(z\) explicitly.

Implicit Partial Derivatives
\[\frac{\partial z}{\partial x} = -\frac{f_x}{f_z}, \qquad \frac{\partial z}{\partial y} = -\frac{f_y}{f_z}\]
\(f(x,y,z) = 0\) — implicit equation \(f_z \neq 0\) required
Example 3.11 — Implicit Differentiation

Problem: Let \(x^2 + y^2 + z^2 + 2xyz - 10 = 0\). Find \(\partial z/\partial x\) at \(P(1,1,2)\).

Show Solution ▾

Set \(f = x^2+y^2+z^2+2xyz-10\). Then \(f_x = 2x+2yz = 6\), \(f_z = 2z+2xy = 6\) at \(P\).

\(\partial z/\partial x = -f_x/f_z = -6/6 = -1\).

\[\boxed{\frac{\partial z}{\partial x}\bigg|_P = -1}\]

§3.6.3 — Tangent Planes and Normal Lines

In single-variable calculus, the tangent line approximates a curve. In multivariable calculus, the tangent plane approximates a surface. The gradient provides the normal direction.

Tangent Plane to f(x,y,z) = 0
\[f_x(P)(x-x_0) + f_y(P)(y-y_0) + f_z(P)(z-z_0) = 0\]
\(P = (x_0,y_0,z_0)\) — point on surface \(\nabla f(P)\) — normal to the plane
★ Special Case: z = g(x,y)

Rewrite as \(g(x,y) - z = 0\). Then \(f_z = -1\) and the tangent plane becomes:

\[z - z_0 = g_x(P)(x-x_0) + g_y(P)(y-y_0)\]

This is the multivariable analog of the tangent line formula \(y-y_0 = f'(x_0)(x-x_0)\).

Normal Line to a Surface
\[\mathbf{r}(t) = P + t\,\nabla f(P) = (x_0+f_x t,\;\;y_0+f_y t,\;\;z_0+f_z t)\]
Direction = \(\nabla f(P)\) Passes through \(P\), perpendicular to tangent plane
Figure 3.4 — The tangent plane touches the surface at P. The gradient ∇f(P) is the normal vector, perpendicular to the plane.
Example 3.12 — Tangent Plane and Normal Line

Problem: Find the tangent plane and normal line to \(x^2+y^2+z^2 = 9\) at \(P(1,2,2)\).

Show Solution ▾

\(f = x^2+y^2+z^2-9\). \(\nabla f = (2x,2y,2z)\). At \(P\): \(\nabla f = (2,4,4)\).

Tangent plane: \(2(x-1)+4(y-2)+4(z-2)=0 \implies x+2y+2z=9\).

Normal line: \(\mathbf{r}(t)=(1+2t,\;2+4t,\;2+4t)\).

\[\boxed{\text{Plane: } x+2y+2z=9 \qquad \text{Normal: } \mathbf{r}(t) = (1+2t,\,2+4t,\,2+4t)}\]
★ Tip

For a sphere centered at the origin, the normal line at any surface point always passes through \((0,0,0)\). Quick check: set \(t=-1/2\) to get \((0,0,0)\). ✓

Example 3.13 — Tangent Plane to z = g(x,y)

Problem: Find the tangent plane to \(z = x^2 + 2y^2\) at \((1,1,3)\).

Show Solution ▾

\(g_x = 2x = 2\), \(g_y = 4y = 4\) at \((1,1)\). Tangent plane: \(z - 3 = 2(x-1)+4(y-1)\).

\[\boxed{z = 2x + 4y - 3}\]

§3.7 — Conservative Fields, Curl, and Divergence

A vector field assigns a vector to every point in space: \(\mathbf{F}(x,y,z) = P\,\mathbf{i} + Q\,\mathbf{j} + R\,\mathbf{k}\). Examples include gravitational fields, electric fields, and fluid velocity fields.

Conservative Vector Field

A vector field \(\mathbf{F}\) is conservative if there exists a scalar field \(f\) (called the potential function) such that \(\mathbf{F} = \nabla f\). Think of it as the multivariable analog of finding an antiderivative.

Curl of a Vector Field
\[\text{curl}\;\mathbf{F} = \nabla \times \mathbf{F} = \begin{vmatrix} \mathbf{i} & \mathbf{j} & \mathbf{k} \\ \partial_x & \partial_y & \partial_z \\ P & Q & R \end{vmatrix}\]
\(\nabla \times \mathbf{F}\) — curl (outputs a vector) Measures rotation at a point

Expanding: \(\text{curl}\;\mathbf{F} = (R_y-Q_z)\,\mathbf{i} - (R_x-P_z)\,\mathbf{j} + (Q_x-P_y)\,\mathbf{k}\).

In 2D (\(\mathbf{F} = P\,\mathbf{i}+Q\,\mathbf{j}\)): \(\text{curl}\;\mathbf{F} = (Q_x - P_y)\,\mathbf{k}\).

Conservative Field Test
\[\mathbf{F} \text{ is conservative} \;\Longleftrightarrow\; \text{curl}\;\mathbf{F} = \mathbf{0}\]
In 2D: \(Q_x = P_y\) In 3D: \(R_y = Q_z\), \(P_z = R_x\), \(Q_x = P_y\)
Divergence of a Vector Field
\[\text{div}\;\mathbf{F} = \nabla \cdot \mathbf{F} = \frac{\partial P}{\partial x} + \frac{\partial Q}{\partial y} + \frac{\partial R}{\partial z}\]
\(\nabla \cdot \mathbf{F}\) — divergence (outputs a scalar) \(>0\): source, \(<0\): sink, \(=0\): incompressible
★ Key Identities

1. \(\nabla \times (\nabla f) = \mathbf{0}\) — curl of a gradient is always zero (conservative ⟹ irrotational).

2. \(\nabla \cdot (\nabla \times \mathbf{F}) = 0\) — divergence of a curl is always zero.

Both follow from Clairaut's theorem: mixed partials are equal.

Figure 3.5 — Divergence measures net outward flow (source/sink). Curl measures rotational tendency. div = 0 means incompressible; curl = 0 means irrotational (conservative).
Example 3.14 — Conservative Field Test (2D)

Problem: Is \(\mathbf{F} = (x^2+y^2)\,\mathbf{i} + 2xy\,\mathbf{j}\) conservative?

Show Solution ▾

\(P = x^2+y^2\), \(Q = 2xy\). Check: \(Q_x = 2y\), \(P_y = 2y\). Equal ✓.

\[\boxed{\text{Yes — } Q_x = P_y = 2y, \text{ so } \mathbf{F} \text{ is conservative.}}\]
Example 3.15 — Finding a Potential Function

Problem: Find \(f\) such that \(\nabla f = (4x^3y^4+2x)\,\mathbf{i} + (4y^3x^4+2y)\,\mathbf{j}\).

Show Solution ▾

Step 1: Integrate \(f_x = 4x^3y^4+2x\) w.r.t. \(x\): \(f = x^4y^4 + x^2 + g(y)\).

Step 2: Differentiate w.r.t. \(y\): \(f_y = 4x^4y^3 + g'(y)\). Set equal to \(Q = 4y^3x^4+2y\): \(g'(y) = 2y\), so \(g(y) = y^2+C\).

\[\boxed{f(x,y) = x^4y^4 + x^2 + y^2 + C}\]
★ Tip

Always verify: \(\nabla f = (4x^3y^4+2x,\;4x^4y^3+2y) = \mathbf{F}\). ✓ If starting with \(f_x\) leads to messy integrals, try starting with \(f_y\) instead.

Example 3.16 — Conservative Field Test (3D)

Problem: Show \(\mathbf{F} = 4xy^2z\,\mathbf{i} + 4x^2yz\,\mathbf{j} + 2x^2y^2\,\mathbf{k}\) is conservative.

Show Solution ▾

\(P = 4xy^2z\), \(Q = 4x^2yz\), \(R = 2x^2y^2\).

\(Q_x = 8xyz = P_y\) ✓. \(R_x = 4xy^2 = P_z\) ✓. \(R_y = 4x^2y = Q_z\) ✓.

All three conditions met: curl \(\mathbf{F} = \mathbf{0}\).

\[\boxed{\text{curl } \mathbf{F} = \mathbf{0} \implies \mathbf{F} \text{ is conservative.}}\]
Figure 3.6 — The grad–curl–div hierarchy. Gradient maps scalars to vectors; curl maps vectors to vectors; divergence maps vectors to scalars. Two key identities: curl of any gradient is zero; divergence of any curl is zero.
Example 3.17 — Computing Divergence

Problem: Find \(\text{div}\;\mathbf{F}\) for \(\mathbf{F} = \sin x\,\mathbf{i} + x\cos y\,\mathbf{j} + \sin z\,\mathbf{k}\) at \(P(\pi/2, 0, \pi)\).

Show Solution ▾

\(\text{div}\;\mathbf{F} = \cos x - x\sin y + \cos z\). At \(P\): \(0 - 0 + (-1) = -1\).

\[\boxed{\text{div}\;\mathbf{F}(P) = -1 \;\text{(sink)}}\]
Example 3.18 — Proving div(curl F) = 0

Problem: Show \(\nabla \cdot (\nabla \times \mathbf{F}) = 0\) for any twice-differentiable \(\mathbf{F} = (P,Q,R)\).

Show Solution ▾

\(\nabla \times \mathbf{F} = (R_y-Q_z,\;P_z-R_x,\;Q_x-P_y)\).

\(\nabla \cdot (\nabla \times \mathbf{F}) = R_{yx}-Q_{zx}+P_{zy}-R_{xy}+Q_{xz}-P_{yz}\).

By Clairaut's theorem: \(R_{yx}=R_{xy}\), \(Q_{zx}=Q_{xz}\), \(P_{zy}=P_{yz}\). Each pair cancels.

\[\boxed{\nabla \cdot (\nabla \times \mathbf{F}) = 0}\]
