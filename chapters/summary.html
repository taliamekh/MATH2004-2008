Which Tool Do I Use?

Start at the top. Follow the arrows based on what the problem gives you.

Integral Type Selector

What are you integrating over?

→ A curve (1D path)?

→ Scalar function \(f\) along curve? → ∫f ds (scalar line integral, §4.1)

→ Vector field \(\mathbf{F}\) along curve? → ∫F·dr (work integral, §4.2)

→ Is \(\mathbf{F}\) conservative (curl = 0)? → FTCLI: just evaluate the potential at endpoints (§4.3)

→ Closed curve in 2D? → Consider Green's theorem (§6.3)

→ Boundary of a surface in 3D? → Consider Stokes' theorem (§6.4)

→ A flat region (2D)?

→ Rectangle or simple region? → ∬f dA via Fubini (§5.2)

→ Circular region (\(x^2+y^2\) in bounds)? → Polar: \(dA = r\,dr\,d\theta\) (§5.4)

→ A curved surface (2D in 3D)?

→ Scalar function over surface? → ∬F dS with \(\sqrt{1+f_x^2+f_y^2}\) (§6.1)

→ Vector field through surface (flux)? → ∬F·dS = \(\iint(-Pf_x-Qf_y+R)\,dA\) (§6.2)

→ Closed surface? → Use Divergence theorem instead (§6.8)

→ A solid region (3D)?

→ Box / tetrahedron (flat faces)? → ∭f dV in rectangular (§6.5)

→ Cylinder (\(x^2+y^2\) in bounds)? → Cylindrical: \(dV = r\,dr\,d\theta\,dz\) (§6.6)

→ Sphere / cone (\(x^2+y^2+z^2\) in bounds)? → Spherical: \(dV = \rho^2\sin\varphi\,d\rho\,d\theta\,d\varphi\) (§6.6)

The Big Three Theorems — Which One?

Is the boundary a closed curve in the xy-plane? → Green's: \(\oint P\,dx+Q\,dy = \iint(Q_x-P_y)\,dA\)

Is the boundary a curve on a surface in 3D? → Stokes': \(\oint\mathbf{F}\cdot d\mathbf{r} = \iint(\nabla\times\mathbf{F})\cdot\mathbf{n}\,dS\)

Is the boundary a closed surface enclosing a solid? → Divergence: \(\iiint\text{div}\,\mathbf{F}\,dV = \iint\mathbf{F}\cdot\mathbf{n}\,dS\)

Each theorem says: integral over the interior = integral over the boundary. Pick the side that's easier to compute.

Optimization — Which Method?

Open domain (no constraints)? → Find critical points: \(f_x = 0\) and \(f_y = 0\) → Classify with SDT (§7.2)

Closed bounded region? → Interior CPs + boundary CPs + corners → Compare all \(f\)-values (§7.2)

Constrained by \(g(x,y) = c\)? → Lagrange: \(\nabla f = \lambda\nabla g\) + constraint (§7.3)

Chapter Cheat Sheets

Ch 1 — Vectors

Key ideas: Vectors as magnitude + direction objects in ℝ² and ℝ³. Dot product measures alignment (angle, projection, work). Cross product produces perpendicular vectors (normals, area, volume). Everything in later chapters builds on these operations.

When to use what:

Angle between vectors → dot product · Perpendicularity test → \(\vec{v}\cdot\vec{w} = 0\) · Projection / component along a direction → proj formula · Normal to a plane → cross product of two in-plane vectors · Area of triangle/parallelogram → \(|\vec{u}\times\vec{v}|\) · Volume of parallelepiped → \(|\vec{u}\cdot(\vec{v}\times\vec{w})|\) · Coplanarity test → triple product = 0

⚠ Traps: Cross product is NOT commutative (\(\vec{v}\times\vec{w} = -\vec{w}\times\vec{v}\)). Dot product gives a scalar, cross product gives a vector — don't mix them up. Projection formula has \(|\vec{v}|^2\) in the denominator for the vector version but just \(|\vec{v}|\) for the scalar version.

Ch 2 — Lines, Planes & Parametric Curves

Key ideas: Lines and planes in 3D via parametric and normal equations. Parametric curves separate the "shape" from the "traversal." Polar coordinates for curves with circular symmetry. Arc length and area from parametric/polar formulas.

When to use what:

Line through two points → \(\mathbf{r}(t) = P + t\vec{v}\) · Plane from 3 points → normal = \(\vec{PQ}\times\vec{PR}\) · Tangent slope of parametric curve → \(dy/dx = (dy/dt)/(dx/dt)\) · Area under parametric curve → \(\int y\,x'\,dt\) · Arc length → \(\int\sqrt{(x')^2+(y')^2}\,dt\) · Polar area → \(\frac{1}{2}\int r^2\,d\theta\) · Conic type → discriminant \(B^2-4AC\)

⚠ Traps: Second derivative is \(\frac{d/dt(dy/dx)}{dx/dt}\), NOT \(y''/x''\). Polar area between curves is \(\frac{1}{2}\int(r_{\text{out}}^2 - r_{\text{in}}^2)\,d\theta\), not the difference of two separate areas. Skew lines are not parallel AND don't intersect — check both.

Ch 3 — Partial Derivatives & Vector Fields

Key ideas: Partial derivatives extend single-variable calculus to functions of several variables. The gradient ∇f points in the direction of steepest ascent. Chain rule for multivariable compositions. Tangent planes via gradient. Conservative fields have curl = 0 and a potential function.

When to use what:

Rate of change in specific direction → \(D_{\mathbf{u}}f = \nabla f \cdot \mathbf{u}\) · Steepest ascent → direction of \(\nabla f\) with magnitude \(|\nabla f|\) · Related rates → chain rule · Implicit ∂z/∂x → \(-f_x/f_z\) · Tangent plane → \(\nabla f\) is the normal · Is F conservative? → check \(\text{curl}\,\mathbf{F} = \mathbf{0}\) · Potential function → integrate component by component

⚠ Traps: Limit exists only if ALL paths give the same value — showing two paths agree proves nothing. Clairaut (fxy = fyx) requires continuity of mixed partials. Conservative test: in 2D check Qx = Py; in 3D check full curl = 0. div(curl F) = 0 always — don't confuse with the test.

Ch 4 — Line Integrals

Key ideas: Two types: scalar (∫f ds — weighted arc length) and vector (∫F·dr — work/circulation). The FTCLI shortcuts conservative field integrals to endpoint evaluation. Path independence ↔ conservative ↔ curl = 0 ↔ ∮F·dr = 0.

When to use what:

Mass of a wire → ∫f ds (scalar line integral) · Work done by force → ∫F·dr (vector line integral) · F is conservative? → FTCLI: find potential f, answer is f(end) − f(start) · Closed loop with conservative F → answer is 0 · Multiple paths → check conservative first before parametrizing

⚠ Traps: Scalar line integrals (ds) are orientation-independent. Vector line integrals (F·dr) reverse sign when you reverse direction. The FTCLI only works for conservative fields — always verify curl = 0 first. Piecewise curves: parametrize each segment separately and add.

Ch 5 — Double Integrals & Surfaces

Key ideas: Double integrals compute area, volume, and mass over 2D regions. Fubini converts to iterated integrals (two orderings — pick the easier one). Polar coordinates for circular regions. Quadric surface catalog for 3D visualization. Parametric surfaces r(u,v).

When to use what:

Integral over rectangle → Fubini, either order · Integral over general region → sketch it, pick vertical or horizontal slices · \(x^2+y^2\) in bounds or integrand → switch to polar (\(dA = r\,dr\,d\theta\)) · Non-standard coordinates → Jacobian change of variables · Identify a 3D surface → count squared terms + check signs (quadric catalog)

⚠ Traps: Don't forget the r in polar \(dA = r\,dr\,d\theta\). When reversing integration order, re-derive the limits from scratch — don't just swap. x-simple (vertical slices) → integrate dy first; y-simple (horizontal) → dx first.

Ch 6 — The Three Big Theorems

Key ideas: Surface integrals extend double integrals to curved surfaces. Flux measures net flow through a surface. Green's/Stokes'/Divergence connect boundary integrals to interior integrals at different dimensions. Triple integrals for 3D solids with cylindrical and spherical coordinate transforms.

When to use what:

Surface integral (scalar) → \(\iint F\sqrt{1+f_x^2+f_y^2}\,dA\) or parametric \(\iint F|\mathbf{r}_u\times\mathbf{r}_v|\,dA\) · Flux through surface → \(\iint(-Pf_x - Qf_y + R)\,dA\) · Closed curve in 2D → Green's: check if \(Q_x - P_y\) simplifies · Surface with boundary → Stokes': swap hard surfaces for easy ones · Closed surface → Divergence: compute div F and triple-integrate · Triple integral over cylinder → cylindrical (J = r) · Over sphere/cone → spherical (J = ρ² sin φ)

⚠ Traps: Forgetting the Jacobian: r for cylindrical, ρ² sin φ for spherical. Green's requires CCW orientation; inner boundaries of holed regions go CW. Stokes' orientation: right-hand rule (curl fingers along 𝒞, thumb = n). Divergence theorem only works for closed surfaces — open surfaces need Stokes'. φ in spherical is measured from the z-axis (colatitude), NOT from the equator.

Ch 7 — Max/Min & Lagrange

Key ideas: Critical points where ∇f = 0 are candidates for local extrema. The Second Derivative Test classifies them via the discriminant D. Closed bounded regions guarantee global extrema exist — check interior CPs + boundary + corners. Lagrange multipliers handle constraints by requiring ∇f ∥ ∇g.

When to use what:

Unconstrained max/min → find CPs (\(f_x = f_y = 0\)) → classify with SDT (\(D = f_{xx}f_{yy} - f_{xy}^2\)) · D > 0 + fxx > 0 → min · D > 0 + fxx < 0 → max · D < 0 → saddle · D = 0 → inconclusive · Closed region → interior + boundary + corners, compare all values · Constraint g = c → Lagrange: ∇f = λ∇g + g = c

⚠ Traps: D = 0 is inconclusive — not "saddle" or "flat." On a closed region, global extrema might be at boundary points that are NOT critical points of f. In Lagrange, don't forget to solve for λ AND plug back into the constraint. Substitution and Lagrange should give the same answer — use substitution as a check when possible.
