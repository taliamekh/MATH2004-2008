A1 — Equation of a Plane (Perpendicular + Two Points)
🧠 What is this asking?

Find the equation of a plane Π₁ that passes through two given points P(0,1,2) and Q(−1,0,3) AND is perpendicular to another plane Π₂: 2x − 4y + 6z = 2.

📋 Formulas Needed
  • Plane equation: \(ax + by + cz = d\), where \(\mathbf{n} = (a, b, c)\) is the normal vector
  • Normal from two vectors: \(\mathbf{n} = \mathbf{v}_1 \times \mathbf{v}_2\) (cross product)
  • Perpendicular planes: their normals are perpendicular → \(\mathbf{n}_1 \cdot \mathbf{n}_2 = 0\)
  • Cross product: \(\mathbf{u} \times \mathbf{v} = \begin{vmatrix} \mathbf{i} & \mathbf{j} & \mathbf{k} \\ u_1 & u_2 & u_3 \\ v_1 & v_2 & v_3 \end{vmatrix}\)

📋 Exam Sheet: Cross product formula, Plane equation

🔧 Setup — How to Recognize This

You need a plane through 2 points AND perpendicular to another plane. The key insight: the plane Π₁ must contain the direction vector \(\vec{PQ}\), and its normal must be perpendicular to the normal of Π₂. So Π₁'s normal is perpendicular to BOTH \(\vec{PQ}\) and \(\mathbf{n}_2\). That means: \(\mathbf{n}_1 = \vec{PQ} \times \mathbf{n}_2\).

📝 Full Solution

Step 1: Find \(\vec{PQ}\).

\[\vec{PQ} = Q - P = (-1 - 0,\; 0 - 1,\; 3 - 2) = (-1, -1, 1)\]

Step 2: Identify \(\mathbf{n}_2\) from Π₂.

The plane \(2x - 4y + 6z = 2\) has normal vector \(\mathbf{n}_2 = (2, -4, 6)\).

Step 3: Compute \(\mathbf{n}_1 = \vec{PQ} \times \mathbf{n}_2\).

Why? Because \(\mathbf{n}_1\) must be perpendicular to both \(\vec{PQ}\) (so the plane contains the line PQ) and \(\mathbf{n}_2\) (so the planes are perpendicular).

\[\mathbf{n}_1 = \begin{vmatrix} \mathbf{i} & \mathbf{j} & \mathbf{k} \\ -1 & -1 & 1 \\ 2 & -4 & 6 \end{vmatrix}\]

\[\mathbf{i}[(-1)(6) - (1)(-4)] - \mathbf{j}[(-1)(6) - (1)(2)] + \mathbf{k}[(-1)(-4) - (-1)(2)]\]

\[= \mathbf{i}[-6 + 4] - \mathbf{j}[-6 - 2] + \mathbf{k}[4 + 2]\]

\[= (-2, 8, 6)\]

Simplify by dividing by −2: \(\mathbf{n}_1 = (1, -4, -3)\) or equivalently \((-1, 4, 3)\).

Step 4: Write the plane equation using point P(0,1,2).

\[-1(x - 0) + 4(y - 1) + 3(z - 2) = 0\]

\[-x + 4y - 4 + 3z - 6 = 0\]

\[-x + 4y + 3z = 10\]

✅ Answer: (c) \(-x + 4y + 3z = 10\)
⚡ Quick Guide — Plane Through 2 Points, Perpendicular to Another Plane
  1. Compute direction vector \(\vec{PQ} = Q - P\)
  2. Read off normal \(\mathbf{n}_2\) from the given plane equation
  3. Cross product: \(\mathbf{n}_1 = \vec{PQ} \times \mathbf{n}_2\)
  4. Simplify \(\mathbf{n}_1\) if possible (divide out common factors)
  5. Plug one of the given points into \(a(x-x_0)+b(y-y_0)+c(z-z_0)=0\)
  6. Expand and simplify to standard form \(ax+by+cz=d\)
  7. Verify: plug BOTH points into your equation to confirm they satisfy it
A2 — Area of a Polar Region
🧠 What is this asking?

Find the area of the region \(R = \{(r, \theta) : 0 \le r \le 1,\; 0 \le \theta \le \pi/5\}\). This is a circular sector — a "pie slice" of a unit circle.

📋 Formulas Needed
  • Polar area: \(A = \int_\alpha^\beta \int_0^{r(\theta)} r\,dr\,d\theta\) — or for a full sector of a circle: \(A = \frac{1}{2}\int_\alpha^\beta r^2\,d\theta\)
  • Jacobian: In polar coordinates, \(dA = r\,dr\,d\theta\)
📝 Full Solution

\[A = \int_0^{\pi/5}\int_0^1 r\,dr\,d\theta = \int_0^{\pi/5}\left[\frac{r^2}{2}\right]_0^1 d\theta = \int_0^{\pi/5}\frac{1}{2}\,d\theta = \frac{1}{2}\cdot\frac{\pi}{5} = \frac{\pi}{10}\]

✅ Answer: (a) \(\dfrac{\pi}{10}\)
⚡ Quick Guide — Area of Polar Sector
  1. Set up the double integral \(\int_\alpha^\beta \int_0^{R} r\,dr\,d\theta\)
  2. Integrate the inner integral in \(r\): gives \(R^2/2\)
  3. Integrate the outer integral in \(\theta\): gives \(R^2(\beta - \alpha)/2\)
  4. For a unit circle sector: area = (angle)/2
A3 — Area of a Conic Section (Completing the Square)
🧠 What is this asking?

Given \(9x^2 + 4y^2 - 18x - 8y - 23 = 0\), identify this as an ellipse and find its area.

📋 Formulas Needed
  • Ellipse standard form: \(\frac{(x-h)^2}{a^2} + \frac{(y-k)^2}{b^2} = 1\)
  • Ellipse area: \(A = \pi a b\)
  • Completing the square: \(ax^2 + bx = a\left[(x + \frac{b}{2a})^2 - \frac{b^2}{4a^2}\right]\)
📝 Full Solution

Step 1: Group x and y terms.

\[9x^2 - 18x + 4y^2 - 8y = 23\]

Step 2: Factor out the coefficients of squared terms.

\[9(x^2 - 2x) + 4(y^2 - 2y) = 23\]

Step 3: Complete the square for each variable.

For \(x^2 - 2x\): take half of −2 (which is −1), square it (gives 1). So \(x^2 - 2x + 1 = (x-1)^2\). Add \(9 \times 1 = 9\) to RHS.

For \(y^2 - 2y\): take half of −2 (which is −1), square it (gives 1). So \(y^2 - 2y + 1 = (y-1)^2\). Add \(4 \times 1 = 4\) to RHS.

\[9(x-1)^2 + 4(y-1)^2 = 23 + 9 + 4 = 36\]

Step 4: Divide both sides by 36.

\[\frac{(x-1)^2}{4} + \frac{(y-1)^2}{9} = 1\]

Step 5: Read off a and b.

\(a^2 = 4 \Rightarrow a = 2\), \(b^2 = 9 \Rightarrow b = 3\). Center at (1, 1).

Step 6: Compute area.

\[A = \pi a b = \pi(2)(3) = 6\pi\]

✅ Answer: (b) \(6\pi\)
⚡ Quick Guide — Area of Conic Section
  1. Group x-terms and y-terms; move constant to RHS
  2. Factor out coefficient of x² and y² from each group
  3. Complete the square inside each group (add the same to BOTH sides, multiplied by the coefficient)
  4. Divide everything by the RHS constant to get = 1
  5. Read off a² and b² from denominators
  6. Area = πab

A5 — Chain Rule for Partial Derivatives
🧠 What is this asking?

Given \(f(x,y)\) where \(x = u(s,t)\), \(y = v(s,t)\), with \(f_x = 2\), \(f_y = 3\), \(x_s = -1\), \(y_s = 1\) at \((s,t) = (1,0)\), find \(\partial f/\partial s\).

📋 Formulas Needed
  • Chain rule (two parameters): \(\frac{\partial f}{\partial s} = \frac{\partial f}{\partial x}\frac{\partial x}{\partial s} + \frac{\partial f}{\partial y}\frac{\partial y}{\partial s}\)
📝 Full Solution

\[\frac{\partial f}{\partial s} = f_x \cdot x_s + f_y \cdot y_s = (2)(-1) + (3)(1) = -2 + 3 = 1\]

✅ Answer: (a) 1
⚡ Quick Guide — Chain Rule
  1. Draw the dependency tree: f depends on x,y; x,y depend on s,t
  2. Apply: \(\partial f/\partial s = f_x \cdot x_s + f_y \cdot y_s\)
  3. Plug in the given values at the specified point
  4. Compute — just dot product of (f_x, f_y) with (x_s, y_s)
A6 — Conservative Vector Field Test
🧠 What is this asking?

Which of the given vector fields has curl = 0 (i.e., is conservative)?

📋 Formulas Needed
  • 2D test: \(\mathbf{F} = (P, Q)\) is conservative iff \(\frac{\partial P}{\partial y} = \frac{\partial Q}{\partial x}\)
  • 3D test: \(\nabla \times \mathbf{F} = \mathbf{0}\), i.e., \(R_y = Q_z\), \(P_z = R_x\), \(Q_x = P_y\)
📝 Full Solution

Check (b): \(\mathbf{F}(x,y,z) = (ze^{xz}\sin y,\; e^{xz}\cos y,\; xe^{xz}\sin y)\).

Let \(P = ze^{xz}\sin y\), \(Q = e^{xz}\cos y\), \(R = xe^{xz}\sin y\).

Test \(P_y = Q_x\):

\(P_y = ze^{xz}\cos y\). \(Q_x = ze^{xz}\cos y\). ✓ Equal.

Test \(P_z = R_x\):

\(P_z = e^{xz}\sin y + xze^{xz}\sin y = (1+xz)e^{xz}\sin y\).

\(R_x = e^{xz}\sin y + x \cdot ze^{xz}\sin y = (1+xz)e^{xz}\sin y\). ✓ Equal.

Test \(Q_z = R_y\):

\(Q_z = xe^{xz}\cos y\). \(R_y = xe^{xz}\cos y\). ✓ Equal.

All three conditions satisfied → \(\nabla \times \mathbf{F} = \mathbf{0}\) → conservative.

✅ Answer: (b) \(\mathbf{F}(x,y,z) = (ze^{xz}\sin y,\; e^{xz}\cos y,\; xe^{xz}\sin y)\)
⚡ Quick Guide — Conservative Field Test
  1. For 2D \((P,Q)\): check \(P_y = Q_x\). Done.
  2. For 3D \((P,Q,R)\): check all three cross-partials: \(P_y=Q_x\), \(P_z=R_x\), \(Q_z=R_y\)
  3. If ALL match → conservative. If ANY fails → not conservative.
  4. Tip: start with the easiest partial to compute — if it fails, move to next option.
A7 — Mass of a Cable (Line Integral of Scalar Field)
🧠 What is this asking?

A cable goes from (0,0) to (L,L) in a straight line, with mass density \(\delta(x,y) = xy\). Find total mass.

📋 Formulas Needed
  • Mass of a wire: \(m = \int_C \delta(x,y)\,ds\)
  • Scalar line integral: \(\int_C f\,ds = \int_a^b f(\mathbf{r}(t))|\mathbf{r}'(t)|\,dt\)
📝 Full Solution

Step 1: Parametrize the cable. Straight line from (0,0) to (L,L): \(\mathbf{r}(t) = (t, t)\), \(0 \le t \le L\).

Step 2: Compute \(|\mathbf{r}'(t)|\). \(\mathbf{r}'(t) = (1, 1)\), so \(|\mathbf{r}'(t)| = \sqrt{1^2 + 1^2} = \sqrt{2}\).

Step 3: Set up and evaluate.

\[m = \int_0^L (t)(t)\sqrt{2}\,dt = \sqrt{2}\int_0^L t^2\,dt = \sqrt{2}\cdot\frac{t^3}{3}\bigg|_0^L = \frac{\sqrt{2}\,L^3}{3}\]

✅ Answer: (c) \(\dfrac{\sqrt{2}\,L^3}{3}\)
⚡ Quick Guide — Mass of a Wire/Cable
  1. Parametrize the curve C: \(\mathbf{r}(t)\), find the range of t
  2. Compute \(|\mathbf{r}'(t)|\) — the speed
  3. Substitute into \(m = \int \delta(\mathbf{r}(t))|\mathbf{r}'(t)|\,dt\)
  4. Evaluate the integral
A8 — Reversing Order of Integration
🧠 What is this asking?

The integral \(\int_0^1 \int_x^1 e^{y^2}\,dy\,dx\) has the inner integral in y from x to 1. We need to reverse the order to dx dy.

📋 Formulas Needed
  • Reversing order: Sketch the region, re-describe with the other variable as outer integral.
📝 Full Solution

Step 1: Identify the region. Original: \(0 \le x \le 1\), \(x \le y \le 1\). This is the triangle above the line y = x, below y = 1, between x = 0 and x = 1.

Step 2: Re-describe with y as outer. For a fixed y (ranging 0 to 1), x goes from 0 to y (since x ≤ y in the region).

Step 3: Write reversed integral.

\[\int_0^1\int_0^y e^{y^2}\,dx\,dy\]

The integrand stays the same, only the limits change.

✅ Answer: (d) \(\displaystyle\int_0^1\int_0^y e^{y^2}\,dx\,dy\)
⚡ Quick Guide — Reversing Integration Order
  1. From the original limits, identify the region R in the xy-plane
  2. Sketch R — plot the boundary curves
  3. Re-describe R with the other variable as outer: find new inner limits as functions of the new outer variable
  4. Write the new integral with same integrand, new limits
  5. Key insight: \(e^{y^2}\) has no elementary antiderivative in y, which is WHY you reverse — to integrate in x first (trivially), leaving a doable y-integral
A9 — Volume Inside Paraboloid Cut by Plane
🧠 What is this asking?

Find the volume of the solid region inside the paraboloid \(z = x^2 + y^2\) that is below the plane \(z = 4\).

📋 Formulas Needed
  • Volume via double integral: \(V = \iint_R (z_{\text{top}} - z_{\text{bottom}})\,dA\)
  • Polar conversion: \(x^2+y^2 = r^2\), \(dA = r\,dr\,d\theta\)
📝 Full Solution

Step 1: The solid is bounded below by \(z = r^2\) and above by \(z = 4\). The shadow in xy-plane: where they meet: \(r^2 = 4 \Rightarrow r = 2\).

Step 2: Convert to polar.

\[V = \int_0^{2\pi}\int_0^2 (4 - r^2)\,r\,dr\,d\theta\]

Step 3: Evaluate inner integral.

\[\int_0^2 (4r - r^3)\,dr = \left[2r^2 - \frac{r^4}{4}\right]_0^2 = 8 - 4 = 4\]

Step 4: Outer integral.

\[V = \int_0^{2\pi} 4\,d\theta = 8\pi\]

✅ Answer: (a) \(8\pi\)
⚡ Quick Guide — Volume Under a Cap
  1. Identify top surface (\(z_{\text{top}}\)) and bottom surface (\(z_{\text{bot}}\))
  2. Find the shadow region: set \(z_{\text{top}} = z_{\text{bot}}\) to get boundary
  3. Set up \(V = \iint (z_{\text{top}} - z_{\text{bot}})\,dA\)
  4. Convert to polar if circular symmetry: \(dA = r\,dr\,d\theta\)
  5. Evaluate inner, then outer integral
A10 — Lagrange Multipliers — Distance to Hyperbola
🧠 What is this asking?

Find the smallest distance from (0,0) to the curve \(xy = 1\) with \(x > 0, y > 0\).

📋 Formulas Needed
  • Distance squared: \(D^2 = x^2 + y^2\) (minimize this — avoids square root)
  • Lagrange: \(\nabla f = \lambda \nabla g\), with constraint \(g(x,y) = xy - 1 = 0\)
📝 Full Solution

Minimize \(f(x,y) = x^2 + y^2\) subject to \(g(x,y) = xy - 1 = 0\).

\(\nabla f = (2x, 2y)\), \(\nabla g = (y, x)\).

Lagrange: \(2x = \lambda y\) and \(2y = \lambda x\).

From eq 1: \(\lambda = 2x/y\). From eq 2: \(\lambda = 2y/x\).

Setting equal: \(2x/y = 2y/x \Rightarrow x^2 = y^2 \Rightarrow x = y\) (since both positive).

Constraint: \(x \cdot x = 1 \Rightarrow x^2 = 1 \Rightarrow x = 1\). So \(x = y = 1\).

Distance: \(\sqrt{1^2 + 1^2} = \sqrt{2}\).

But wait — the answer choices are 4, 8, 1, 2. The distance squared is 2, but \(\sqrt{2}\) isn't listed. The answer key says (c) 1.

Re-reading: "smallest distance" — perhaps they want the minimum of \(x^2+y^2\)? No, that gives 2.

Actually looking at the choices: if they're asking for the minimum of \(x^2+y^2\) rather than the distance itself, the answer would be 2 (choice d). But the key says 1. Perhaps there's a different interpretation — maybe the minimum of \(x+y\)? At x=y=1: x+y=2. Or perhaps the minimum of \(\sqrt{x^2+y^2}\) rounded? \(\sqrt{2} \approx 1.41\)...

Given the official answer is (c) 1: at the critical point \((1,1)\), the minimum value of f(x,y) = xy = 1 on the constraint, while the distance = \(\sqrt{2}\). The answer key likely interprets "smallest distance" as the minimum distance squared divided by 2, or there may be a different intended reading. For the exam: pick (c) 1.

✅ Answer: (c) 1 [per official key; the Euclidean distance is \(\sqrt{2}\)]
A11 — Line Integral Along Parabolic Arc
🧠 What is this asking?

Evaluate \(\int_C (x^2-y)\,dx + (y^2+x)\,dy\) where C is the parabolic arc from (0,0) to (1,1).

📋 Formulas Needed
  • Line integral of F·dr: Parametrize C, then \(\int_C P\,dx + Q\,dy = \int_a^b [P\,x'(t) + Q\,y'(t)]\,dt\)
  • Parabolic arc (0,0) to (1,1): \(y = x^2\), so parametrize as \(x = t, y = t^2, 0 \le t \le 1\)
📝 Full Solution

Parametrize: \(x = t\), \(y = t^2\), \(dx = dt\), \(dy = 2t\,dt\), \(0 \le t \le 1\).

\[I = \int_0^1 [(t^2 - t^2)(1) + (t^4 + t)(2t)]\,dt = \int_0^1 [0 + 2t^5 + 2t^2]\,dt\]

\[= \int_0^1 (2t^5 + 2t^2)\,dt = \left[\frac{t^6}{3} + \frac{2t^3}{3}\right]_0^1 = \frac{1}{3} + \frac{2}{3} = 1\]

✅ Answer: (d) 1
⚡ Quick Guide — Line Integral Along a Curve
  1. Parametrize the curve: for \(y=x^2\), use \(x=t, y=t^2\)
  2. Find dx and dy in terms of dt
  3. Substitute everything into \(\int P\,dx + Q\,dy\)
  4. Simplify and integrate in t
A12 — Flux via Divergence Theorem (Box)
🧠 What is this asking?

Given \(\text{div}\,\mathbf{F} = 8xyz\), find the net flux through the box \(0 \le x \le a, 0 \le y \le b, 0 \le z \le c\).

📋 Formulas Needed
  • Divergence Theorem: \(\iint_S \mathbf{F}\cdot\mathbf{n}\,dS = \iiint_T \text{div}\,\mathbf{F}\,dV\)
📝 Full Solution

\[\text{Flux} = \int_0^a\int_0^b\int_0^c 8xyz\,dz\,dy\,dx\]

\[= 8\int_0^a x\,dx \cdot \int_0^b y\,dy \cdot \int_0^c z\,dz \quad\text{(separable)}\]

\[= 8 \cdot \frac{a^2}{2} \cdot \frac{b^2}{2} \cdot \frac{c^2}{2} = 8 \cdot \frac{a^2b^2c^2}{8} = a^2b^2c^2\]

✅ Answer: (b) \(a^2b^2c^2\)
⚡ Quick Guide — Flux via Divergence Theorem
  1. Flux = \(\iiint \text{div}\,\mathbf{F}\,dV\) (Divergence Theorem)
  2. Set up limits from the surface description
  3. If the integrand is separable (\(f(x)g(y)h(z)\)), split into three 1D integrals
  4. Evaluate each independently and multiply

B1 [8 marks] — Green's Theorem on Ellipse
🧠 What is this asking?

Evaluate \(I = \oint_C -y\,dx + x\,dy\) over the perimeter of the ellipse \(4x^2 + 9y^2 = 36\), counterclockwise.

📋 Formulas Needed
  • Green's Theorem: \(\oint_C P\,dx + Q\,dy = \iint_R \left(\frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y}\right)dA\)
  • Ellipse area: \(\frac{x^2}{a^2}+\frac{y^2}{b^2}=1\) has area \(\pi ab\)
📝 Full Solution

Step 1: Identify P and Q. \(P = -y\), \(Q = x\).

Step 2: Compute the 2D curl.

\[\frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y} = 1 - (-1) = 2\]

Step 3: Apply Green's Theorem.

\[I = \iint_R 2\,dA = 2\cdot\text{Area}(R)\]

Step 4: Find the area of the ellipse. Write \(4x^2 + 9y^2 = 36\) in standard form: \(\frac{x^2}{9} + \frac{y^2}{4} = 1\). So \(a = 3\), \(b = 2\).

\[\text{Area} = \pi(3)(2) = 6\pi\]

Step 5: Final answer.

\[I = 2 \cdot 6\pi = 12\pi\]

✅ Answer: \(I = 12\pi\)
⚡ Quick Guide — Green's Theorem on Closed Curves
  1. Identify P and Q from \(\oint P\,dx + Q\,dy\)
  2. Compute \(Q_x - P_y\) — the 2D curl
  3. If it's a constant → integral = constant × area of the enclosed region
  4. For ellipse \(\frac{x^2}{a^2}+\frac{y^2}{b^2}=1\): area = \(\pi ab\)
  5. Special case: \(\oint -y\,dx + x\,dy = 2\cdot\text{Area}\) (this is the area formula!)
B2 [7 marks] — Surface Integral of y over Hemisphere
🧠 What is this asking?

Evaluate \(I = \iint_S y\,dS\) where S is the upper hemisphere \(x^2+y^2+z^2=R^2\), \(z>0\).

📋 Formulas Needed
  • Surface integral (graph): \(\iint_S f\,dS = \iint_D f(x,y,g(x,y))\sqrt{1+g_x^2+g_y^2}\,dA\)
  • Symmetry argument: if the integrand is odd in one variable and the domain is symmetric about that variable's axis, the integral = 0.
📝 Full Solution

Method: Symmetry. The hemisphere S is symmetric about the xz-plane (i.e., symmetric under \(y \to -y\)). The function \(f(x,y,z) = y\) is odd in y: \(f(x,-y,z) = -y = -f(x,y,z)\).

The surface element \(dS\) is symmetric. Therefore, the contribution from \(y > 0\) exactly cancels the contribution from \(y < 0\).

\[I = \iint_{S^+} y\,dS + \iint_{S^-} y\,dS = I^+ + I^-\]

By the detailed computation (solving for y as a function of x and z on each half):

On \(S^+\) (y > 0): \(y = \sqrt{R^2-x^2-z^2}\). The projection onto the xz-plane is the half-disk \(R^+ = \{(r,\theta): 0 \le r \le R, 0 \le \theta \le \pi\}\).

\[I^+ = \iint_{R^+}\sqrt{R^2-x^2-z^2}\cdot\frac{R}{\sqrt{R^2-x^2-z^2}}\,dA = R\iint_{R^+}dA = R\cdot\frac{\pi R^2}{2} = \frac{\pi R^3}{2}\]

On \(S^-\) (y < 0): \(y = -\sqrt{R^2-x^2-z^2}\), same calculation with a minus sign gives \(I^- = -\frac{\pi R^3}{2}\).

\[I = \frac{\pi R^3}{2} + \left(-\frac{\pi R^3}{2}\right) = 0\]

✅ Answer: \(I = 0\)
⚡ Quick Guide — Surface Integrals with Symmetry
  1. Before computing: check if the integrand is odd in one variable AND the surface is symmetric about that variable's plane
  2. Odd function × symmetric domain = 0 (integral cancels)
  3. If no symmetry shortcut: project surface onto a convenient coordinate plane
  4. Use \(dS = \frac{R}{\sqrt{R^2-x^2-z^2}}dA\) for a sphere projected onto the xz-plane (or similar)
B3 [7 marks] — Stokes' Theorem (Curl Flux Through Plane in Cylinder)
🧠 What is this asking?

Let \(\mathbf{F} = (-y^2, x, z^2)\). S is the part of plane \(x+z=2\) inside the cylinder \(x^2+y^2=1\). Find \(\iint_S (\nabla\times\mathbf{F})\cdot\mathbf{n}\,dS\).

📋 Formulas Needed
  • Curl: \(\nabla\times\mathbf{F} = \det\begin{pmatrix}\mathbf{i}&\mathbf{j}&\mathbf{k}\\\partial_x&\partial_y&\partial_z\\P&Q&R\end{pmatrix}\)
  • Flux through graph z=g(x,y): \(\iint (-Pf_x - Qf_y + R)\,dA\) where \(\mathbf{G} = \nabla\times\mathbf{F} = (P,Q,R)\)
  • \(dS = \sqrt{1+f_x^2+f_y^2}\,dA\), and \(\mathbf{n}\,dS = (-f_x,-f_y,1)\,dA\) for upward normal
📝 Full Solution

Step 1: Compute curl F.

\[\nabla\times\mathbf{F} = \left(\frac{\partial z^2}{\partial y}-\frac{\partial x}{\partial z},\;\frac{\partial(-y^2)}{\partial z}-\frac{\partial z^2}{\partial x},\;\frac{\partial x}{\partial x}-\frac{\partial(-y^2)}{\partial y}\right)\]

\[= (0-0,\; 0-0,\; 1+2y) = (0, 0, 1+2y)\]

Step 2: Find the normal and dS. The plane is \(x+z=2\), i.e., \(z = 2-x\). So \(f_x = -1\), \(f_y = 0\). Normal vector: \(\pm(1,0,1)\). The outward (upward-pointing) unit normal: \(\mathbf{n} = \frac{(1,0,1)}{\sqrt{2}}\).

\(dS = \sqrt{1+1+0}\,dA = \sqrt{2}\,dA\).

Step 3: Compute \((\nabla\times\mathbf{F})\cdot\mathbf{n}\,dS\).

\[(0,0,1+2y)\cdot\frac{(1,0,1)}{\sqrt{2}}\cdot\sqrt{2}\,dA = (1+2y)\,dA\]

(The \(\sqrt{2}\) cancels!)

Step 4: Integrate over the projection R (disk \(x^2+y^2 \le 1\)).

\[I = \iint_R (1+2y)\,dA\]

In polar: \(y = r\sin\theta\).

\[I = \int_0^{2\pi}\int_0^1 (1+2r\sin\theta)\,r\,dr\,d\theta\]

\[= \int_0^{2\pi}\int_0^1 (r + 2r^2\sin\theta)\,dr\,d\theta\]

\[= \int_0^{2\pi}\left[\frac{1}{2} + \frac{2}{3}\sin\theta\right]d\theta = \frac{1}{2}(2\pi) + \frac{2}{3}\underbrace{\int_0^{2\pi}\sin\theta\,d\theta}_{=0} = \pi\]

✅ Answer: \(I = \pi\)
⚡ Quick Guide — Curl Flux Through a Surface
  1. Compute \(\nabla\times\mathbf{F}\)
  2. Write the surface as \(z = g(x,y)\), compute \(g_x, g_y\)
  3. Use the shortcut: \((\nabla\times\mathbf{F})\cdot\mathbf{n}\,dS = (-(\nabla\times\mathbf{F})_1 g_x - (\nabla\times\mathbf{F})_2 g_y + (\nabla\times\mathbf{F})_3)\,dA\)
  4. Project to the xy-plane and integrate over the shadow region
  5. Tip: \(\int_0^{2\pi}\sin\theta\,d\theta = \int_0^{2\pi}\cos\theta\,d\theta = 0\) — these terms always vanish
B4 [7 marks] — Flux via Divergence Theorem (Hemisphere)
🧠 What is this asking?

Find \(\iint_S \mathbf{F}\cdot\mathbf{n}\,dS\) for \(\mathbf{F} = (x, z^2x, y^3x^2)\) over the closed hemisphere \(x^2+y^2+z^2=4\), \(z \ge 0\).

📋 Formulas Needed
  • Divergence Theorem: \(\iint_S \mathbf{F}\cdot\mathbf{n}\,dS = \iiint_T \text{div}\,\mathbf{F}\,dV\)
  • Volume of hemisphere: \(\frac{2}{3}\pi R^3\)
📝 Full Solution

Step 1: Compute div F.

\[\text{div}\,\mathbf{F} = \frac{\partial}{\partial x}(x) + \frac{\partial}{\partial y}(z^2x) + \frac{\partial}{\partial z}(y^3x^2) = 1 + 0 + 0 = 1\]

Step 2: Apply Divergence Theorem. The surface S is closed (hemisphere + disk base). The solid T is the upper half-ball of radius 2.

\[\iint_S \mathbf{F}\cdot\mathbf{n}\,dS = \iiint_T 1\,dV = \text{Volume}(T) = \frac{2}{3}\pi(2)^3 = \frac{16\pi}{3}\]

✅ Answer: \(\dfrac{16\pi}{3}\)
⚡ Quick Guide — Divergence Theorem
  1. Verify surface is CLOSED (encloses a solid). If not, close it and subtract.
  2. Compute \(\text{div}\,\mathbf{F} = P_x + Q_y + R_z\)
  3. If div F = constant → flux = constant × volume
  4. Know your volumes: sphere \(\frac{4}{3}\pi R^3\), hemisphere \(\frac{2}{3}\pi R^3\), cylinder \(\pi R^2 h\)
B5 [7 marks] — Conservative Field Test + Potential Function
🧠 What is this asking?

Is \(\mathbf{F} = (e^{y+z},\; xe^{y+z},\; xe^{y+z})\) a gradient field? If so, find \(f\) with \(\nabla f = \mathbf{F}\).

📝 Full Solution

Step 1: Check curl F = 0.

\(P = e^{y+z}\), \(Q = xe^{y+z}\), \(R = xe^{y+z}\).

\(P_y = e^{y+z}\), \(Q_x = e^{y+z}\) ✓

\(P_z = e^{y+z}\), \(R_x = e^{y+z}\) ✓

\(Q_z = xe^{y+z}\), \(R_y = xe^{y+z}\) ✓

All match → F is conservative.

Step 2: Find f by partial integration.

From \(f_x = e^{y+z}\): \(f = xe^{y+z} + g(y,z)\).

From \(f_y = xe^{y+z} + g_y = xe^{y+z}\): so \(g_y = 0\), meaning \(g = h(z)\).

From \(f_z = xe^{y+z} + h'(z) = xe^{y+z}\): so \(h'(z) = 0\), meaning \(h = C\).

✅ Answer: Yes, F is conservative. \(f(x,y,z) = xe^{y+z} + C\)
⚡ Quick Guide — Finding Potential Functions
  1. Check curl F = 0 (all three cross-partial equalities)
  2. Integrate \(f_x = P\) with respect to x → get \(f = \ldots + g(y,z)\)
  3. Differentiate result with respect to y, set equal to Q → solve for \(g_y\)
  4. Integrate \(g_y\) with respect to y → get \(g = \ldots + h(z)\)
  5. Differentiate with respect to z, set equal to R → solve for \(h'(z)\)
  6. Integrate h'(z) → combine everything
B6 [8 marks] — Critical Points on a Closed Disk
🧠 What is this asking?

Find and classify critical points of \(f(x,y) = 8x^2 + 4y^2 + 4y + 5\) on the closed disk \(D = \{x^2+y^2 \le 1\}\). Also find absolute max/min.

📝 Full Solution

Step 1: Interior critical points.

\(f_x = 16x = 0 \Rightarrow x = 0\). \(f_y = 8y + 4 = 0 \Rightarrow y = -\frac{1}{2}\).

CP: \((0, -\frac{1}{2})\). Check: \(0^2 + (-\frac{1}{2})^2 = \frac{1}{4} < 1\) ✓ (inside disk).

Step 2: Classify using SDT.

\(f_{xx} = 16\), \(f_{yy} = 8\), \(f_{xy} = 0\).

\(D = (16)(8) - 0^2 = 128 > 0\) and \(f_{xx} = 16 > 0\) → local minimum.

Value: \(f(0, -\frac{1}{2}) = 0 + 1 - 2 + 5 = 4\).

Step 3: Boundary critical points. On \(x^2+y^2 = 1\), set \(x^2 = 1-y^2\):

\[g(y) = 8(1-y^2) + 4y^2 + 4y + 5 = -4y^2 + 4y + 13\]

\(g'(y) = -8y + 4 = 0 \Rightarrow y = \frac{1}{2}\). Then \(x^2 = 1 - \frac{1}{4} = \frac{3}{4}\), so \(x = \pm\frac{\sqrt{3}}{2}\).

\(g''(y) = -8 < 0\) → local maxima on boundary.

Value: \(g(\frac{1}{2}) = -1 + 2 + 13 = 14\).

Step 4: Check boundary endpoints. \(y \in [-1, 1]\).

\(g(-1) = -4 - 4 + 13 = 5\), \(g(1) = -4+4+13 = 13\).

Step 5: Compare all values.

Pointf-valueType
\((0, -\frac{1}{2})\)4Interior CP — local min
\((\pm\frac{\sqrt{3}}{2}, \frac{1}{2})\)14Boundary CP — local max
\((0, -1)\)5Boundary endpoint
\((0, 1)\)13Boundary endpoint
✅ Answer: Local min at \((0,-\frac{1}{2})\) with value 4. Local maxima at \((\pm\frac{\sqrt{3}}{2}, \frac{1}{2})\) with value 14.
Absolute min = 4 at \((0,-\frac{1}{2})\). Absolute max = 14 at \((\pm\frac{\sqrt{3}}{2}, \frac{1}{2})\).
⚡ Quick Guide — Extrema on Closed Region
  1. Find interior CPs: set \(f_x = 0\), \(f_y = 0\), check they're inside D
  2. Classify with SDT: compute D = \(f_{xx}f_{yy} - f_{xy}^2\)
  3. On boundary: substitute constraint into f to get a 1-variable function
  4. Find boundary CPs: set derivative = 0
  5. Check boundary endpoints too
  6. Compare ALL values → largest = absolute max, smallest = absolute min
